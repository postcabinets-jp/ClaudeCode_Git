# Notion 思考ノートテンプレートボタン 設定手順

## 概要

毎日の思考ノートを作成するときに使うテンプレートボタンをNotionに設定します。
`📋 Claude Tasks` トグルを含むページが自動生成されます。

## テンプレート構成

```
思考ノート - 2026/03/31     ← @today で自動挿入

## 今日の思考・メモ
（自由記述）

## チェックリスト（自分タスク）
- [ ]

▶ 📋 Claude Tasks
  ← ここにClaude向けタスクを自由記述
```

## セットアップ手順

### 1. 思考ノートを置く親ページを開く

「Daily Notes」「思考ログ」など、毎日のノートを置くページを開きます。

### 2. テンプレートボタンを追加

1. ページ内で `/template` と入力
2. 「テンプレートボタン」を選択
3. ボタン名: `📝 今日の思考ノート`

### 3. テンプレート本文を設定

テンプレート編集画面で以下を作成:

| ブロック種別 | 内容 |
|------------|------|
| ページタイトル | `思考ノート - @today`（@todayは自動で日付変換） |
| ヘッダー(H2) | `今日の思考・メモ` |
| テキスト | （空ブロック） |
| ヘッダー(H2) | `チェックリスト（自分タスク）` |
| チェックボックス | （空） |
| **トグル** | タイトル: `📋 Claude Tasks` |
| └ テキスト（子） | （空ブロック） |

### 4. 動作確認

テンプレートボタンをクリックして、`2026/03/31` 形式のタイトルのページが生成されることを確認。

## 書き方の例

`📋 Claude Tasks` トグル内に自由に記述:

```
AI研修の資料構成を叩き台として作っておいて
先週のDiscord通知が届かなかった原因を調べて
LINEボットのウェルカムメッセージを改善したい

- [ ] これはnobu自身がやる（Claudeは無視する）
```

## タスク抽出の実行

```bash
# 今日の思考ノートを処理
npm run notion:extract-tasks

# 過去の日付を指定
npm run notion:extract-tasks -- --date 2026/03/30
```

TasksDBのInboxに自動登録されます。
夜間スキャン（毎朝07:00）はlaunchdで自動実行されます。

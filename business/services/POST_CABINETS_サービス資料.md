# SECTION 01｜AI導入で企業がつまずく3つの壁

生成AIの登場以降、「自社でもAIを活用したい」という企業は増え続けている。しかし、実際にAIが業務に定着している企業はごく一部である。多くの企業は、以下の3つの壁にぶつかって立ち止まっている。

**壁①：何から始めればいいかわからない**
AIツールは無数にあり、情報も溢れている。しかし「自社の業務のどこに、どのツールを、どう使えばいいか」を判断できる人材が社内にいない。結果として、検討だけが続き、何も動かない。

**壁②：一部の人が使うだけで終わる**
意欲のある社員が個人的にChatGPTを使い始めるが、チームや組織全体には広がらない。「便利だとは思うけど、自分の業務にどう使えばいいかわからない」という社員が大半を占め、ツール導入が成果につながらない。

**壁③：導入しても定着しない**
研修を受けた直後は使うが、1ヶ月後には元の業務に戻っている。使い続ける仕組みや、困ったときに相談できる相手がいないため、AIが日常業務に組み込まれない。

POST CABINETSのAIコンサルティングは、この3つの壁を突破するために設計されている。

# SECTION 02｜POST CABINETSのアプローチ

## 3段階で組織を変える

POST CABINETSは、「研修で教えて終わり」でも「ツールを入れて終わり」でもない。**個人 → チーム → 組織**の3段階で、AIが業務に根づくまでの変革プロセスそのものを伴走する。

| フェーズ | やること | 対応サービス | 期間の目安 |
|---------|---------|-------------|-----------|
| **PHASE 1：個人のAI活用** | 社員一人ひとりがAIを日常業務で使える状態にする | AI研修（サブスク / 助成金対応 / オーダーメイド） | 1〜3ヶ月 |
| **PHASE 2：チームの業務自動化** | AIを業務プロセスに組み込み、チーム単位で成果を出す | 伴走支援（月額） | 3〜6ヶ月 |
| **PHASE 3：全社のAI組織化** | 成果を全社に横展開し、AI活用が自走する組織を作る | 戦略策定・体制構築支援 | 企業の状況に応じて |

この3段階は必ずしも順番通りに進む必要はない。「まず1つのチームで伴走支援から始め、成果が出たら全社研修に広げる」ケースもあれば、「研修と伴走支援を並行して進める」こともある。企業の現状に合わせて、最適な入り口と進め方を一緒に設計する。

どのフェーズでも、「これはAIツールの導入や開発が必要だ」という課題が見えてきた場合には、**ソリューション提案**として最適なAIシステムの設計・構築まで対応する。

## AI活用ツールの考え方

POST CABINETSの研修・支援は、**特定のツールを強制しない**。研修カリキュラムではChatGPT、Claude、Notion AI、Cursor、Claude Codeなど幅広いAIツールを扱うが、クライアントが既に導入済みのツール（Microsoft Copilot、Geminiなど）がある場合はそれを活かした提案も行う。

特にClaude Codeは「自然言語で指示するだけで業務の自動化やアプリ構築まで実現できる」ツールとして推奨しているが、あくまで選択肢の一つである。大切なのはツールではなく、**「AIをどう業務に組み込むか」という設計力**。この設計力こそが、研修と伴走支援を通じて企業に残る本質的な資産となる。

# SECTION 03｜PHASE 1：個人のAI活用（AI研修）

## なぜ「個人」から始めるのか

組織のAI活用は、結局のところ「一人ひとりが使えるかどうか」に帰結する。どれほど優れたAIツールを導入しても、現場の社員が使いこなせなければ効果は出ない。PHASE 1の研修は、AI未経験者でも安心して始められる設計でありながら、受講後すぐに業務で使えるレベルまで引き上げることを目指す。

## サービス概要

| 項目 | 内容 |
|------|------|
| **対象** | 全社員（経営層〜現場スタッフ）。AI未経験者を含む |
| **受講方法** | オンライン動画（録画）＋ライブセッション（双方向）＋対面研修（オンサイト） |
| **コンテンツ更新** | AI業界の変化に合わせて既存講座も随時アップデート |

## 3つの研修メニュー

| メニュー | 内容 | こんな企業に |
|---------|------|-------------|
| **(A) サブスクリプション型** | 月額制で全Packにアクセス。企業のペースで自由に受講を進められる。新講座も追加費用なし | まずは少人数で試したい、自分たちのペースで進めたい |
| **(B) 助成金対応パッケージ** | 人材開発支援助成金の要件を満たすカリキュラムを組み、企業の研修コスト負担を大幅に軽減 | コストを抑えつつ、しっかり社員を育成したい |
| **(C) オーダーメイド研修** | クライアント企業の業種・業務・使用ツールに合わせたカスタマイズ研修。Salesforce連携、kintone効率化など、汎用カリキュラムではカバーしきれないテーマにも対応 | 自社の業務に特化した内容で研修してほしい |

# SECTION 04｜助成金活用ガイド

AI人材の育成には投資が必要だが、**人材開発支援助成金**を活用することで企業のコスト負担を大幅に軽減できる。POST CABINETSの研修カリキュラムは、助成金の要件を満たす設計に対応済みである。

## 対象となる2つの助成金コース

### ① 人材育成支援コース

| 項目 | 内容 |
|------|------|
| **要件** | 10時間以上のOFF-JT |
| **経費助成** | 45%（賃金要件等を満たした場合60%） |
| **賃金助成** | 760円/時間 |
| **適合する研修** | Pack 1〜4の各パック（各10講座 = 10時間以上を確保可能） |

Pack 1（AI基礎＆業務自動化）は全社員対象の基礎研修として、最も申請しやすい。

### ② 人への投資促進コース（高度デジタル人材訓練）

| 項目 | 内容 |
|------|------|
| **要件** | 10時間以上のOFF-JT。高度なデジタル技術に関する訓練であること |
| **経費助成** | **75%**（①の約1.7倍） |
| **賃金助成** | **960円/時間** |
| **適合する研修** | 特にPack 2（AI開発実践）＋ Pack 3（AI組織導入）の組み合わせ |

Pack 2のClaude Code・Cursor・OpenClawを使ったAI開発スキル訓練は「高度デジタル人材訓練」の要件に合致する可能性が高い。対象となる企業には、①ではなく②での申請を積極的に提案する。

## 注意事項

- **訓練開始の1ヶ月前までに計画届の提出が必要**。早めの相談を推奨する
- 研修カリキュラムは助成金要件に合わせた時間数・内容の調整が可能
- 助成金の申請手続きサポートも提供（申請代行は提携社労士と連携）
- 令和8年度の助成金制度改定に合わせて、最新の上限額・要件を反映した提案を行う

# SECTION 05｜PHASE 2：チームの業務自動化（伴走支援）

## なぜ「研修だけ」では足りないのか

研修で社員一人ひとりがAIを使えるようになっても、それだけでは組織の生産性は変わらない。個人のスキルが業務プロセスに組み込まれ、チームの日常として回り始めて初めて、本当の成果が出る。PHASE 2の伴走支援は、この**「個人のスキル → チームの成果」への転換**を実現するサービスである。

## サービス概要

| 項目 | 内容 |
|------|------|
| **対象** | AI研修を受講済み（または同等のスキルを持つ）企業・チーム |
| **提供形態** | 月額制。専任コンサルタントが継続的に並走する |
| **頻度** | 月3〜4回のセッション（オンラインまたは対面） |

## 伴走支援で取り組むこと

**業務フローの棚卸しとAI導入ポイントの特定**: 対象チームの業務フローを一緒に可視化し、AI導入で最も効果が出るポイントを見極める。「営業チームの提案書作成」「管理部門の月次レポート」など、具体的な業務単位で目標を設定する。

**Quick Winの創出**: 最初の1〜2週間で、小さくても確実に効果が実感できる施策から着手する。たとえば議事録の自動要約、FAQチャットボットの試作、定型メールの自動生成など。「AIって本当に使える」という実感がチーム内に広がることが、次の施策への推進力になる。

**定着までの伴走**: ツールを入れて終わりではない。チームメンバーが日常的にAIを使い続ける状態を作る。疑問への回答、新しいユースケースの提案、効果測定と改善を継続的に行う。最終的には、**POST CABINETSの支援がなくてもAI活用が回り続ける状態**をゴールとする。

## 月次の進め方（標準例）

| 週 | セッション内容 | 成果物の例 |
|----|--------------|-----------|
| **第1週** | 月初キックオフ：先月の振り返り＋今月の目標設定 | 月次目標と施策リスト |
| **第2週** | 施策の実装支援：AI活用フローの構築・調整 | 業務自動化フロー、プロンプトテンプレート |
| **第3週** | 進捗確認＋現場フォロー：使ってみてのQ&A対応 | FAQ集、改善された運用マニュアル |
| **第4週** | 効果測定＋来月の計画：定量・定性の振り返り | 効果測定レポート、来月の施策計画 |

## 伴走支援から生まれる成果物（例）

- チーム専用のAI活用プロンプト集・テンプレート集
- 業務自動化フロー（議事録自動要約、レポート自動生成など）
- 社内AIチャットボット（FAQ対応、社内ナレッジ検索）
- AI活用ガイドライン（情報セキュリティルール含む）
- 効果測定ダッシュボード（削減時間・コスト改善の可視化）

# SECTION 06｜PHASE 3 & ソリューション提案

## PHASE 3：全社のAI組織化

PHASE 2で特定のチームが成果を出したら、それを全社に広げていく。PHASE 3のゴールは、**社内でAI活用が自律的に回り続ける組織**を作ることであり、POST CABINETSが永久に伴走し続けることではない。

企業の状況に応じて、以下の支援を柔軟に組み合わせる。

**全社AI活用戦略の策定** — 経営層と連携し、中期的なAI活用ビジョンとロードマップを策定する。どの部門から、どのツールで、どの業務を変えていくかを設計する。

**全社ガイドラインの整備** — PHASE 2で一部チーム向けに作ったAI活用ルールを、全社共通のガイドラインに拡張する。情報セキュリティ、品質管理、倫理面のルールを含む。

**AI推進体制の構築** — 社内にAI推進チームを立ち上げ、各部門にAI推進担当者を配置する体制づくりを支援する。

**横展開の仕組みづくり** — 成功事例を他部門に広げるための社内共有会の設計、ナレッジベースの構築など、成果が自然に広がる仕組みを作る。

PHASE 3は企業ごとに必要な支援が大きく異なるため、テンプレート型のサービスではなく、PHASE 1・2の中で見えてきた課題と成果をもとに、個別に設計する。

## ソリューション提案

PHASE 1〜3のどの段階でも、「これは既存ツールの組み合わせでは足りない。専用のAIシステムを構築する必要がある」という課題が見えてくることがある。そうした場合に、POST CABINETSが設計・構築まで対応する。

| 項目 | 内容 |
|------|------|
| **位置づけ** | 各PHASEの中で必要に応じて提案。独立したサービスとしても提供可能 |
| **進め方** | 課題のヒアリング → 複数ソリューションの比較検討 → ROIを明示した導入計画 → 構築・導入 → 効果測定 |
| **対応例** | AIチャットボット構築、業務自動化ワークフロー開発、社内ナレッジベース構築、AIエージェント設計・開発 |
| **対応ツール** | ChatGPT、Claude、Gemini、Microsoft Copilot、n8n、Dify、Zapier、Notion AI、Claude Code、Cursor、OpenClaw、その他 |

研修や伴走支援でクライアントの業務を深く理解した上で提案するため、「導入したが現場で使われない」というありがちな失敗を回避できる。

## POST CABINETSを選ぶ理由

**研修と伴走が一体である**。研修だけ提供する会社、コンサルだけの会社は多いが、「教えた人間がそのまま現場に入って一緒に実装する」体制はPOST CABINETSの特徴である。研修で教えた講師が伴走支援にも入るため、クライアントの業務理解にギャップが生まれない。

**AIの実装力を持つチームが直接支援する**。戦略だけ作って「あとは御社で」とはしない。大手企業のAIエージェント開発を主導した黒崎、複数のAIプロダクトをリリースしている松本など、自らAIシステムを設計・構築できるメンバーが、クライアントの現場で手を動かす。

**AI業界の変化に常に追従する**。AI業界は月単位で状況が変わる。POST CABINETSの研修コンテンツはサブスクリプションモデルで常時アップデートされ、伴走支援でも最新のツール・手法を反映した提案を行う。「半年前の研修資料で教えている」ということは起きない。

### AI活用ツールの考え方

POST CABINETSの研修・支援は、**特定のツールを強制するのではなく、クライアント企業の環境に合った最適なAI活用を提案する**方針をとる。

研修カリキュラムでは、ChatGPT、Claude、Notion AI、Cursor、Claude Codeなど幅広いAIツールを扱う。特にClaude Codeは「自然言語で指示するだけで業務の自動化まで実現できる」ツールとして推奨しているが、クライアントが既に導入済みのツール（Microsoft Copilot、Geminiなど）がある場合はそれを活かした活用方法も併せて提案する。

大切なのはツールではなく、「AIをどう業務に組み込むか」という設計力。この設計力こそが、POST CABINETSの研修で身につく本質的なスキルとなる。

## PHASE 1：個人のAI活用（AI研修）

### サービス概要

| 項目 | 内容 |
|------|------|
| **目的** | 社員一人ひとりが日々の業務でAIを「自分の道具」として使える状態にする |
| **対象** | 全社員（経営層〜現場スタッフ）。AI未経験者を含む |
| **提供形態** | サブスクリプション型（月額）＋助成金対応の単発研修パッケージ |
| **受講方法** | オンライン動画（録画）＋ライブセッション（双方向）＋対面研修（オンサイト） |
| **コンテンツ更新** | AI業界の変化に合わせて既存講座も随時アップデート |

### 2つの研修メニュー

#### (A) サブスクリプション型AI研修

月額制で全コンテンツにアクセスできるプラン。企業のペースで自由に受講を進められる。新しい講座が追加されるたびに追加費用なしでアクセスできるため、AI業界の変化についていける。

#### (B) 助成金対応研修パッケージ

人材開発支援助成金を活用した研修パッケージ。Pack 1〜4の中から助成金要件（10時間以上のOFF-JT等）を満たすカリキュラムを組み、企業の研修コスト負担を大幅に軽減する。

#### (C) オーダーメイド研修

Pack 1〜4の講座に加え、**クライアント企業の業種・業務・使用ツールに合わせたカスタマイズ研修**も提供する。たとえば「自社が導入しているSalesforceとAIの連携」「社内で使っているkintoneの業務をAIで効率化する」など、汎用カリキュラムではカバーしきれないテーマにも対応できる。

### 助成金活用ガイド

POST CABINETSの研修カリキュラムは、**人材開発支援助成金**を活用した設計に対応している。企業の研修コスト負担を大幅に軽減できる。

#### 対象となる助成金コース

**① 人材育成支援コース**

| 項目 | 内容 |
|------|------|
| **対象** | 雇用する被保険者に対する職業訓練 |
| **要件** | 10時間以上のOFF-JT |
| **経費助成** | 45%（賃金要件等を満たした場合60%） |
| **賃金助成** | 760円/時間 |
| **適合する研修** | Pack 1〜4の各パック（各10講座で10時間以上を確保可能） |

Pack 1（AI基礎＆業務自動化）は、AIに初めて触れる社員を含む幅広い層に対応しているため、人材育成支援コースとして最も申請しやすい。

**② 人への投資促進コース（高度デジタル人材訓練）**

| 項目 | 内容 |
|------|------|
| **対象** | 高度デジタル人材（IT・デジタル分野）の育成訓練 |
| **要件** | 10時間以上のOFF-JT。高度なデジタル技術に関する訓練であること |
| **経費助成** | 75% |
| **賃金助成** | 960円/時間 |
| **適合する研修** | 特にPack 2（AI開発実践）・Pack 3（AI組織導入）の組み合わせ |

Pack 2はClaude Code・Cursor・OpenClawなどを使った実践的なAI開発スキルの訓練であり、「高度デジタル人材訓練」の要件に合致する可能性が高い。Pack 3と組み合わせることで、AI戦略策定からリスク管理まで含む包括的な高度デジタル人材育成プログラムとなる。

#### 助成金活用のポイント

- **訓練開始の1ヶ月前までに計画届の提出が必要**。早めのご相談を推奨
- 研修カリキュラムは助成金要件に合わせた時間数・内容の調整が可能
- 助成金の申請手続きに関するサポートも提供（申請代行は社労士と連携）
- 令和8年度の助成金制度改定に合わせて、最新の上限額・要件を反映した提案を行う

## PHASE 2：チームの業務自動化（伴走支援）

### サービス概要

| 項目 | 内容 |
|------|------|
| **目的** | PHASE 1で身につけたAIスキルを、実際の業務に組み込み、チーム単位で成果を出す |
| **対象** | AI研修を受講済み（または同等のスキルを持つ）企業・チーム |
| **提供形態** | 月額制の伴走支援。専任コンサルタントが継続的に並走する |
| **頻度** | 月3〜4回のセッション（オンラインまたは対面） |

### 伴走支援で実現すること

PHASE 2では、研修で学んだ「個人のAIスキル」を「チームの業務プロセス」に落とし込む。具体的には以下のような支援を行う。

**業務プロセスへのAI組み込み**: 対象チームの業務フローを一緒に棚卸しし、AI導入によって最も効果が出るポイントを特定。たとえば「営業チームの提案書作成を60%時短する」「管理部門の月次レポート作成を自動化する」といった具体的な目標を設定し、実現する。

**Quick Winの創出**: まず小さく・早く成果を出すことを重視する。たとえば議事録の自動要約、FAQチャットボットの構築、定型メールの自動生成など、1〜2週間で効果を実感できる施策から着手し、チーム内に「AIって使える」という実感を広げる。

**定着支援**: ツールを導入して終わりではなく、チームメンバーが日常的にAIを使い続ける状態を作る。使い方の疑問への回答、新しいユースケースの提案、効果測定と改善を継続的に行う。

### 月次の進め方（標準例）

| 週 | セッション内容 | 成果物の例 |
|----|--------------|-----------|
| **第1週** | 月初キックオフ：先月の振り返り＋今月の目標設定 | 月次目標と施策リスト |
| **第2週** | 施策の実装支援：AI活用フローの構築・調整 | 業務自動化フロー、プロンプトテンプレート |
| **第3週** | 進捗確認＋現場フォロー：使ってみてのQ&A対応 | FAQ集、改善された運用マニュアル |
| **第4週** | 効果測定＋来月の計画：定量・定性の振り返り | 効果測定レポート、来月の施策計画 |

### 伴走支援の成果物（例）

伴走支援を通じて、以下のような成果物が生まれる（企業の課題に応じてカスタマイズ）。

- チーム専用のAI活用プロンプト集・テンプレート集
- 業務自動化フロー（議事録自動要約、レポート自動生成など）
- 社内AIチャットボット（FAQ対応、社内ナレッジ検索）
- AI活用ガイドライン（情報セキュリティルール含む）
- 効果測定ダッシュボード（削減時間・コスト改善の可視化）

## PHASE 3：全社のAI組織化

### サービス概要

PHASE 1・2で作った成果をチーム横断で展開し、「AI活用が当たり前の組織」を目指す段階。企業の状況に応じて、以下のような支援を柔軟に組み合わせる。

**全社AI活用戦略の策定**: 経営層と連携し、中期的なAI活用ビジョンとロードマップを策定する。どの部門から、どのツールで、どの業務を変えていくかを設計する。

**全社ガイドラインの整備**: PHASE 2で一部チーム向けに作ったAI活用ルールを、全社共通のガイドラインに拡張する。情報セキュリティ、品質管理、倫理面のルールを含む。

**AI推進体制の構築**: 社内にAI推進チーム（CoE: Center of Excellence）を立ち上げ、各部門にAI推進担当者を配置する体制づくりを支援する。POST CABINETSの役割は、社内チームが自走できるまでの伴走。

**横展開の仕組みづくり**: 成功事例を他部門に広げるための社内共有会の設計、ナレッジベースの構築など、成果が自然に広がる仕組みを作る。

研修や伴走支援の中で「これはツール導入や開発が必要」と判明した課題に対して、最適なAIソリューションを提案する。

| 項目 | 内容 |
|------|------|

| **対応例** | AIチャットボット構築、業務自動化ワークフロー開発、社内ナレッジベース構築、AIエージェント設計・開発 |
| **対応ツール** | ChatGPT、Claude、Gemini、Microsoft Copilot、n8n、Dify、Zapier、Notion AI、Claude Code、Cursor、OpenClaw、その他 |

## 料金体系

| サービス | 料金モデル | 目安 |
|---------|-----------|------|
| **サブスクリプション型AI研修** | 月額（企業単位） | 助成金活用を前提に、企業の実質負担を最小化する設計 |
| **助成金対応研修パッケージ** | 単発（パック単位） | 令和8年度の助成金上限額を最大限活用できる価格設定 |
| **伴走支援** | 月額 | 月額10万円〜（支援内容・頻度により変動） |
| **ソリューション提案** | プロジェクト型 | 個別見積もり |

- サブスクリプション型AI研修は、月額制で全Packのコンテンツにアクセスできるモデル。助成金を活用することで実質負担を大幅に軽減できる
- 助成金対応研修パッケージは、企業が受給できる助成金の上限額を最大限活用できるように価格を設計する
- 伴走支援は月額10万円を最低ラインとし、支援範囲・頻度に応じて調整する
- まずは**無料相談**で、企業の課題と最適なサービスの組み合わせを提案する

## ご支援の流れ

1. **無料相談** — 現在の課題やご要望をヒアリングし、最適なサービスを提案

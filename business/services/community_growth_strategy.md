## 概要

株式会社POST CABINETSが運営するDiscordベースのAIコミュニティを、30日以内に有料ローンチするための集客・グロース戦略。

| 項目 | 内容 |
|------|------|
| プラン（一般） | ¥5,000/月 |
| プラン（プレミアム） | ¥100,000/月（経営者向け） |
| プラットフォーム | Discord |
| 体制 | Nobu（戦略・コンテンツ）+ 営業2名 + AI実装者2名 |

## 1. 集客戦略の全体像

### ターゲット

| セグメント | 属性 | 狙うプラン |
|-----------|------|-----------|
| A: 中小企業の経営者・管理職 | 従業員10〜300名規模、AI導入を検討中 | プレミアム（¥100,000/月）または法人一括 |
| B: 一般社員・個人 | AIスキルアップを目指すビジネスパーソン | 一般（¥5,000/月） |
| C: AIフリーランス・副業者 | 実績積み上げ中のAI実装者、コミュニティとパートナー候補を兼ねる | 一般（¥5,000/月）+ パートナー採用候補 |

### 2チャネルの役割分担

```
SNS発信（Nobu）                    営業2名（対面・DM）
    ↓                                      ↓
認知・信頼形成                       直接接触・関係構築
    ↓                                      ↓
    ←──── 無料オンラインセミナー / 無料体験 ────→
                        ↓
                  有料コミュニティ加入
```

**SNSチャネルの役割**: ブランド認知を広め、Nobuへの信頼と共感を蓄積する。直接的な販売よりも「この人のコミュニティに入りたい」という欲求を育てる。

**営業2名の役割**: 商工会議所・BNI・経営者交流会での直接接触で、温度の高いリードを短期間で獲得する。経営者セグメント（プレミアム）は対面クロージングを基本とする。

## 3. SNS戦略（Nobu発信）

### 3つのコンテンツテーマ

#### テーマ1：AI実装実録
「実際に何を作ったか、どう役立ったか」をリアルタイムで発信する。

- 例：「Claude Codeで営業資料を自動生成する仕組みを作った。工数が1/5になった」
- 例：「Notionとボットを繋いで、タスク管理を完全自動化した手順を公開」
- 抽象論ではなく、具体的なツール名・数字・Before/Afterを必ず入れる
- コミュニティのAI実装者メンバーの事例も（許可を得て）発信する

#### テーマ2：経営 × AI
「中小企業の経営者の視点でのAI活用」を語る。

- 例：「社員3名でもAIを組み込めば10名分の仕事ができる理由」
- 例：「AIコンサルで失敗する会社と成功する会社の違い」
- Nobuが経営者であり政治家でもあるという二刀流の視点を活かす
- 読者が「自社に当てはまる」と感じられる解像度で書く

#### テーマ3：失敗・つまずき
「リアルな失敗談でこそ信頼が生まれる」ことを意識する。

- 例：「3ヶ月取り組んだAI自動化が全部崩れた話」
- 例：「AIエージェントを導入したら逆に仕事が増えた失敗」
- 成功談より失敗談のほうがエンゲージメントが上がりやすい
- 教訓・学びまでセットで書くことで「また読みたい」と思われる

### プラットフォーム別の運用方針

| プラットフォーム | 頻度 | 目的 | 投稿スタイル |
|----------------|------|------|-------------|
| X（Twitter） | 毎日（1〜3投稿） | 認知拡大・拡散 | 短文・箇条書き・率直な本音 |
| LinkedIn | 週2〜3回 | 経営者・ビジネス層へのリーチ | 少し長め・実績ベース・専門性を出す |
| Instagram | 週2回 | ビジュアルでの補完・若い層へのリーチ | 図解・スクリーンショット・Reels |
| YouTube / Loom | 月2本 | 深い信頼形成・SEO資産化 | 10〜15分の解説動画・実装デモ |

### ハッシュタグ（X・Instagram共通）

主要タグ（必ず使用）:
`#AI活用` `#中小企業DX` `#AI経営` `#AIコミュニティ`

補助タグ（投稿内容に応じて選択）:
`#Claude` `#ChatGPT活用` `#業務自動化` `#副業AI` `#AIフリーランス` `#経営者` `#DX推進`

## 4. 営業2名によるリアル接触戦略

### 主要な接点チャネル

| チャネル | 頻度 | 担当 | アプローチ対象 |
|---------|------|------|--------------|
| 商工会議所イベント | 月2〜4回 | 営業2名 | 中小企業の経営者・管理職 |
| BNI（ビジネスネットワーク） | 週1回（入会済みの場合） | 営業1名 | 経営者・士業・フリーランス |
| 経営者交流会（民間） | 月1〜2回 | 営業2名 | プレミアム候補の経営者 |
| LinkedIn DM | 週5〜10件 | 営業2名 | 経営者・管理職（1次接触） |

### 対面アプローチの流れ（スクリプト）

**Step 1: 自己紹介（30秒）**
> 「POST CABINETSという会社でAIを使った業務改善の支援をしています。先日、クライアントの社内業務を自動化して月30時間削減できた事例があって、今そういった事例をコミュニティでシェアしているんです。」

**Step 2: 相手への関心（ヒアリング）**
> 「御社は今、AIはどの程度使われていますか？」→ 課題を引き出す

**Step 3: 無料セミナーへの誘導**
> 「来週オンラインで無料の勉強会があって、実際の事例を30分でお伝えするんですが、もしよかったら参加してみませんか？」

**Step 4: 有料コミュニティ案内**
> 無料セミナー参加後に個別フォローで案内。経営者には1on1で提案。

### LinkedIn DMの流れ

1. 接続申請（メッセージなし、まずつながる）
2. 2〜3日後: 挨拶DM（自己紹介 + 共通点の言及）
3. 返信があれば: 課題ヒアリング or 無料セミナー案内
4. 返信なし: 1週間後に投稿へのコメントで存在感を出す

DM文例:
> 「初めまして、POST CABINETS代表の前田と申します。○○さんの投稿を拝見し、AI活用に積極的に取り組まれていると感じてご連絡しました。私は中小企業向けのAI実装支援とコミュニティ運営をしており、来週無料の勉強会を開催します。よろしければぜひ。」

## 5. 30日ローンチ計画

### Week 1（Day 1〜7）：プレビュー発信・存在を知らせる

**目的**: コミュニティの存在を認知させる。「何かが始まる」という期待感を作る。

| 施策 | 内容 |
|------|------|
| SNS発信（毎日） | 「AIコミュニティを作っています」という告知を含む投稿を始める |
| ティーザー投稿 | コミュニティで扱うテーマのプレビューを投稿 |
| 営業2名 | 接触先リスト作成・LinkedIn接続申請開始 |
| Discord準備 | チャンネル設計・ウェルカムメッセージ・ルール整備 |

**KPI**: 告知投稿のインプレッション数 / プロフィールへの流入数

### Week 2（Day 8〜14）：招待制先行招待（20〜30名）

**目的**: コミュニティに最初の熱量を作る。Nobu人脈から信頼できる人を先行招待。

| 施策 | 内容 |
|------|------|
| 先行招待DM | Nobuが直接20〜30名にメッセージ（LINE / DM） |
| 招待文 | 「無料で先行参加してほしい。フィードバックをもらいたい」という文脈 |
| Discord開設 | 招待者限定でDiscordを開く |
| SNS発信 | 「先行メンバーが集まっています」という発信で興味を煽る |

招待対象:
- 過去の仕事でつながった経営者・ビジネスパーソン
- AIに関心がある知人・友人
- Cセグ候補のAIフリーランス・副業者

### Week 3（Day 15〜21）：無料オンラインイベントでコミュニティ体験

**目的**: 参加者にコミュニティの価値を体感させ、有料転換への意欲を高める。

| 施策 | 内容 |
|------|------|
| 無料セミナー開催 | 60分。テーマ: 「2026年、中小企業がAIで差をつける3つの実装事例」 |
| 参加者のDiscord誘導 | セミナー終了後にDiscordリンクを配布（無料体験として） |
| 営業フォロー | セミナー参加者に対して翌日以内に個別フォロー |
| SNS発信 | セミナーのハイライト・参加者の声を発信 |

### Week 4（Day 22〜30）：有料ローンチ（¥5,000/月）

**目的**: 有料メンバーを最初の10〜20名獲得する。

| 施策 | 内容 |
|------|------|
| ローンチ告知 | SNSとDiscordで有料化の告知（2〜3日前から予告） |
| アーリーバード特典 | 最初の30名は初月50%オフ（¥2,500）などの特典を検討 |
| 先行メンバーへの案内 | Week 2の無料先行メンバーに「有料化のご案内」を送る |
| 経営者への個別提案 | プレミアム（¥100,000/月）候補者に対して営業2名が1on1でクロージング |

## 8. 無料イベント設計（集客のフック）

### 基本設計

| 項目 | 内容 |
|------|------|
| 開催頻度 | 月1回 |
| 形式 | オンライン（Zoom / Google Meet）60分 |
| 定員 | 制限なし（録画あり） |
| 参加費 | 無料 |
| 申込方法 | Googleフォーム または Connpass |

### テーマ案（6ヶ月分）

| 月 | テーマ |
|----|--------|
| 1ヶ月目 | 2026年、中小企業がAIで差をつける3つの実装事例 |
| 2ヶ月目 | 社員がAIを使いこなせない会社と使える会社の違い |
| 3ヶ月目 | AI×マーケティング：問い合わせを2倍にした実話 |
| 4ヶ月目 | 経営者がAIに丸投げして失敗した話と学んだこと |
| 5ヶ月目 | 助成金を使ったAI研修の導入事例（実費ゼロの進め方） |
| 6ヶ月目 | AIエージェント組織を作って3ヶ月で変わったこと |

### セミナーの構成（60分）

```
0:00〜 0:05  オープニング・Nobu自己紹介
0:05〜 0:35  メインコンテンツ（実装事例 × 3本 or 深掘り1本）
0:35〜 0:50  Q&A（チャットから質問を拾う）
0:50〜 1:00  コミュニティ案内 + クロージング
```

### CTAの設計（参加者をコミュニティへ誘導）

セミナー終盤で必ず伝えること:

1. **Discord無料体験の案内**
   > 「今日参加してくれた方には、Discordの1週間無料体験を用意しています。今話したような事例を毎週共有しているので、ぜひ覗いてみてください。」

2. **QRコード / リンクの提示**
   > スライドの最終ページにDiscord招待リンクを掲載。Zoomのチャットにも貼る。

3. **フォローアップメール（翌日以内）**
   > 申込フォームで取得したメールアドレスに、録画URL + Discordリンクを送付。

4. **営業からの個別フォロー**
   > 経営者・管理職の参加者には翌営業日以内に営業から個別DMまたは電話。

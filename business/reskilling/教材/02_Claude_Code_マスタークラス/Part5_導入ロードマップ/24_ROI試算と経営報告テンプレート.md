## 1. ROI試算ツール

### 試算の基本公式

```
年間コスト = 人数 × 月額単価 × 12ヶ月
年間効果  = 人数 × 月間削減時間 × 時給 × 12ヶ月
ROI      = (年間効果 - 年間コスト) / 年間コスト × 100
投資回収期間 = 年間コスト / 年間効果 × 12ヶ月
```

### 料金プラン早見表（2025年時点）

| プラン | 月額(USD) | 月額(JPY概算) | 対象 |
|--------|----------|-------------|------|
| Free | $0 | ¥0 | 個人お試し |
| Pro | $20 | ¥3,000 | 個人業務利用 |
| Team | $30/人 | ¥4,500/人 | チーム利用（推奨） |
| Enterprise | カスタム | カスタム | 大規模導入 |

※ JPY概算は$1=¥150で算出。契約時の為替レートで再計算してください。

### 自社データでのカスタマイズ手順

**Step 1: 対象業務の洗い出し**

```
【業務棚卸しシート】

| # | 業務名 | 月間発生回数 | 1回あたり時間(分) | Claude化可能性 | 削減見込み(%) |
|---|--------|------------|----------------|---------------|-------------|
| 1 |        |            |                | 高/中/低       |             |
| 2 |        |            |                | 高/中/低       |             |
| 3 |        |            |                | 高/中/低       |             |
| 4 |        |            |                | 高/中/低       |             |
| 5 |        |            |                | 高/中/低       |             |
```

**Step 2: 削減時間の積み上げ計算**

```
月間削減時間 = Σ（発生回数 × 1回あたり時間 × 削減率）

例:
メール作成: 20回 × 15分 × 50% = 150分
議事録整理: 4回 × 30分 × 60% = 72分
企画書作成: 2回 × 120分 × 40% = 96分
リサーチ:   8回 × 45分 × 30% = 108分
文章校正:   10回 × 10分 × 70% = 70分
-----------------------------------------
合計: 496分 = 約8.3時間/人/月
```

**Step 3: 時給の算出**

```
年間人件費の概算時給:
時給 = 年収 × 1.3（社会保険料等） / (年間営業日 × 8時間)

例: 年収500万の場合
時給 = 500万 × 1.3 / (245日 × 8h) = ¥3,316
→ 概算: ¥3,300/時
```

## 4. 効果を「見える化」する5つのKPI

### KPI 1: AI利用率（月間アクティブ率）

```
【定義】
AI利用率 = 月間ログインユーザー数 / 対象ユーザー数 × 100

【目標値】
導入1ヶ月目: 60%
導入3ヶ月目: 80%
導入6ヶ月以降: 85%以上

【測定方法】
- Claude Team管理画面のアクティビティデータ
- または月次アンケートでの自己申告

【改善アクション（利用率が低い場合）】
60%未満:
  → 研修の再実施
  → 「Claude活用チャレンジウィーク」の開催
  → ペアワーク（ヘビーユーザー×未利用者）

60-80%:
  → 未利用者への個別ヒアリング
  → プロンプトテンプレートの配布
  → 好事例の社内共有

80%以上:
  → 現状維持 + 活用の深度向上
```

---

### KPI 2: 時間削減効果（時間/人/月）

```
【定義】
時間削減効果 = Σ(各業務のBefore時間 - After時間) / 対象人数

【目標値】
導入1ヶ月目: 2時間/人/月
導入3ヶ月目: 5時間/人/月
導入6ヶ月以降: 8時間/人/月以上

【測定方法】

### KPI 3: コスト削減効果（円/月）

```
【定義】
コスト削減効果 = 時間削減効果(全体) × 平均時給

【計算例】
50時間/月 × ¥3,500/時 = ¥175,000/月

【表示方法】
月次: ¥175,000
四半期: ¥525,000
年間: ¥2,100,000

【注意点】
- 時給は「会社が負担する総コスト」で計算する（給与 × 1.3倍程度）
- 残業削減分は残業単価（通常の1.25倍）で計算するとより正確
- 削減時間が「新しい業務に使われた」場合は、機会コストとして計上も可能

【段階的な効果の表現方法】
Level 1: 直接的な時間削減効果のみ
Level 2: + 品質向上による手戻り削減
Level 3: + 新規事業・改善活動への再投資効果
```

```

## まとめ: 数字で語る AI 導入

### 経営層が知りたい3つのこと

1. **いくらかかるのか?** → コスト試算で明確に提示
2. **いくら戻ってくるのか?** → ROI試算で3シナリオ提示
3. **リスクは何か?** → 月単位の解約可能、パイロット方式で段階的に

### 報告のコツ

- **数字は3つまで**: エグゼクティブサマリーの数字は最大3つ
- **保守的な数字で語る**: 過大な効果を提示すると信頼を失う
- **事例をセットで**: 数字だけでなく具体的な利用シーンを添える
- **次のアクションを明確に**: 報告で終わらず、承認事項を提示する
- **撤退基準も示す**: 「効果がなければやめる」という姿勢が信頼を生む

### 撤退基準（参考）

以下の条件が**3ヶ月連続で満たされない**場合、縮小・撤退を検討:
- 利用率50%未満
- ROIがマイナス
- 満足度3.0未満

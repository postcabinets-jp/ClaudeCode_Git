## はじめに

このプランは、Claude未経験の方が**30日間で業務にClaudeを定着させる**ためのステップバイステップガイドです。
1日15〜30分の取り組みで、確実にスキルが身につく設計になっています。

### 30日後のゴール
- Claudeへの「まず聞いてみる」習慣が定着している
- 自分専用のProjectが完成し、日常業務で使いこなしている
- Before/Afterの効果を数字で説明できる

### 事前準備
- [ ] Claude.ai のアカウント作成（Pro $20/月 または Team $30/人/月）
- [ ] ブラウザのブックマークに追加
- [ ] このプランを印刷またはNotionにコピー

### Day 2: プロンプト5技法を試す
**所要時間**: 20分

**やること**: 以下の5つの技法を1つずつ試す

**技法1: 役割指定**
```
あなたはベテランのマーケティングコンサルタントです。
中小企業のSNS運用について、最初の3ヶ月で取り組むべきことを教えてください。
```

**技法2: 具体的な条件指定**
```
以下の条件で企画書のアウトラインを作成してください。
- テーマ: 社内DX推進
- 対象: 経営層（ITリテラシーは低め）
- 分量: A4で2枚程度
- トーン: フォーマルだが分かりやすく
```

**技法3: 出力形式の指定**
```
以下の情報をMarkdownの表形式で整理してください。
[整理したい情報を貼り付ける]
```

**技法4: ステップバイステップ指示**
```
新規事業のアイデア評価を以下のステップで行ってください。
Step 1: 市場規模の推定
Step 2: 競合分析
Step 3: 自社の強みとの適合度
Step 4: リスク分析
Step 5: 総合評価（5段階）

アイデア: [あなたのアイデア]
```

**技法5: 例示（Few-shot）**
```
以下の例を参考に、お客様への返信メールを作成してください。

【良い例】
件名: ご提案資料のご送付
○○様、いつもお世話になっております。先日ご相談いただいた件について...

【今回の状況】
[状況を記載]
```

**チェックリスト**:
- [ ] 5つの技法をすべて試した
- [ ] 最も使えそうだと感じた技法をメモした

**効果測定シート**:

| 議事録整理 |             |             |        |           |            |
| 企画書作成 |             |             |        |           |            |

| 文章校正   |             |             |        |           |            |
| データ整理 |             |             |        |           |            |
| **合計**   |             |             |        |           | **___時間** |

**ROI計算**:
```
月間削減時間: ___時間
時給換算: ___円
月間効果: 削減時間 × 時給 = ___円
月間コスト: $20 × 約150円 = 3,000円（Pro）
月間ROI: (効果 - コスト) / コスト × 100 = ___%
```

**チェックリスト**:

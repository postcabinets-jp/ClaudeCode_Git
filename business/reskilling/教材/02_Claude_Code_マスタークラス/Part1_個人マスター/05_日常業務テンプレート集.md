## 1. 営業

### 1-1. 提案書の骨子作成

**目的**: 顧客向け提案書のドラフトを短時間で作成する

**プロンプト:**
```
<role>あなたはBtoB営業の提案書作成のプロフェッショナルです。</role>

以下の情報を元に、提案書の骨子を作成してください。

<client_info>
顧客名: [株式会社〇〇]
業種: [製造業/IT/小売など]
従業員数: [約〇〇名]
課題: [顧客が抱える課題を記載]
予算感: [年間〇〇万円程度]
決裁者: [役職]
</client_info>

<our_service>
提案するサービス: [サービス名・概要]
</our_service>

<output_format>
1. 表紙（タイトル案）
2. エグゼクティブサマリー（200字）
3. 貴社の課題認識（3点）
4. 提案内容（サービス概要、導入スコープ、スケジュール）
5. 期待される効果（定量的に）
6. 費用概算
7. 導入事例（簡潔に1社）
8. 次のステップ
</output_format>
```

**出力例**: 各セクションが見出し付きで構造化された提案書骨子が生成される

**カスタマイズのコツ**: `<client_info>`に顧客の最近のプレスリリースやIR情報を追加すると、より具体的な提案になる

### 1-2. 営業メール（初回アプローチ）

**目的**: 新規見込み客への初回コンタクトメールを作成する

**プロンプト:**
```
以下の条件で、新規見込み客への初回営業メールを3パターン作成してください。

宛先: [会社名]の[役職][氏名]様
当社サービス: [サービスの一言説明]
接点: [展示会で名刺交換/Webからの資料請求/紹介など]
ベネフィット: [顧客が得られるメリット]

条件:
- 件名は20字以内
- 本文は200字以内（短く、読まれることを最優先）
- 売り込み感を出さない（情報提供のスタンス）
- CTA: 15分のオンラインミーティングの提案
- 3パターン: (A)課題提起型 (B)事例紹介型 (C)情報提供型
```

**カスタマイズのコツ**: 顧客の業界特有の課題を入れると開封率が上がる

## テンプレートの使い方Tips

### 1. カスタマイズの基本
- `[ ]` 内を自分の情報に書き換える
- 不要なセクションは削除してOK
- 足りない条件は `<rules>` に追加する

### 2. Projects機能との連携
- よく使うテンプレートはProjectsのナレッジベースに保存
- カスタム指示に「このテンプレート集に従って回答してください」と書くと自動適用

### 3. テンプレートの改善サイクル
1. テンプレートを使う
2. 出力を確認する
3. 「もっとこうしてほしい」を制約に追加
4. 改良版テンプレートを保存する

### 4. 組み合わせの活用
- 例: 「営業日報」テンプレートの出力 → 「データ分析」テンプレートに入力 → 月次トレンド分析

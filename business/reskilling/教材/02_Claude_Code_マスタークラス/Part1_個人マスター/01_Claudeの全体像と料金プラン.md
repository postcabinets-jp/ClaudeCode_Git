### オープニング（2分）

皆さん、こんにちは。POST CABINETSのClaude Code マスタークラスへようこそ。

今日から始まるこの講座では、Anthropic社が開発したAIアシスタント「Claude」を、仕事で最大限に活用する方法をお伝えしていきます。

「ChatGPTは使ったことあるけど、Claudeは初めて」という方も、「すでにClaudeを使っているけど、もっと深く使いこなしたい」という方も、安心してください。基礎の基礎から、プロの活用法まで、段階的に進めていきます。

このModule 1では、まずClaudeとは何か、どんなモデルがあるのか、料金プランの選び方、そして他社のAIとの違いまで、全体像を20分で把握していただきます。

### セクション1: Claudeとは何か（4分）

#### Anthropic社について

Claudeを作っているのは、Anthropic（アンソロピック）というAI企業です。2021年にOpenAIの元メンバーが設立しました。

ここで大事なのは、Anthropicは「安全なAI」を最優先に掲げている会社だということです。単に賢いAIを作るだけでなく、「安全で、誠実で、役に立つAI」を理念としています。

これは私たちユーザーにとって、具体的にどういう意味があるでしょうか？

- **ハルシネーション（嘘）が少ない**: わからないことは「わかりません」と答える設計
- **指示を正確に守る**: 複雑な条件でも忠実に従う
- **プライバシーへの配慮**: ユーザーデータの学習利用に慎重

#### Claudeの名前の由来

「Claude」という名前は、情報理論の父クロード・シャノンから取られています。情報を正確に伝えるという使命が、名前にも込められているわけです。

> **講師メモ**: ここでclaude.aiを開いて画面を見せる。「これがClaudeのトップ画面です」と紹介。

### セクション2: モデルファミリーを理解する（5分）

Claudeには3つのモデルがあります。それぞれ特徴が異なるので、用途に応じて使い分けることが重要です。

#### Claude Opus 4.6 — 最高性能モデル

- **一言で**: 「最も賢い、考える力が最も強いモデル」
- **得意なこと**: 複雑な分析、長文の戦略立案、高度なコーディング、学術的な議論
- **コンテキスト**: 最大100万トークン対応（書籍数冊分を一度に処理可能）
- **速度**: やや遅め（深く考えるため）
- **使いどころ**: 重要な企画書、複雑な問題解決、深い分析が必要な場面

#### Claude Sonnet 4.6 — バランスモデル

- **一言で**: 「速さと賢さのベストバランス」
- **得意なこと**: 日常的な文書作成、メール対応、要約、一般的なコーディング
- **コンテキスト**: 200Kトークン（充分に長い）
- **速度**: 高速
- **使いどころ**: 日常業務の大半はこれで十分。迷ったらSonnet

#### Claude Haiku 4.5 — 軽量高速モデル

- **一言で**: 「とにかく速い、軽い作業向け」
- **得意なこと**: 短い文章の生成、簡単な質問回答、分類・仕分け作業
- **コンテキスト**: 200Kトークン
- **速度**: 最速
- **使いどころ**: 大量の定型処理、簡単な質問、チャットボット

> **講師メモ**: 「同じプロンプトを3モデルに投げて比較するデモ」を見せる。例:「日本のDX推進における課題を3つ挙げてください」

#### モデル選びの目安

```
重要度・複雑さ が高い → Opus 4.6
日常業務の大半 → Sonnet 4.6
大量処理・速度重視 → Haiku 4.5
```

### セクション3: 料金プラン徹底比較（5分）

#### プラン一覧（2026年3月時点）

| プラン | 月額 | モデル | メッセージ量 | 主な機能 |
|--------|------|--------|-------------|---------|
| **Free** | $0 | Sonnet 4.6 / Haiku 4.5 | 制限あり（少量） | 基本チャット |
| **Pro** | $20 | 全モデル（Opus含む） | 5倍 | Projects, Artifacts, ファイルアップロード |
| **Max** | $100〜$200 | 全モデル | 20倍〜無制限 | 大量利用向け、優先アクセス |
| **Team** | $30/人 | 全モデル | Proの2倍 | チーム共有、管理機能 |
| **Enterprise** | 要問合せ | 全モデル | カスタム | SSO, SCIM, 専用環境 |

#### 各プランの詳細解説

**Freeプラン**
- 「まず試してみたい」人向け
- メッセージ数に制限があり、混雑時はさらに制限される
- Projects機能は使えない
- **結論**: お試し用。業務利用には向かない

**Proプラン（$20/月 = 約3,000円）**
- **最もコスパが良い**。個人の業務利用はこれで十分
- Opus 4.6が使える（Freeでは使えない）
- Projects機能で「自分専用AI」が作れる
- Artifacts機能でコード・文書・図表を生成
- ファイルアップロード（PDF、画像、CSVなど）
- **結論**: ほとんどのビジネスパーソンにおすすめ

**Maxプラン（$100〜$200/月）**
- Proの上位版。メッセージ量が圧倒的に多い
- $100プラン: Proの20倍のメッセージ量
- $200プラン: 実質無制限
- **結論**: 1日中Claudeを使い倒すヘビーユーザー、開発者向け

**Teamプラン（$30/人/月）**
- チーム向け。Proの2倍のメッセージ量
- チームでProjectsを共有できる
- 管理者コンソール
- **結論**: 5人以上のチームで使う場合

**Enterpriseプラン**
- 大企業向け。SSO、SCIM、監査ログなどセキュリティ機能が充実
- 専用の環境で、データが学習に使われない保証
- **結論**: 50人以上、セキュリティ要件が厳しい組織

#### プラン選択フローチャート

```
START
  │
  ├─ まず試したいだけ → Free
  │
  ├─ 個人で業務利用 → Pro ($20)
  │    └─ 1日中使い倒す → Max ($100〜$200)
  │
  ├─ チームで使いたい
  │    ├─ 5〜50人 → Team ($30/人)
  │    └─ 50人以上 or セキュリティ要件厳 → Enterprise
  │
  └─ よくわからない → まずProから始めて、足りなくなったら上げる
```

> **講師メモ**: 「月3,000円で、毎日1〜3時間の時短ができるなら、時給換算で圧倒的にお得ですよね」と伝える。

### セクション4: 競合との比較 — なぜClaudeなのか（3分）

#### Claude vs ChatGPT vs Gemini

| 比較項目 | Claude | ChatGPT (GPT-4o) | Gemini |
|----------|--------|-------------------|--------|
| **長文理解** | ★★★★★ | ★★★☆☆ | ★★★★☆ |
| **指示追従性** | ★★★★★ | ★★★★☆ | ★★★☆☆ |
| **文章の自然さ（日本語）** | ★★★★★ | ★★★★☆ | ★★★☆☆ |
| **コーディング** | ★★★★★ | ★★★★★ | ★★★★☆ |
| **画像生成** | ☆☆☆☆☆（非対応） | ★★★★★ | ★★★★☆ |
| **検索連携** | ★★★☆☆ | ★★★★★ | ★★★★★ |
| **安全性・信頼性** | ★★★★★ | ★★★★☆ | ★★★★☆ |
| **Artifacts（成果物生成）** | ★★★★★ | ☆☆☆☆☆ | ☆☆☆☆☆ |

#### Claudeが特に強い領域

1. **長文処理**: 契約書、報告書、マニュアルなど大量のテキストを一度に処理できる。100ページ超のPDFも扱える
2. **指示への忠実さ**: 「箇条書き5つ、各100字以内、敬語で」といった細かい制約をきちんと守る
3. **日本語の品質**: 自然で読みやすい日本語を生成する。ビジネス文書の品質が高い
4. **Artifacts機能**: コード、文書、図表をその場で生成・プレビュー・編集できる唯一の機能
5. **安全性**: 不確かなことは「確認が必要です」と正直に言う

#### Claudeが苦手なこと（正直に）

- **画像生成**: Claudeは画像を生成できない（ChatGPTのDALL-Eのような機能がない）
- **リアルタイム検索**: Web検索機能はあるが、ChatGPTやGeminiほど統合されていない
- **プラグイン/拡張**: ChatGPTのようなプラグインエコシステムはない

> **講師メモ**: 「すべてのAIが万能なわけではありません。Claudeは文書業務に圧倒的に強い。画像生成が必要ならChatGPT、検索ならGemini、と使い分けるのがプロの仕事術です」

## 補足資料

### よくある質問（FAQ）

**Q: ClaudeにアップロードしたデータはAIの学習に使われますか？**
A: Freeプランでは利用規約上、学習に使われる可能性があります。Proプラン以上では、会話データはモデルの学習に使用されません。機密情報を扱う場合はPro以上を推奨します。

**Q: 日本語で使っても性能は落ちませんか？**
A: Claudeは日本語を非常に高いレベルで理解・生成できます。英語と比較して大きな性能差はありません。ビジネス文書の品質は、日本語ネイティブの評価でも高い水準です。

**Q: 会社で使う場合、セキュリティは大丈夫ですか？**
A: Team/Enterpriseプランでは、データの学習非利用の保証、SSO連携、監査ログなどのセキュリティ機能があります。情報システム部門と相談の上、プランを選択してください。

**Q: ChatGPTと併用してもいいですか？**
A: もちろんです。画像生成はChatGPT、長文分析はClaude、検索はGeminiなど、用途別に使い分けるのがプロの方法です。

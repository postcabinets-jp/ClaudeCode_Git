## 台本（講師用）

### オープニング（2分）

Module 4、Artifacts活用マスターです。

ここまで、Claudeとの対話の仕方を学んできました。プロンプトエンジニアリング、Projects機能。これだけでも十分強力ですが、今日はさらにもう一段階上の使い方をお伝えします。

Artifacts（アーティファクツ）機能。これは、Claudeがチャットの中で「成果物」を直接生成し、プレビューし、その場で編集できる機能です。

普通のAIチャットだと、コードや文書はテキストとしてチャット欄に流れるだけですよね。Artifactsを使うと、画面の右側に独立したパネルが開いて、そこにきちんとフォーマットされた成果物が表示されます。HTMLならブラウザプレビュー、コードなら実行可能、Markdownならフォーマット済み。

「AIにツールを作らせる」「AIにデザインを作らせる」――この発想転換が、今日の一番のテーマです。

---

### セクション1: Artifactsとは（3分）

#### 基本概念

Artifactsは、Claudeの回答の中で生成される「独立した成果物」です。

チャットのテキストとは別に、右側のパネルに表示され:
- そのまま使える形でプレビューされる
- 部分的な修正・追加指示ができる
- ダウンロードやコピーが簡単

#### 対応する成果物の種類

| 種類 | 拡張子/形式 | 用途 |
|------|------------|------|
| **Markdown** | .md | レポート、提案書、マニュアル |
| **HTML/CSS/JS** | .html | Webページ、ランディングページ、ツール |
| **React (JSX)** | .jsx/.tsx | インタラクティブなUI、ダッシュボード |
| **SVG** | .svg | ロゴ、アイコン、図解 |
| **Mermaid** | mermaid記法 | フローチャート、シーケンス図、ER図 |
| **コード** | .py, .js, etc. | スクリプト、プログラム |
| **CSV** | .csv | データテーブル、分析用データ |

> **講師メモ**: 「つまり、Claudeは文章だけでなく、Webサイトもダッシュボードも図表も作れるということです。しかもプレビュー付きで」

### セクション2: 活用シーン別ガイド（12分）

#### シーン1: Markdown/HTML → 提案書・レポート

**ユースケース**: 社内報告書、顧客向け提案書、月次レポート

**Before（通常のチャット）:**
- チャット欄にテキストが流れるだけ
- フォーマットが崩れやすい
- コピペして別のソフトで整形が必要

**After（Artifacts）:**
- きれいにフォーマットされたレポートが右パネルに
- 見出し、テーブル、箇条書きがプレビュー表示
- そのままHTMLとしてダウンロード可能

**実演プロンプト:**

```
以下の情報を元に、月次マーケティングレポートをHTML形式のArtifactで作成してください。

レポート情報:
- 期間: 2026年2月
- PV: 45,000（前月比+12%）
- CV: 180件（前月比+8%）
- CVR: 0.4%（前月比-0.02pt）
- 広告費: 150万円（ROAS 320%）
- トップページ: サービスA紹介ページ（PV 8,500）

デザイン要件:
- ヘッダーに会社ロゴ用のスペース
- メインカラー: #2563EB（青）
- フォント: sans-serif
- グラフは棒グラフ（CSS/JSで描画）
- レスポンシブ対応
- 印刷にも対応したレイアウト
```

#### シーン2: React Code → ダッシュボード・フォーム・ツール

**ユースケース**: 社内ツール、計算機、フォーム、簡易ダッシュボード

**これがArtifactsの真骨頂です。** Claudeに「ツールを作って」と頼むと、その場で動くWebアプリが生成されます。

**実演プロンプト:**

```
見積書を簡単に作成できるWebツールをReactのArtifactで作ってください。

機能:
1. 顧客情報入力（会社名、担当者名、住所）
2. 品目の追加（品名、単価、数量）→ 複数行追加可能
3. 小計・消費税・合計の自動計算
4. 見積書プレビュー（A4縦レイアウト）
5. 印刷ボタン

デザイン:
- シンプルでプロフェッショナル
- フォームは左側、プレビューは右側
- 消費税は10%固定（切り替え可能だとなお良い）
```

> **講師メモ**: 「これ、実際に動きます。ボタンを押したら印刷もできます。エンジニアでなくても、自分の業務ツールが作れる。これがArtifactsの衝撃です」

**Before**: ツールが欲しい → エンジニアに依頼 → 2週間待ち → 要件の齟齬 → 修正依頼...
**After**: ツールが欲しい → Claudeに依頼 → 5分で完成 → その場で修正 → 即使える

#### シーン3: SVG/Mermaid → フローチャート・組織図・ER図

**ユースケース**: 業務フロー図、組織図、プロジェクト構成図、データベース設計図

**実演プロンプト（Mermaid フローチャート）:**

```
以下の業務プロセスをMermaid形式のフローチャートArtifactで作成してください。

プロセス: 新規顧客の契約フロー
1. 問い合わせ受付
2. ヒアリング（電話 or 対面）
3. 提案書作成
4. 見積書作成
5. 顧客レビュー
   → 承認 → 契約書作成 → 締結
   → 修正依頼 → 3に戻る
   → お断り → 失注記録 → 終了
6. キックオフミーティング
7. プロジェクト開始

色分け:
- 営業担当のタスク: 青
- 顧客のアクション: 緑
- 管理部門のタスク: オレンジ
```

**実演プロンプト（SVG 組織図）:**

```
以下の組織図をSVGのArtifactで作成してください。

代表取締役
├── 営業部（部長: 田中）
│   ├── 第一営業課（5名）
│   └── 第二営業課（4名）
├── 制作部（部長: 鈴木）
│   ├── デザイン課（3名）
│   └── 開発課（6名）
├── マーケティング部（部長: 佐藤）
│   └── 企画課（3名）
└── 管理部（部長: 高橋）
    ├── 経理課（2名）
    └── 人事課（2名）

デザイン:
- 角丸の四角形
- 部署ごとに色分け
- 人数を小さく表示
- 全体は横幅800px程度
```

### セクション3: 「AIにツールを作らせる」発想転換（3分）

#### マインドセットの変化

従来: 「AIにテキストを書かせる」
これから: **「AIにツールを作らせる」**

この発想転換が、Claudeの活用レベルを一気に上げます。

#### 日常業務で作れるツールの例

| 困りごと | Claudeに作らせるもの |
|----------|---------------------|
| Excelの関数が複雑すぎる | ブラウザで動く専用計算ツール |
| 報告書のフォーマットがバラバラ | テンプレートジェネレーター |
| スケジュール調整が面倒 | 空き時間マッチングツール |
| 会議のタイマーがほしい | カスタムタイマーアプリ |
| 進捗管理をシンプルにしたい | カンバンボード |
| 社内用語がわからない | インタラクティブ用語集 |
| プレゼンの練習がしたい | スライドビューワー |
| 顧客情報を一覧で見たい | 検索・フィルタ付きデータテーブル |

> **講師メモ**: 「これは全部、エンジニアに依頼しなくても、Claudeに頼めば5〜10分で作れます。もちろん本格的なシステムはエンジニアが必要ですが、『自分だけが使う小さなツール』なら、Artifactsで十分です」

#### 修正・改善のコツ

Artifactsは一度作って終わりではありません。対話的に改善できます:

```
修正指示の例:
- 「テーブルの文字サイズを少し大きくして」
- 「色を企業カラー（#2563EB）に変えて」
- 「印刷時にヘッダーを非表示にして」
- 「入力欄にバリデーションを追加して」
- 「グラフにY軸の目盛りをつけて」
```

### クロージング（2分）

Artifacts機能のポイントをまとめます。

1. **Artifactsは「チャットの中で成果物を生成・プレビュー・編集」する機能**
2. **HTML、React、SVG、Mermaid、Markdownなど多彩な形式に対応**
3. **「AIにツールを作らせる」発想転換で、業務効率が劇的に向上**
4. **生成→修正→改善を対話的に繰り返せる**

Artifactsを使いこなせば、あなたは自分の仕事に必要なツールを、自分で・すぐに作れるようになります。エンジニアに頼まなくても、Excelマクロに悩まなくても。

### 演習1: HTMLレポートを作ってみよう（5分）

以下のプロンプトをそのまま入力し、Artifactが生成されるのを確認:

```
今月の営業成績レポートをHTML形式のArtifactで作成してください。

データ（仮のサンプル）:
- 訪問件数: 45件
- 新規商談: 12件
- 成約: 4件
- 売上: 850万円
- 目標達成率: 85%

デザイン:
- プロフェッショナルなビジネスレポート
- グラフ付き（目標vs実績の棒グラフ）
- A4で印刷できるレイアウト
```

### 演習2: 自分だけのツールを作ろう（10分）

以下の中から1つ選んで、Claudeに作ってもらいましょう:

**A. タスクタイマー**
```
ポモドーロ・テクニック用のタイマーをReactのArtifactで作ってください。
- 25分作業 / 5分休憩のサイクル
- 開始/停止/リセットボタン
- サイクル数のカウント
- 音声通知（ブラウザの通知API）
```

**B. 日報テンプレートジェネレーター**
```
日報を簡単に作成できるフォームをReactのArtifactで作ってください。
- 今日やったこと（複数行追加可能）
- 明日やること（複数行追加可能）
- 所感（テキストエリア）
- 生成ボタンを押すと、フォーマット済みテキストが表示される
- コピーボタン付き
```

**C. 会議アジェンダビルダー**
```
会議のアジェンダを作成するツールをReactのArtifactで作ってください。
- 会議名、日時、参加者を入力
- 議題を複数追加（議題名 + 時間配分 + 担当者）
- 合計時間の自動計算
- プレビュー表示（印刷可能）
```

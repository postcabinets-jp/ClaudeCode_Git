## 4. Claude導入プロセス — 4週間の全記録

### Week 1: Claude Team契約＋Brand Voice Project設定

**最初にやった最も重要なこと: ブランドボイスの定義**

高橋マネージャーがClaudeのProjectに設定した「KOMOREBIブランドボイス」:

```
あなたはスキンケアブランド「KOMOREBI」のマーケティングアシスタントです。

【ブランドコンセプト】
「木漏れ日のように、自然で優しいスキンケア」
- 天然由来成分95%以上
- 国内製造（愛媛の工場）
- サステナブルパッケージ（詰め替え推奨）
- ターゲット: 25〜40歳女性、ナチュラル志向、成分にこだわる

【ブランドボイス】
- トーン: 温かく、誠実で、知的。押し売りしない
- 一人称: 私たち / KOMOREBI
- 読者への呼びかけ: 「あなた」（敬語ベース、でも堅すぎない）
- 避ける表現: 「激安」「最強」「絶対」「奇跡」— 誇大な言葉は使わない
- 好む表現: 「丁寧に」「こだわりの」「素肌が喜ぶ」「自分らしく」
- 絵文字: 🌿🍃☀️🌸 は使ってOK。🔥💰🎉 は使わない

【NGワード】
- 「アンチエイジング」→「年齢に寄り添う」に言い換え
- 「肌荒れが治る」→ 薬機法抵触。「肌のキメを整える」等に
- 競合ブランド名を直接言及しない

【製品ライン】
1. モイスチャーローション（化粧水） ¥3,800
2. リッチクリーム（保湿クリーム） ¥4,500
3. クレンジングオイル ¥3,200
4. UVプロテクト（日焼け止め） ¥2,800
5. ナイトセラム（美容液） ¥5,500 ← 最新＆一番人気
```

**ナレッジとして追加**:
- 過去1年間のSNS投稿で反応が良かったもの30本
- 顧客レビュー（星5レビュー50件をスクリーンショット）
- 成分説明資料
- ブランドガイドライン（ロゴ使用規定・カラーパレット）

### Week 2: SNSコンテンツ量産体制構築

**「1.5時間が20分になった」という衝撃**

SNS担当の木村さんが初めてClaudeでInstagram投稿を作った日のこと。

> 木村さん:「いつも1本に1.5時間かかっていたInstagram投稿が、20分でできた。しかもキャプションの質が、正直に言って自分が書くより良い。ブランドの声がちゃんと出てる。最初は認めたくなかったけど（笑）」

**この週で構築した"量産体制"**:

1. **月曜朝**: Claudeに「今週のSNSカレンダー」を生成させる
2. **各投稿**: テーマと画像の概要を入力 → Claude がキャプション＋ハッシュタグ生成
3. **人間の作業**: 画像選定/撮影 ＋ Claudeの出力をブランドトーンで微調整
4. **予約投稿**: Buffer等のツールでスケジューリング

**結果**: SNS投稿が**週3本 → 毎日1〜2本**に。作業時間は逆に減少。

## 9. 注意点: ブランドボイスを守るために

### 「Claudeっぽさ」を消す3つのルール

KOMOREBIのチームが運用の中で確立したルールです。

**ルール1: 最初の一文は必ず人間が書く**
> Claudeの出力をそのまま使うと、どうしても"AI感"が出る。最初の一文だけは自分の言葉で書く。そうすると全体の温度が変わる。

**ルール2: ブランド固有のエピソードを差し込む**
> 「愛媛の工場で、職人の山本さんが一つひとつ手作業で…」こういうリアルなエピソードはAIには書けない。人間が足す。

**ルール3: 週1回「ブランドボイスチェック会」**
> 毎週金曜に15分、今週の投稿を全員で見返す。「これはKOMOREBIっぽくないね」という投稿があれば、なぜそうなったかを分析して、Projectの指示を更新する。

## 講師ノート

### 冒頭（2分）

> 「D2Cブランドのマーケティングチーム、5人しかいません。Instagram、X、TikTok、ブログ、メルマガ……全部やりたいけど、物理的に手が足りない。この状況、心当たりがある方は多いんじゃないでしょうか。今日は、そんなチームがClaudeを使ってどう変わったかをお話しします」

### 本編のハイライト（18分）

> 「このケースの最大のポイントは"ブランドボイスの設定"です。Week 1でここに時間をかけたのが成功の鍵でした。
>
> D2Cブランドにとって、"声"はブランドの生命線です。AIが書いたコンテンツでブランドの温度感が崩れたら、ファンは離れます。だから、まずProjectにブランドボイスを徹底的に教え込む。トーン、NGワード、好む表現、製品情報。全部入れる。
>
> その上でClaudeに書かせると、驚くほどブランドに合ったコンテンツが出てくる。SNS担当の木村さんは"自分が書くより良い"と言いました。でも、これは木村さんが不要になったという意味ではありません。木村さんは"何を投稿するか"という戦略に集中できるようになったんです。
>
> コンテンツの量が3倍になっても、ブランドの声が保てる。この両立ができるのが、Claude活用の真髄です」

### 締め（5分）

> 「最後に一つ。Claudeで量産できるようになると、"何でも投稿したくなる"衝動が出ます。でも、投稿の量より"ブランドとして何を伝えたいか"が先です。戦略なきコンテンツ量産は、ただのノイズになります。
>
> Claudeは制作の時間を解放してくれます。解放された時間で、"なぜこのブランドが存在するのか"を深く考える。それが、本当のマーケティング変革です」

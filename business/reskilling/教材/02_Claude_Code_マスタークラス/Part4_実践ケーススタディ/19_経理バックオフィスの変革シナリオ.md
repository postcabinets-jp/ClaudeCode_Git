## 2. Before — 月末月初の「地獄の5日間」

### 経理課長・渡辺さんの月末

> 「毎月25日を過ぎると胃が痛くなるんです。月次クローズまでの5日間は残業が当たり前。3人でやってますが、1人でも休んだら回らない。社長からは"もっと早く数字を出せ"と言われるし、現場からは"また経費精算の催促ですか"と嫌がられるし……」

### 月次クローズの流れ（Before）

```
1日目: 売上・仕入データ確認、未着請求書の催促
2日目: 経費精算処理（紙の領収書をひたすら入力）
3日目: 減価償却・引当金計算、仕訳入力
4日目: 試算表作成、部門別P/L作成、差異チェック
5日目: 経営報告資料作成（PowerPoint 15枚）、社長報告
```

### 数字で見る「Before」

| 業務 | 所要時間 | 頻度 | 月間合計 |
|------|----------|------|----------|
| 月次クローズ全体 | 5営業日（3人×8h×5日） | 月1回 | 120時間 |
| 請求書受領・確認・入力 | 8時間/週 | 毎週 | 32時間 |
| 経費精算処理 | 4時間/週 | 毎週 | 16時間 |
| 経営報告資料作成 | 8時間 | 月1回 | 8時間 |
| 予算差異分析 | 6時間 | 月1回 | 6時間 |
| 税理士対応・資料準備 | 4時間 | 月1回 | 4時間 |
| **合計（チーム3名分）** | | | **186時間/月** |

### 経理担当・小林さんの本音

> 「請求書の入力作業って、本当に人間がやる仕事なのかなと思うことがあります。でも、数字を間違えたら大変だし、結局自分の目で確認するしかない。Excelで数式組んでるけど、イレギュラーな仕訳は毎回悩む」

## 4. Claude導入プロセス — 4週間の全記録

### Week 1: 経費分析・レポートテンプレート

**まず"書く仕事"から始めた**

渡辺課長の判断で、最初は「数字を扱う作業」ではなく「文章を書く作業」から始めました。

> 「経理の人間は数字に慎重です。"AIが仕訳を間違えたらどうする"という恐怖がある。だから最初は、経営報告のコメント作成や分析レポートの文章化から入りました」

**最初のProjectを設定**:

```
あなたは製造業（精密部品）の経理部門のアシスタントです。

【会社概要】
- 自動車部品製造（Tier 2サプライヤー）
- 年商20億円、経常利益率5%
- 主要取引先: トヨタ系Tier 1が60%、ホンダ系が25%、その他15%
- 従業員150名、うち製造現場110名

【経理部門の業務】
- 月次決算（5営業日でクローズ）
- 部門別P/L（製造・営業・管理の3部門）
- 原価計算（標準原価 vs 実際原価の差異分析）
- 予算管理（四半期ごとのローリング予算）

【レポートのルール】
- 数字は千円単位（百万円は使わない）
- 前年同月比・予算比の両方を記載
- 差異が±5%を超える項目には理由コメントをつける
- 社長向けレポートは「結論→データ→アクション提案」の順
```

---

### Week 2: 請求書読み取り→分類ワークフロー

**Claudeに「仕訳の下書き」をさせる**

経理担当の小林さんが、受け取った請求書の情報をClaudeに入力して仕訳の下書きを作るワークフローを構築しました。

**ワークフロー**:
1. 請求書の主要情報を入力（取引先、金額、内容、日付）
2. Claudeが勘定科目を提案＋仕訳ドラフトを作成
3. 人間が確認→会計ソフトに入力

**「迷う仕訳」がなくなった効果**

> 小林さん:「月に10件くらい"これ何の科目だっけ"と迷う仕訳があったんです。Claudeに聞くと、根拠つきで科目を提案してくれる。しかも過去の類似取引との整合性も考えてくれる。迷う時間がゼロになりました」

---

### Week 3: 月次レポート自動生成

**月次クローズのゲームチェンジャー**

試算表のデータ（数字だけ）をClaudeに入力すると、以下を自動生成:

- 部門別P/Lサマリー
- 前年同月比・予算比の差異分析コメント
- 重要KPI（売上総利益率、労務費率、在庫回転率等）のトレンド分析
- 社長向けエグゼクティブサマリー（A4 1枚）

**Before**: 数字をExcelで計算→PowerPointに転記→コメントを書く = **8時間**
**After**: 数字をClaudeに入力→生成されたレポートを確認・調整 = **1.5時間**

---

### Week 4: 予算差異分析・経営報告自動化

**最終週で「経営参謀」への第一歩を踏み出した**

Claudeに四半期の実績データを入力し、予算との差異分析を依頼。単なる数字の差異ではなく、「なぜ差異が生じたか」「来四半期のリスクと機会」までコメントさせるようにした。

> 渡辺課長:「社長が驚いてましたね。"経理からこんな分析が出てくるのは初めてだ"と。いや、以前もやりたかったんです。時間がなかっただけで……」

| 項目 | 月額 | 年額 |

| 効果項目 | 算出方法 | 年間効果 |

### ROI

```
ROI = (2,400,000 - 564,000) ÷ 564,000 × 100 = 325%

投資回収期間 = 564,000 ÷ (2,400,000 ÷ 12) = 2.8ヶ月
```

> **経理部門のROIは驚異的です。少人数で大量の定型業務をこなす部門ほど、Claude導入の効果は大きくなります。**

## 8. 1年後の姿 — 「守りの経理」から「攻めの経営参謀」へ

### 経理部門の役割が変わった

> 渡辺課長:「以前の経理は"過去の数字を正確に記録する部門"でした。今は"未来の数字を予測して経営にアドバイスする部門"になりつつあります。月次クローズが2.5日で終わるようになって、残りの時間で原価分析や利益シミュレーションができるようになった」

### 具体的に何が変わったか

1. **月次報告が翌月3営業日目に出る** → 社長の経営判断が2日早くなった
2. **予算シミュレーション** → 「材料費が10%上がったら利益はどうなるか」をClaudeで即座に分析
3. **キャッシュフロー予測** → 毎週の資金繰り見通しをClaudeに作成させ、資金ショートリスクを早期発見
4. **税制改正への対応** → Claudeに改正内容と自社への影響を分析させ、税理士との打ち合わせの質が向上

### 社長の評価

> 「渡辺に"AI使いたい"と言われたときは正直不安だった。数字を扱う部署だから。でも3ヶ月経って、経理からの報告の質が明らかに変わった。数字だけじゃなくて"だからどうすべきか"まで言ってくれるようになった。これは数字には表れないが、一番大きな変化だ」

## 9. 注意点: 経理×AI活用で絶対守るべきこと

### ルール1: 数字の最終確認は必ず人間がやる

Claudeは計算を間違えることがあります。特に:
- 消費税の端数処理
- 複合仕訳の貸借バランス
- 前期比較の数字の引用

**対策**: Claudeの出力は「下書き」と位置づけ、必ず会計ソフトの数字と突合する。

### ルール2: 機密性の高い情報の扱い

- 個人の給与データはClaudeに入力しない
- 取引先の与信情報は匿名化して入力
- Claude Proの場合、入力データはモデル学習に使用されない（利用規約確認）

### ルール3: 税務判断はClaudeに任せない

Claudeは税法の一般的な解釈を提示できますが、個別の税務判断は税理士に確認する。Claudeは「論点の整理」と「質問の準備」に使う。

## 講師ノート

### 冒頭（2分）

> 「経理部門でAIを使う、と聞くとどう思いますか？ "数字を間違えたら怖い" "税務に影響したらどうする"……そう思う方が多いでしょう。実は、だからこそ経理はAI活用の恩恵が最も大きい部門の一つなんです。今日は製造業の経理チーム3人がClaudeで何を変えたかをお話しします」

### 本編のハイライト（18分）

> 「このケースの鍵は"最初に文章から始めた"ことです。経理の人は数字に慎重です。だから最初から仕訳にClaudeを使おうとすると、抵抗感が強い。渡辺課長が賢かったのは、まず経営報告のコメント作成から始めたこと。
>
> 試算表の数字を入れると、"売上は前年同月比108%と好調。一方で材料費率が1.4ポイント悪化しており、アルミ価格の高騰が主因"といったコメントが出てくる。これ、以前は渡辺課長が夜遅くまで残って書いていた文章です。
>
> "文章作成なら間違えても修正すればいい"という安心感から入って、徐々に仕訳の下書き、差異分析へと範囲を広げていく。この段階的アプローチが、経理部門のAI導入では非常に重要です。
>
> そして最大の効果は、月次クローズが5日から2.5日になったこと以上に、"経理部門が経営に対して提言できるようになった"ことです。守りの部門から攻めの経営参謀へ。これは渡辺課長にとって長年の夢だったそうです」

### 締め（5分）

> 「最後に強調したいのは、Claudeは電卓の代わりではないということです。数字の正確性は人間が担保する。Claudeの価値は"数字の意味を言語化し、分析の切り口を提案すること"にあります。
>
> ROI 325%、回収2.8ヶ月。これは経理部門のClaude導入が極めてコスト効率が高いことを示しています。3人の経理チームで年間240万円の効果。経営者の皆さん、経理部門へのAI投資は、最もリターンが確実な投資の一つです」

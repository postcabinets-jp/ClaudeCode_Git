# 61. 安全衛生・リスクアセスメント
**Claude Code スペシャリストクラス — S10 製造・品質管理**
**POST CABINETS Inc.**

## この講義で学ぶこと

- リスクアセスメントの基本手順とClaudeによる効率化
- 危険源（ハザード）の体系的な特定方法
- KYT（危険予知トレーニング）シートの作成
- 安全教育資料の自動生成
- 労災報告書・安全パトロールチェックリストの作成

---

## 業務の概要

安全衛生管理は、労働者の安全と健康を確保するための活動全体を指す。労働安全衛生法により、事業者にはリスクアセスメントの実施が努力義務として課されている（一部業種では義務）。

| 活動 | 内容 | 頻度 |
|------|------|------|
| **リスクアセスメント** | 危険源の特定→リスク見積り→対策の優先順位付け | 新規設備導入時、作業変更時、定期的 |
| **KYT（危険予知トレーニング）** | 作業前に潜在的な危険を予知し、対策を話し合う | 毎日（作業前ミーティング） |
| **安全教育** | 雇入れ時教育、作業内容変更時教育、特別教育 | 法定・随時 |
| **安全パトロール** | 職場の安全状態を巡回点検 | 週次〜月次 |
| **労災報告** | 労働災害発生時の報告・記録 | 発生時 |
| **安全衛生委員会** | 安全衛生に関する審議 | 毎月 |

### リスクアセスメントの基本手順（厚生労働省指針準拠）
1. 危険性・有害性の特定（ハザードの洗い出し）
2. リスクの見積り（重篤度×可能性）
3. リスク低減措置の検討（優先順位に従い対策）
4. 結果の記録と周知

---

## 従来の課題

1. **危険源の見落とし** — ベテランほど「慣れ」で危険を感じなくなる
2. **リスク見積りの属人性** — 同じ作業でも評価者によってリスクレベルが異なる
3. **KYTのマンネリ化** — 毎日同じ内容で形骸化し、新たな危険に気づけない
4. **安全教育資料の作成負担** — 法定教育の内容を整理するだけでも大変
5. **労災報告書の作成に不慣れ** — 発生頻度が低いため、書き方が分からない
6. **パトロールの形骸化** — チェックリストが古い、確認項目が現状に合わない

---

## Claudeによる自動化・効率化

### 1. 危険源の体系的な洗い出し
- 作業内容・設備情報を入力すると、考えうる危険源をカテゴリ別にリストアップ
- 厚生労働省の「危険性又は有害性の分類」に基づいた網羅的な検討

### 2. リスクアセスメントシートの作成
- 危険源ごとにリスクの見積り（重篤度×頻度×可能性）を支援
- リスクレベルに応じた対策の優先順位を提案

### 3. KYTシートの自動生成
- 作業内容を入力すると、想定される危険とその対策を含むKYTシートを生成
- バリエーションを増やし、マンネリ化を防止

### 4. 安全教育資料の作成
- 法定教育項目に沿った教育資料のテンプレートを生成
- 災害事例を基にしたケーススタディ教材の作成

### 5. 労災報告書の作成支援
- 事故の状況を入力すると、労災報告書（様式23号・24号）の記述案を生成
- 再発防止策の検討を支援

### 6. 安全パトロールチェックリストの更新
- 職場環境に応じたチェックリストを自動生成
- 季節要因（夏の熱中症、冬の凍結等）を反映

## プロンプト例（5つ以上、コピペで使えるレベル）

### プロンプト1: リスクアセスメントの実施

```
あなたは労働安全衛生の専門家（安全衛生コンサルタント）です。
以下の作業についてリスクアセスメントを実施してください。

【作業名】[例：天井クレーンによる重量物の吊り上げ・運搬作業]
【作業場所】[例：製造工場 組立エリア]
【使用設備】[例：天井クレーン（定格荷重5t）、玉掛け用ワイヤーロープ]
【作業内容の概要】
[例：鋼製フレーム（約2t）をクレーンで吊り上げ、
組立ステーションまで約15m運搬し、据え付ける]
【作業者】[例：クレーン運転士1名、玉掛け作業者2名、合図者1名]

以下の形式でリスクアセスメントシートを作成してください:

| No | 作業ステップ | 危険源（ハザード） | 危険事象（何がどうなる） | 重篤度(1-5) | 可能性(1-5) | リスクレベル | リスク低減措置 | 措置後リスク |

重篤度の基準:
5=死亡・永久障害  4=重傷（休業1ヶ月以上）  3=中程度（休業1週間以上）
2=軽傷（休業1週間未満）  1=微傷（不休）

可能性の基準:
5=確実に起こる  4=起こりやすい  3=起こりうる
2=まれに起こる  1=ほぼ起こらない

リスクレベル = 重篤度 × 可能性
20-25=直ちに作業中止  12-19=早急に対策  6-11=計画的に対策
1-5=現状維持（経過観察）

リスク低減措置は以下の優先順位で検討:
1. 本質的対策（危険源の除去・代替）
2. 工学的対策（ガード、インターロック等）
3. 管理的対策（手順書、教育、標識等）
4. 個人用保護具（PPE）
```

## 注意点

1. **リスクアセスメントは現場で実施** — Claudeでの事前検討は有効だが、実際の現場確認は必須。机上では見えない危険がある
2. **法令遵守が最優先** — 労働安全衛生法、労働安全衛生規則、各種特別規則（有機則、特化則等）の要求事項はClaude任せにしない
3. **災害事例は最新のものを使う** — Claudeの知識は訓練データの時点まで。最新の災害事例は厚生労働省「職場のあんぜんサイト」等で確認
4. **労災報告は正確に** — 労災かくしは犯罪（50万円以下の罰金）。事実を正確に報告すること
5. **KYTは参加者の当事者意識が重要** — Claudeで作ったシートをそのまま読み上げるのではなく、参加者自身が考える時間を確保する
6. **化学物質のSDS（安全データシート）はメーカー発行のものを使用** — ClaudeにSDSの内容を作らせない

---

## まとめ

- リスクアセスメントの危険源洗い出しは、Claudeにより網羅性が大幅に向上する
- KYTシートは毎日異なるパターンを生成でき、マンネリ化を防止
- 安全教育資料は法定項目をカバーした構成で効率的に作成可能
- 安全パトロールチェックリストは、季節要因や最近のヒヤリハットを反映した動的な更新が可能
- 安全に関わる最終判断（リスクの許容可否、対策の妥当性）は必ず安全衛生管理者・有資格者が行うこと

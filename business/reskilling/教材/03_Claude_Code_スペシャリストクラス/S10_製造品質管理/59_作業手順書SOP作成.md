## 業務の概要

SOP（作業手順書）は、作業の標準化と品質の均一化を実現するための基本文書である。

| 項目 | 内容 |
|------|------|
| **目的** | 誰がやっても同じ品質・安全・効率で作業できるようにする |
| **対象** | 製造作業、検査作業、設備操作、清掃・保全、荷受け・出荷等 |
| **作成者** | 現場のベテラン作業者＋品質管理担当者 |
| **利用者** | 作業者（新人からベテランまで）、教育訓練担当者 |
| **管理** | 版数管理、定期見直し、変更時の教育訓練 |

### SOPに含めるべき要素
1. 作業の目的と適用範囲
2. 必要な資格・力量
3. 使用する設備・工具・材料
4. 安全上の注意事項（PPE等）
5. 作業手順（ステップごと）
6. 品質チェックポイント
7. 異常時の対応
8. 記録の方法

---

## 従来の課題

1. **作成に時間がかかる** — 1本のSOPに数日〜数週間、全工程だと数十本が必要
2. **暗黙知の文書化が困難** — ベテランの「コツ」「勘所」が言語化されない
3. **更新が追いつかない** — 工程変更しても旧版のSOPが現場に残る
4. **新人に伝わらない** — 文字だらけで読まれない、専門用語が多い
5. **多言語対応の負担** — 外国人技能実習生向けに翻訳が必要だが、コストと時間がかかる
6. **教育訓練との連携不足** — SOPは作ったが教育したかどうかの記録が不十分

---

## Claudeによる自動化・効率化

### 1. SOPの構成設計とテンプレート化
- 工程情報を入力すると、SOPの構成（見出し・ステップ数）を自動提案
- 業界・規格に応じたテンプレートを生成

### 2. 暗黙知の言語化支援
- ベテランの口述をテキスト化し、SOPの文章に変換
- 「感覚的な表現」を「数値・条件による判定基準」に変換

### 3. 写真・図の指示書作成
- 各ステップで必要な写真・図の撮影指示書を自動生成
- 写真のキャプション、注釈テキストを作成

### 4. 多言語翻訳
- 日本語SOPを英語・中国語・ベトナム語・インドネシア語等に翻訳
- 技術用語の対訳表を同時に生成

### 5. 改訂管理支援
- 変更箇所の差分を明確にした改訂版を作成
- 改訂履歴・改訂理由の記述を自動生成

### 6. 教育訓練記録の連動
- SOPに基づく理解度テスト問題を自動生成
- 教育訓練記録のテンプレートを作成

## プロンプト例（5つ以上、コピペで使えるレベル）

### プロンプト1: SOPの新規作成

```
あなたは製造現場の作業標準化の専門家です。以下の情報からSOPを作成してください。

【作業名】[例：CNC旋盤によるシャフト加工]
【工程】[例：加工工程]
【対象設備】[例：CNC旋盤 MAZAK QT-250]
【加工品】[例：ステンレス製シャフト φ30×L200mm]
【対象者】[例：CNC旋盤操作資格保有者]

以下の構成で作成してください:

1. 文書管理情報（文書番号、版数、作成日、承認欄）
2. 目的
3. 適用範囲
4. 必要な力量・資格
5. 使用設備・工具・測定器一覧（表形式）
6. 使用材料・消耗品一覧
7. 安全上の注意事項
   - 必要なPPE（保護具）
   - 禁止事項
   - 緊急時の対応
8. 作業手順（ステップ形式）
   各ステップに以下を記載:
   - ステップ番号
   - 作業内容（具体的に）
   - ポイント/コツ（💡マーク）
   - 品質チェックポイント（✅マーク）
   - 写真/図の指示（📷マーク：何を撮影すべきか）
   - 所要時間目安
9. 品質確認項目一覧（表形式）
10. 異常発生時の対応フロー
11. 関連文書
12. 改訂履歴
```

### プロンプト2: 暗黙知の言語化

```
あなたは製造現場の技術伝承の専門家です。
以下のベテラン作業者の口述内容を、SOPに記載可能な明確な手順に変換してください。

【作業名】[例：溶接品質の目視判定]
【口述内容】
「溶接ビードを見て、まず色を見るんや。きれいな銀色ならOK。
黒っぽいのは酸化してるから不良。幅はだいたい均一で、
波打ちが細かく揃ってたらええ。アンダーカットは指で触って
引っかかるようならアウト。スパッタは多すぎたらあかんけど、
まぁちょっとくらいはしゃーない。」

以下の形式に変換してください:
1. 各判定項目を個別のステップに分解
2. 「だいたい」「ちょっと」等の曖昧表現を数値基準に変換
   （数値化が難しい場合は限度見本との比較方式を提案）
3. 合格/不合格の判定基準を明確化
4. 各項目の確認順序と所要時間
5. 判定に迷った場合のエスカレーション先

判定基準表（表形式）:
| 判定項目 | 確認方法 | 合格基準 | 不合格基準 | 判定困難時の対応 |
```

### プロンプト3: 多言語SOP作成

```
あなたは技術翻訳の専門家です。以下の日本語SOPを指定言語に翻訳してください。

【翻訳先言語】[例：ベトナム語 / 英語 / 中国語（簡体字）]

【日本語SOP】
（SOPの全文を貼り付け）

翻訳にあたっての要件:
1. 技術用語は以下の方針で翻訳:
   - 設備名・型番: そのまま（翻訳しない）
   - 一般的な技術用語: 対象言語の標準的な訳語を使用
   - 社内用語: 日本語をカッコ書きで併記
2. 安全に関する注意事項は、特に正確に翻訳
3. 数値・単位はそのまま保持
4. 文化的な配慮（例：色の表現、慣用句）
5. 翻訳後の文書の末尾に「技術用語対訳表」を付ける

| 日本語 | [対象言語] | 英語（参考） | 備考 |

6. 翻訳の確信度が低い箇所には【要確認】マークを付ける
```

### プロンプト4: 写真・図の撮影指示書

```
あなたはSOPの視覚化の専門家です。
以下のSOPの各ステップについて、撮影が必要な写真・図の指示書を作成してください。

【SOP名】[例：プレス機段取り替え作業手順書]
【作業ステップ一覧】
1. [ステップ1の概要]
2. [ステップ2の概要]
3. [ステップ3の概要]
...

各ステップについて以下を指示してください:
| ステップ | 写真/図の種類 | 撮影対象 | 撮影アングル | 必要な注釈 | OK/NG例の要否 | 備考 |

追加指示:
1. 安全上重要なステップには「⚠️ 必須撮影」マークを付ける
2. 判定基準に関わる箇所は「OK例」と「NG例」の両方を撮影指示
3. 写真内に記載すべき寸法線・矢印・テキストの指示
4. 推奨する写真解像度とファイル命名規則
```

### プロンプト5: SOP改訂管理

```
あなたは文書管理の専門家です。以下のSOPの改訂を支援してください。

【文書名】[SOP名]
【現行版数】[Rev.X]
【改訂理由】[例：工程変更に伴い、ステップ5の作業順序を変更]

【現行版の該当箇所】
（変更前の文章を貼り付け）

【変更したい内容】
（変更内容を説明）

以下を作成してください:
1. 改訂後の文章（変更箇所を■で囲んで明示）
2. 改訂履歴（改訂箇所・理由の記述）
3. 変更の影響範囲分析
   - 品質への影響:
   - 安全への影響:
   - 他の文書への影響:
4. 改訂に伴う教育訓練計画
   - 対象者
   - 教育内容
   - 教育方法（OJT/座学/eラーニング）
   - 理解度確認方法
5. 改訂通知文（現場掲示用、A4 1枚）
```

### プロンプト6: 理解度テスト問題の作成

```
あなたは製造現場の教育訓練担当者です。
以下のSOPに基づき、作業者の理解度を確認するテスト問題を作成してください。

【SOP名】[SOP名]
【SOP内容】
（SOPの全文または要約を貼り付け）

以下の形式で出題してください:
1. 選択式問題: 5問（4択、正解は1つ）
   - 安全に関する問題: 2問以上
   - 品質チェックポイントに関する問題: 2問以上
   - 手順に関する問題: 1問以上
2. 穴埋め問題: 3問（数値や重要キーワード）
3. 実技確認チェックリスト: 5項目
   | 確認項目 | 評価基準 | 合格/不合格 | 備考 |

合格基準: 筆記80%以上 + 実技全項目合格
不合格時の再教育方法も提案してください。
```

## 注意点

1. **現場の実態と乖離しないこと** — Claudeで作成したSOPは必ず現場作業者のレビューを受ける。机上の手順書は使われない
2. **安全に関する記述は慎重に** — 安全上の注意事項や緊急時対応は、安全衛生担当者の確認を必ず受けること
3. **多言語翻訳の品質確認** — Claudeの翻訳は、対象言語のネイティブスピーカー（できれば技術者）の確認を受けること。安全に関わる誤訳は重大事故につながる
4. **写真・図は実際に撮影が必要** — Claudeは撮影指示は出せるが、実際の写真は現場で撮影する。生成AIの画像は使用しない
5. **改訂時の教育訓練を忘れない** — SOPを改訂しただけでは不十分。対象者への教育と理解度確認まで実施して初めて有効

---

## まとめ

- SOPの構成設計からステップ記述まで、Claudeが初稿を大幅に短縮（1本あたり数日→数時間）
- ベテランの暗黙知を言語化する際、Claudeが「曖昧な表現→定量的な基準」への変換を支援
- 多言語対応は、技術用語対訳表の同時生成により翻訳の一貫性を確保
- 教育訓練との連動（理解度テスト自動生成）で、SOPの実効性を高める
- 最終的な内容の正確性は現場作業者と管理者が確認すること。Claudeは「優秀なテクニカルライターのアシスタント」として活用する

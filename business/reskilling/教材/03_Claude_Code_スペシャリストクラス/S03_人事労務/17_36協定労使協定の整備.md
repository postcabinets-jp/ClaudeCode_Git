## 業務の概要

### 36協定（サブロク協定）とは

労働基準法第36条に基づく「時間外労働・休日労働に関する協定」。法定労働時間（1日8時間、週40時間）を超えて労働させる場合、または法定休日に労働させる場合に、労使間で締結し、労働基準監督署に届け出なければならない。

> **36協定なしに時間外労働をさせると、労基法第32条違反で6ヶ月以下の懲役または30万円以下の罰金**

### 時間外労働の上限規制（2019年4月施行）

| 区分 | 上限 |
|------|------|
| 原則 | 月45時間、年360時間 |
| 特別条項付き | 年720時間以内 |
| 特別条項発動月 | 月100時間未満（休日労働含む） |
| 複数月平均 | 2〜6ヶ月平均で80時間以内（休日労働含む） |
| 特別条項の回数 | 年6回まで |

### 主な労使協定の種類

| 労使協定 | 根拠条文 | 届出義務 |
|----------|----------|----------|
| 36協定（時間外・休日労働） | 労基法第36条 | **必要** |
| 1年単位の変形労働時間制 | 労基法第32条の4 | **必要** |
| 1ヶ月単位の変形労働時間制 | 労基法第32条の2 | **必要** |
| フレックスタイム制（清算期間1ヶ月超） | 労基法第32条の3 | **必要** |
| 賃金控除協定（社宅費等） | 労基法第24条 | 不要 |
| 年次有給休暇の計画的付与 | 労基法第39条第6項 | 不要 |
| 年次有給休暇の時間単位付与 | 労基法第39条第4項 | 不要 |
| 育児・介護休業の適用除外 | 育介法第6条等 | 不要 |
| 事業場外みなし労働時間制 | 労基法第38条の2 | 要（みなし時間が法定超の場合） |
| 専門業務型裁量労働制 | 労基法第38条の3 | **必要** |

## 従来の課題

| 課題 | 詳細 |
|------|------|
| 協定の期限切れ | 36協定の有効期間が切れたまま時間外労働をさせているケース |
| 特別条項の形骸化 | 特別条項の発動条件が曖昧で、常態的に月45時間を超えている |
| 上限規制への理解不足 | 月100時間未満、複数月平均80時間以内のルールを管理できていない |
| 労使協定の一覧管理ができていない | どの協定がいつ締結され、いつ届出されたか把握できていない |
| 労働者代表の選出に問題 | 管理監督者が代表になっている、使用者が指名している等 |

## Claudeによる自動化・効率化

### 1. 36協定のドラフト作成
業種・業務内容・想定残業時間を入力し、適切な上限時間と特別条項を設計。

### 2. 特別条項の設計支援
特別条項の発動事由、手続き、上限時間を法令に準拠して設計。

### 3. 労使協定一覧の管理テンプレート
全労使協定の締結日・届出日・有効期限を一覧管理するシートを生成。

### 4. 違反リスクチェック
毎月の残業時間データから、上限規制違反の予兆を検出するチェックリストを作成。

### 5. 届出スケジュールの自動生成
各協定の有効期限から逆算した更新スケジュールを作成。

## 注意点・法的留意事項

### 36協定に関する重要ポイント
- **届出は事業場単位**: 本社・支店がそれぞれ別の事業場であれば、各事業場で締結・届出が必要
- **有効期間**: 1年間が望ましい（行政通達）。自動更新条項は認められない
- **届出前の時間外労働**: 36協定は届出が効力発生要件。届出前の時間外労働は違法
- **新様式の使用**: 2021年4月以降、新様式（様式第9号/第9号の2）を使用

### 上限規制違反の罰則
- **6ヶ月以下の懲役または30万円以下の罰金**（労基法第119条）
- 罰則は**使用者個人**と**法人の両方**に科される（両罰規定）
- 労基署の臨検で発覚するだけでなく、労災事故時に重要な証拠となる

### 労使協定の保存義務
- 36協定を含む労使協定は、その協定の有効期間中および期間経過後**3年間（経過措置。本則は5年間）**保存
- 労働者代表の選出記録も保管しておくことが望ましい

### 建設業・自動車運転業務等の適用猶予
- 2024年4月から建設業・自動車運転業務・医師にも上限規制が適用（一部特例あり）

### 管理監督者の取扱い
- 労基法第41条の管理監督者には36協定は不要（時間外・休日の規定が適用除外）
- ただし、深夜業の規制は適用される
- 「名ばかり管理職」問題に注意（実態で判断される）

> **重要**: 本教材の内容は一般的な実務ガイドラインです。36協定・労使協定の具体的な設計・届出については、必ず社会保険労務士または弁護士に確認してください。上限規制の違反は刑事罰の対象であり、慎重な対応が求められます。

## まとめ

- 36協定は**時間外労働の法的根拠**。未届で残業させると刑事罰の対象
- 上限規制は**絶対的上限**（月100時間未満、複数月平均80時間以内、年720時間以内）を超えてはならない
- 特別条項は**臨時的な事由**に限定し、年6回以内の発動ルールを設計する
- 労使協定は**一覧管理**し、有効期限切れを防ぐ。更新スケジュールを事前に策定する
- 労働者代表の選出は**適法な手続き**（使用者の指名は不可）を踏むことが必須
- Claudeは協定のドラフト作成、残業時間の違反チェック、スケジュール管理に活用できるが、**届出と実際の労務管理は人間の責任**

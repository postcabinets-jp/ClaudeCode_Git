## 業務の概要

### 退職手続きの全体フロー

```
退職届受理 → 退職日確定 → 引継ぎ計画作成 → 退職面談 → 最終出勤日
→ 社会保険資格喪失届 → 雇用保険資格喪失届・離職票 → 住民税切替届
→ 源泉徴収票交付 → 退職金計算・支払い → 貸与品回収 → 退職証明書交付
```

### 退職の種類

| 種類 | 概要 | 法的根拠 |
|------|------|----------|
| **自己都合退職** | 労働者からの申出による退職 | 民法第627条（期間の定めなし: 2週間前の申出） |
| **会社都合退職** | 解雇、退職勧奨、雇止め等 | 労基法第20条（30日前の予告 or 予告手当） |
| **合意退職** | 労使双方の合意による退職 | — |
| **定年退職** | 就業規則に定めた定年年齢に達した場合 | 高年齢者雇用安定法 |
| **契約期間満了** | 有期契約の期間満了 | 労基法第14条 |

### 退職に伴う届出・手続き一覧

| 手続き | 届出先 | 期限 |
|--------|--------|------|
| 社会保険資格喪失届 | 年金事務所 | 退職日翌日から5日以内 |
| 雇用保険資格喪失届 | ハローワーク | 退職日翌日から10日以内 |
| 離職証明書（離職票） | ハローワーク | 退職日翌日から10日以内 |
| 住民税特別徴収の異動届 | 市区町村 | 退職月の翌月10日まで |
| 源泉徴収票の交付 | 本人 | 退職後1ヶ月以内（目安） |
| 退職証明書の交付 | 本人 | 請求があれば遅滞なく |

## 従来の課題

| 課題 | 詳細 |
|------|------|
| 届出の期限超過 | 退職が急で手続きが間に合わない、担当者が期限を把握していない |
| 離職票の遅延 | 退職者が転職・失業給付の手続きを急いでいるのに、離職票の交付が遅れる |
| 引継ぎの不備 | 退職者の業務が引き継がれず、顧客対応に支障が出る |
| 退職金計算のミス | 勤続年数の計算間違い、税額控除の適用ミス |
| 貸与品の回収漏れ | PC・社員証・鍵等の回収が漏れ、情報漏洩リスクが残る |
| 退職面談の未実施 | 退職理由の真因が把握できず、同様の離職が繰り返される |

### プロンプト4: 退職面談（Exit Interview）の質問設計

```
退職面談の質問セットを設計してください。

【前提】
- 対象: 自己都合退職者
- 面談者: 人事担当者（直属の上司ではない）
- 目的: 退職の真因把握と組織改善への活用

以下の構成で:
1. アイスブレイク（2〜3問）
   - 緊張をほぐし、率直に話してもらうための導入

2. 退職理由の深掘り（5問）
   - 表面的な理由（転職先が決まった等）の裏にある真因を探る
   - 「何がなければ辞めなかったか」を明らかにする

3. 職場環境について（5問）
   - 上司・同僚との関係
   - 業務量・裁量・成長機会
   - 会社の文化・制度

4. 改善提案（3問）
   - 会社に変えてほしかったこと
   - 後任・チームへのアドバイス

5. クロージング（2問）
   - 将来的な関係性（アルムナイネットワーク等）
   - 言い残したこと

注意:
- 引き止めの場にしない（決定を尊重する姿勢）
- 秘密厳守を冒頭で伝える
- 回答は強制しない
- 面談記録のフォーマットも添付
```

## 注意点・法的留意事項

### 退職届・退職願の法的効力
- **退職届**: 一方的な意思表示。到達した時点で効力発生（撤回は原則不可）
- **退職願**: 合意退職の申込み。会社が承諾するまでは撤回可能
- **期間の定めのない契約**: 2週間前の告知で退職可能（民法第627条）
- **就業規則で1ヶ月前等を定めている場合**: 民法の2週間が優先するとの見解が有力だが、実務上は就業規則に従うことが望ましい

### 解雇の場合の注意
- **解雇予告**: 30日前の予告、または30日分以上の平均賃金の支払い（労基法第20条）
- **解雇制限**: 業務上の傷病による休業期間中、産前産後休業中は原則解雇不可（労基法第19条）
- **解雇権の濫用**: 客観的に合理的な理由を欠き、社会通念上相当でない解雇は無効（労契法第16条）

### 離職票に関する注意
- 離職理由は**退職者の失業給付に直結**する重要事項
- 会社都合を自己都合に偽ることは違法であり、ハローワークに異議申立てされるリスクがある
- **特定受給資格者**（会社都合等）と**特定理由離職者**（正当な理由のある自己都合）は給付日数が優遇される

### 退職後の競業避止義務
- 就業規則や誓約書で定めることは可能だが、**合理的な範囲**に限定される
- 期間（1〜2年が目安）、地域、職種の制限が過度であれば無効
- 代償措置（退職金の上乗せ等）がない場合は無効とされやすい

### 退職金の税務
- 退職所得は**分離課税**で税負担が軽減される
- **退職所得控除**: 勤続20年以下 = 40万円 × 勤続年数、20年超 = 800万円 + 70万円 ×（勤続年数 - 20年）
- 「退職所得の受給に関する申告書」を提出しない場合、退職金の20.42%が源泉徴収される

### 健康保険の選択肢
退職者には以下の選択肢を案内する:
1. **任意継続被保険者**: 退職日の翌日から20日以内に申請。最長2年間
2. **国民健康保険**: 退職日の翌日から14日以内に届出
3. **家族の被扶養者**: 収入要件（年収130万円未満等）を満たす場合

> **重要**: 本教材の内容は一般的な実務ガイドラインです。退職手続き、特に解雇や離職理由の判定については、必ず社会保険労務士または弁護士に確認してください。不適切な処理は労働紛争につながるリスクがあります。

## まとめ

- 退職手続きは**期限が厳格**。社会保険（5日以内）、雇用保険（10日以内）を必ず守る
- 離職票の**離職理由は正確に記載**する。虚偽の記載は法的リスクと退職者との信頼関係を損なう
- **退職面談**は組織改善の貴重な情報源。引き止めの場にせず、率直な声を聴く
- **引継ぎ計画書**を作成し、業務の空白を最小限にする。退職日から逆算したスケジュールが重要
- **退職金計算**は退職所得控除の適用を正しく行い、税額計算のミスを防ぐ
- 退職は「終わり」ではなく、**アルムナイ（卒業生）ネットワーク**として将来の関係性を維持する視点も大切
- Claudeはチェックリスト、計算検証、面談質問設計に活用できるが、**最終的な届出判断と法的対応は専門家に委ねる**

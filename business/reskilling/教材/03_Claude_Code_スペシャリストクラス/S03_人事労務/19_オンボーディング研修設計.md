## 業務の概要

### 入社手続きとオンボーディングの違い

| 区分 | 内容 | 目的 |
|------|------|------|
| **入社手続き** | 法的書類の提出、社会保険加入、備品配布 | 法令遵守、就労環境の整備 |
| **オンボーディング** | 会社文化の理解、業務習得、人間関係構築 | 早期戦力化、定着率向上 |

### オンボーディングの効果

- 適切なオンボーディングにより**新入社員の定着率が最大82%向上**（Brandon Hall Groupの調査）
- 戦力化までの期間が**平均2ヶ月短縮**
- 新入社員の**エンゲージメントスコアが30%以上向上**

### 入社手続きに必要な書類

| 書類 | 法的根拠 | 提出先 |
|------|----------|--------|
| 雇用契約書（労働条件通知書） | 労基法第15条 | 本人交付 |
| 社会保険資格取得届 | 健保法・厚年法 | 年金事務所 |
| 雇用保険資格取得届 | 雇用保険法 | ハローワーク |
| 扶養控除等申告書 | 所得税法 | 会社保管 |
| マイナンバー届出 | マイナンバー法 | 会社保管 |
| 住民税の特別徴収切替届 | 地方税法 | 市区町村 |
| 通勤経路届 | — | 会社保管 |
| 銀行口座届 | — | 会社保管 |

## 従来の課題

| 課題 | 詳細 |
|------|------|
| 手続き漏れ | 社会保険の届出期限（5日以内）を超過するケースが頻発 |
| 入社初日の体制不備 | PC・アカウント・座席が準備されておらず、新入社員が放置される |
| 研修の属人化 | 教える人によって内容がバラバラ。OJTの名のもとに放任 |
| メンター制度の形骸化 | メンターが忙しくてフォローできず、新入社員が孤立 |
| 早期離職 | 入社3ヶ月以内の離職が多く、採用コストが無駄に |

## 注意点・法的留意事項

### 入社手続きの法的義務
- **労働条件の明示義務**（労基法第15条）: 書面（電子メール等も可）で明示が必要
- **2024年4月の改正**: 就業場所・業務の変更の範囲の明示が義務化
- **社会保険の届出期限**: 資格取得届は事実発生から**5日以内**
- **雇用保険の届出期限**: 被保険者となった月の**翌月10日まで**

### 試用期間に関する注意
- 試用期間中も**労働契約は成立**している（解約権留保付き）
- 試用期間中の解雇も、14日を超えた場合は**解雇予告または解雇予告手当**が必要
- 試用期間の延長は、就業規則に根拠がなければ原則不可
- 本採用拒否は「通常の解雇よりは広い範囲」で認められるが、客観的に合理的な理由が必要（三菱樹脂事件判例）

### 研修中の労働時間
- 会社が命じた研修は**労働時間**に該当し、賃金の支払いが必要
- 自主参加であっても、実質的に参加が強制されている場合は労働時間
- 研修が所定労働時間外に及ぶ場合は**時間外手当**の支払いが必要

### 個人情報の取扱い
- 入社時に取得する個人情報は、利用目的を明示した上で取得
- マイナンバーは**厳格な安全管理措置**が必要（特定個人情報）
- 身元保証書は**5年を超える期間を定められない**（身元保証法）

> **重要**: 本教材の内容は一般的な実務ガイドラインです。入社手続きや労働条件の明示については、最新の法令を確認の上、社会保険労務士に相談してください。

## まとめ

- 入社手続きは**法定期限の厳守**が最重要。チェックリストで漏れを防止する
- オンボーディングは入社手続きとは別に、**戦力化と定着**を目的とした体系的なプログラムを設計する
- 研修カリキュラムは**座学・OJT・e-learningのブレンド**で、職種・レベルに応じて設計
- メンター制度は**メンターの負荷管理**が成否を分ける。適切な選定と支援体制が必要
- **90日プラン**は明確な到達目標とマイルストーンを設定し、定期的にチェックインする
- Claudeはチェックリスト、カリキュラム、90日プランの生成に強力だが、**人間同士の信頼関係構築は対面のコミュニケーションが不可欠**

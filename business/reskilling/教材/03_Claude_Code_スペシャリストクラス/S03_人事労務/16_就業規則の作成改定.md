## 業務の概要

### 就業規則とは

就業規則は、会社と従業員の間のルールブックである。労働条件や服務規律を明文化し、労使双方の権利義務を明確にする。

### 作成義務（労働基準法第89条）

**常時10人以上**の労働者を使用する事業場は、就業規則を作成し、所轄の労働基準監督署に届け出なければならない。

> 10人未満の事業場でも作成することが望ましい。トラブル防止と労務管理の基盤になる。

### 就業規則の全体構成

```
第1章  総則（目的、適用範囲、定義）
第2章  採用（採用手続き、試用期間、労働条件の明示）
第3章  服務規律（遵守事項、ハラスメント防止、副業）
第4章  勤務（勤務時間、休憩、休日、時間外労働）
第5章  休暇・休業（年次有給休暇、特別休暇、育児・介護休業）
第6章  賃金（賃金体系、計算方法、支払方法、昇給・賞与）
第7章  退職・解雇（退職手続き、解雇事由、定年）
第8章  表彰・制裁（表彰制度、懲戒の種類と手続き）
第9章  安全衛生（安全衛生管理、健康診断、ストレスチェック）
第10章 雑則（損害賠償、改定手続き）

【別規程】
- 賃金規程
- 育児・介護休業規程
- ハラスメント防止規程
- テレワーク規程
- 機密情報管理規程
```

### 就業規則に記載が必要な事項

| 区分 | 事項 |
|------|------|
| **絶対的必要記載事項** | 始業・終業時刻、休憩、休日、休暇、賃金の決定・計算・支払方法、昇給、退職に関する事項 |
| **相対的必要記載事項** | 退職手当、臨時の賃金・賞与、食費・作業用品の負担、安全衛生、職業訓練、災害補償、表彰・制裁、その他全労働者に適用される事項 |

## 従来の課題

| 課題 | 詳細 |
|------|------|
| ひな形のコピペ | インターネット上のテンプレートをそのまま使い、自社の実態と乖離 |
| 法改正への未対応 | 数年前の規則のまま運用され、現行法に違反している条文がある |
| 曖昧な条文 | 「会社が必要と認めた場合」等の抽象的な表現が多く、運用で揉める |
| 周知不足 | 作成しても従業員が内容を知らず、実質的に機能していない |
| 改定プロセスの不備 | 意見聴取や届出の手続きを踏まずに改定してしまう |

## Claudeによる自動化・効率化

### 1. 就業規則のドラフト作成
業種・規模・勤務形態を指定するだけで、法令準拠の就業規則ドラフトを生成。

### 2. 法改正対応チェック
既存の就業規則をClaudeに読み込ませ、最新の法改正に未対応の箇所を洗い出す。

### 3. 条文の解説・カスタマイズ支援
各条文の趣旨、判例上のポイント、自社向けのカスタマイズ案を提示。

### 4. 意見書・届出書類のドラフト
労働者代表の意見書や届出に必要な書類のひな形を生成。

### 5. 従業員向け説明資料の作成
専門用語を平易な言葉に変換し、従業員向けの説明資料を作成。

## 注意点・法的留意事項

### 就業規則の効力に関する原則
- **労働契約法第12条**: 就業規則で定める基準に達しない労働条件を定める労働契約は、その部分について無効
- **労働契約法第9条・10条**: 就業規則の不利益変更は原則として労働者の合意が必要。合意がない場合でも、変更の合理性があれば有効となり得る（秋北バス事件判例法理の明文化）
- **周知義務**: 就業規則は労働者に周知されていなければ効力を生じない（労契法第7条）

### 不利益変更の合理性判断基準
労働契約法第10条に基づき、以下の要素を総合的に判断:
1. 労働者の受ける不利益の程度
2. 労働条件の変更の必要性
3. 変更後の就業規則の内容の相当性
4. 労働組合等との交渉の状況
5. その他の就業規則の変更に係る事情

### 届出の注意点
- 届出がなくても就業規則は有効（周知されていれば効力あり）だが、届出義務違反は30万円以下の罰金
- 事業場ごとに届出が必要（本社で一括届出も可能だが要件あり）
- 労働者代表の意見が反対であっても、意見聴取の手続きを踏んでいれば届出可能

### Claudeで作成した就業規則の取扱い
- AIが生成した条文は**あくまでドラフト**として扱う
- 自社の実態・業種・規模に合わせたカスタマイズが必須
- 最終版は必ず社会保険労務士または弁護士のリーガルチェックを受ける

> **重要**: 本教材の内容は一般的な実務ガイドラインです。就業規則は企業の法的リスクに直結する重要文書です。作成・改定の際は必ず社会保険労務士または弁護士に相談してください。

## まとめ

- 就業規則は常時10人以上の事業場で**作成・届出が法的義務**。10人未満でも作成を推奨
- **絶対的必要記載事項**（労働時間、賃金、退職）は必ず網羅する
- 法改正は毎年のように行われるため、**定期的なチェック体制**を構築する
- 不利益変更には**合理性**が求められる。手続きの適正さが重要
- **労働者代表の選出**は適法に行い、意見聴取の手続きを確実に踏む
- Claudeはドラフト作成と法改正チェックに強力だが、**リーガルチェックは専門家に委ねる**

## 業務の概要

採用業務は、企業の成長を支える最も重要な人事機能の一つである。以下のフローで構成される。

1. **採用計画策定** — 事業計画に基づく必要人材の定義
2. **求人票作成** — 職種・業務内容・条件の明文化
3. **母集団形成** — 求人媒体・エージェント・リファラル等
4. **書類選考** — 応募書類のスクリーニング
5. **面接** — 構造化面接による適性評価
6. **内定・オファー** — 条件提示と合意形成
7. **入社準備** — オンボーディングへの接続

### 関連法規
- 職業安定法（求人票の記載義務）
- 男女雇用機会均等法（性別による差別禁止）
- 労働施策総合推進法（年齢制限の原則禁止）
- 個人情報保護法（応募者情報の取扱い）

## 従来の課題

| 課題 | 詳細 |
|------|------|
| 求人票の品質ばらつき | 部門ごとに書き方が異なり、応募者に正確な情報が伝わらない |
| 書類選考の属人化 | 担当者の主観で合否が決まり、基準が不透明 |
| 面接の非構造化 | 質問がその場の雰囲気で決まり、評価の信頼性が低い |
| オファー条件の不整合 | 口頭での条件提示が多く、後からトラブルになる |
| 採用プロセスの長期化 | 各ステップの遅延が積み重なり、優秀な候補者を逃す |

## Claudeによる自動化・効率化

### 1. 求人票の自動生成
職種・必須スキル・勤務条件を入力するだけで、法令準拠の求人票を生成。性別・年齢等による差別表現を自動チェック。

### 2. 書類スクリーニング基準の設計
職種ごとの必須要件・歓迎要件をマトリクス化し、客観的な評価基準を作成。

### 3. 構造化面接の質問セット生成
コンピテンシーに基づく行動面接（STAR手法）の質問を自動設計。

### 4. 評価シートの作成
面接官が統一基準で評価できるシートを、職種・レベルに応じて生成。

### 5. オファーレターの作成
労働条件通知書の要件を満たしたオファーレターのドラフトを自動生成。

## プロンプト例

### プロンプト1: 職種別求人票の作成

```
あなたは採用コンサルタントです。以下の条件で求人票を作成してください。

【職種】Webマーケティングマネージャー
【雇用形態】正社員
【勤務地】大阪府岸和田市（リモート可）
【給与レンジ】年収500万〜700万円
【必須スキル】
- Web広告運用経験3年以上
- GA4を用いたデータ分析経験
- チームマネジメント経験

【歓迎スキル】
- SNSマーケティング経験
- BtoB マーケティング経験

以下の形式で出力してください:
1. 募集職種名（魅力的なタイトル）
2. 仕事内容（具体的な業務を5つ以上）
3. 応募資格（必須/歓迎を分けて）
4. 給与・待遇
5. 勤務条件（勤務地、勤務時間、休日）
6. 選考プロセス
7. 会社の魅力（3つ）

※ 職業安定法に基づく明示義務事項を漏れなく含めること
※ 性別・年齢による制限表現を含めないこと
```

### プロンプト2: 書類スクリーニング基準の設計

```
以下の職種について、書類選考のスクリーニング基準を設計してください。

【職種】法人営業（SaaS）
【採用目的】新規開拓チームの増員

以下の形式で出力してください:
1. 必須要件チェックリスト（Yes/Noで判定できる項目を5つ以上）
2. 加点要素（配点付き、合計100点）
3. 足切りライン（何点以上で面接に進むか）
4. 書類選考NGとなるケース（具体例3つ以上）
5. 判断に迷った場合の対応ルール

※ 年齢・性別・出身校による差別的基準を含めないこと
```

### プロンプト3: 構造化面接の質問設計（STAR手法）

```
あなたは人事面接の専門家です。以下の職種・コンピテンシーに基づき、
構造化面接の質問セットを設計してください。

【職種】プロジェクトマネージャー
【評価するコンピテンシー】
1. リーダーシップ
2. 問題解決力
3. コミュニケーション力
4. ストレス耐性
5. 計画力

各コンピテンシーについて以下を出力:
- メイン質問（STAR手法で回答を引き出す質問）
- 深掘り質問（2つ）
- 評価のポイント（何を見ているか）
- 5段階評価の基準（1=不十分 〜 5=卓越）

※ 思想・信条・家族構成等、就職差別につながる質問を含めないこと
（厚生労働省「公正な採用選考の基本」に準拠）
```

### プロンプト4: 面接評価シートの作成

```
以下の条件で面接評価シートを作成してください。

【職種】カスタマーサクセス
【面接段階】二次面接（現場マネージャー）
【評価項目】
- 顧客折衝力
- 論理的思考力
- 自社サービスへの理解度
- カルチャーフィット

出力形式:
1. 評価シート（各項目の5段階評価基準を具体的に記述）
2. 総合評価欄（A:即内定 / B:次ステップへ / C:保留 / D:見送り）

```

```
以下の条件でオファーレター（内定通知書兼労働条件通知書）を作成してください。

【候補者名】（氏名を入れる）
【職種】マーケティングスペシャリスト
【雇用形態】正社員（試用期間3ヶ月）
【入社予定日】2026年5月1日
【給与】月給35万円（固定残業代45時間分 92,000円含む）

【休日】完全週休2日制（土日）、祝日、年末年始

出力に含めること:
1. 内定通知の文面（歓迎の気持ちを込めた文章）
2. 労働条件通知書（労働基準法施行規則第5条に基づく必須記載事項をすべて含む）

### プロンプト7: 不採用通知の文面作成

```
以下の状況に応じた不採用通知メールを作成してください。

## 注意点・法的留意事項

### 求人票に関する法的義務（職業安定法）
- **明示が義務付けられている事項**: 業務内容、契約期間、就業場所、勤務時間・休日、賃金、社会保険、募集者の氏名または名称
- 2024年4月改正で**就業場所・業務の変更の範囲**の明示が義務化
- 虚偽の求人は罰則対象（6月以下の懲役または30万円以下の罰金）

### 採用選考における差別禁止
- **性別**: 男女雇用機会均等法により、性別を理由とする差別的取扱いは禁止
- **年齢**: 労働施策総合推進法により、年齢制限は原則禁止（例外あり）
- **障害**: 障害者雇用促進法により、合理的配慮の提供義務
- **出自・思想等**: 厚生労働省「公正な採用選考の基本」に基づき、本籍・出生地、家族に関すること、思想・信条に関する質問は不適切

### 個人情報保護
- 応募者の個人情報は、採用選考の目的の範囲内でのみ利用
- 不採用者の書類は速やかに返却または適切に廃棄
- 個人情報の第三者提供には本人の同意が必要

### 固定残業代の記載
- 時間数と金額を明記しなければ無効とされるリスクがある
- 超過分は別途支給する旨を明記すること

> **重要**: 本教材の内容は一般的な実務ガイドラインです。個別の案件については、必ず社会保険労務士または弁護士に確認してください。特に求人票の法的要件や差別禁止に関する判断は、専門家の助言を得ることを推奨します。

## まとめ

- 求人票は**法令準拠**と**魅力的な訴求**の両立が重要。Claudeで雛形を生成し、法的要件の漏れを防ぐ
- 書類選考は**客観的な基準**を事前に設計することで、属人化と差別リスクを排除
- 構造化面接は**STAR手法**とコンピテンシー評価を組み合わせ、面接の信頼性を向上
- 評価シートは面接官のバイアスを軽減し、採用判断の**透明性**を確保
- オファーレターは**労働基準法の必須記載事項**を必ず網羅し、入社前トラブルを防止
- Claudeは採用業務の「たたき台作成」と「チェック」の両面で活用できるが、**最終的な採用判断は人間が行う**

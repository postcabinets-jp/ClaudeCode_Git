## 業務の概要

### 給与計算の基本フロー

```
勤怠データ集計 → 総支給額計算 → 控除額計算 → 差引支給額確定 → 振込処理 → 明細書発行
```

**総支給額** = 基本給 + 各種手当（残業手当、通勤手当、住宅手当 等）
**控除額** = 社会保険料 + 雇用保険料 + 所得税 + 住民税 + その他控除
**差引支給額** = 総支給額 - 控除額

### 社会保険の主な手続き

| 手続き | タイミング | 届出先 |
|--------|-----------|--------|
| 資格取得届 | 入社時 | 年金事務所 / 健保組合 |
| 資格喪失届 | 退職時 | 年金事務所 / 健保組合 |
| 算定基礎届 | 毎年7月 | 年金事務所 / 健保組合 |
| 月額変更届 | 随時（固定的賃金変動時） | 年金事務所 / 健保組合 |
| 賞与支払届 | 賞与支給後5日以内 | 年金事務所 / 健保組合 |
| 被扶養者（異動）届 | 異動発生時 | 年金事務所 / 健保組合 |

### 住民税特別徴収

- 会社が従業員の住民税を毎月の給与から天引きし、市区町村に納付する制度
- 毎年5月頃に市区町村から「特別徴収税額決定通知書」が届く
- 入社・退職時の「切替届」が必要

## 従来の課題

| 課題 | 詳細 |
|------|------|
| 計算ミスの発生 | 手当の計算間違い、控除額の誤りが毎月のように発生 |
| 届出期限の管理漏れ | 資格取得届の5日以内提出、算定基礎届の7月10日期限を忘れる |
| 法改正への対応遅延 | 社会保険料率の改定、最低賃金の改定への反映が遅れる |
| 属人化 | 担当者1名に業務が集中し、休暇時に対応できない |
| 残業代計算の複雑さ | 変形労働時間制、フレックス、固定残業代等の計算が難しい |

## Claudeによる自動化・効率化

### 1. 給与計算チェックリストの生成
月次・年次のチェックポイントを網羅したリストを自動生成。法改正情報も含めて更新。

### 2. 社会保険届出書類のドラフト支援
従業員情報を入力するだけで、届出に必要な情報を整理・出力。

### 3. 算定基礎届の計算補助
4月・5月・6月の報酬月額から標準報酬月額を算出するロジックの説明と検証。

### 4. 法改正の影響分析
料率変更・制度改正が自社に与える影響をシミュレーション。

### 5. 手続きスケジュールの自動生成
年間の届出スケジュールをカレンダー形式で出力。

## 注意点・法的留意事項

### 給与計算に関する法的義務
- **賃金支払いの5原則**（労働基準法第24条）: 通貨払い、直接払い、全額払い、毎月1回以上、一定期日払い
- **割増賃金の計算**（労基法第37条）: 時間外1.25倍、休日1.35倍、深夜0.25倍加算。月60時間超は1.50倍（中小企業も2023年4月〜適用）
- **最低賃金の遵守**: 地域別最低賃金と特定最低賃金の両方を確認
- **賃金台帳の保存義務**（労基法第108条）: 5年間（経過措置で当面3年間）

### 社会保険に関する注意
- **届出期限の厳守**: 資格取得届・喪失届は**事実発生から5日以内**
- **算定基礎届の注意**: 支払基礎日数が17日未満の月は除外（パートは別基準）
- **月額変更届の条件**: 固定的賃金の変動があり、3ヶ月間の平均報酬月額が2等級以上変動した場合
- **賞与の上限**: 健康保険は年度累計573万円、厚生年金は1回150万円

### 住民税に関する注意
- **特別徴収は原則義務**: 従業員がいる事業所は特別徴収が義務
- **退職時の一括徴収**: 1月1日〜4月30日の退職は残額を一括徴収が原則
- **届出の遅延**: 特別徴収への切替が遅れると、従業員に普通徴収の納付書が届き混乱する

### 電子申請
- 一定規模以上の企業は社会保険・労働保険の電子申請が義務化
- e-Gov または届書作成プログラムを使用

> **重要**: 本教材の内容は一般的な実務ガイドラインです。給与計算や社会保険手続きの最終確認は、必ず社会保険労務士に相談してください。保険料率や制度は毎年変更される可能性があるため、最新情報を確認してから実務に適用してください。

## まとめ

- 給与計算は**チェックリスト運用**で計算ミスを防止。Claudeでチェックリストを自動生成・更新できる
- 社会保険届出は**期限管理が最重要**。年間スケジュールと随時手続きの両方を管理する
- 算定基礎届は**4月〜6月の報酬を正確に集計**し、標準報酬月額を正しく算出する
- 賞与支払届は**支給後5日以内**の届出が必要。上限額の管理も重要
- 住民税特別徴収は入退社時の**切替届**を忘れずに提出する
- Claudeは計算ロジックの説明、チェックリスト作成、手続きフローの整理に強いが、**最終的な数値確認と届出は人間が責任を持つ**

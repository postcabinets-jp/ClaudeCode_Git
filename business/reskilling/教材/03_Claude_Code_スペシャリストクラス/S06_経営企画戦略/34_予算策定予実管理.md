## 業務の概要

予算策定・予実管理は、企業の経営計画を数値に落とし込み、実績との乖離をモニタリングして軌道修正するプロセスである。

### 予算策定の年間サイクル

| 時期 | 作業内容 | 担当 |
|------|----------|------|
| 9月 | 経営方針・重点施策の提示 | 経営層 |
| 10月 | 部門別予算のガイドライン配布 | 経営企画 |
| 10-11月 | 部門別予算案の作成 | 各部門長 |
| 11月 | 予算ヒアリング・調整 | 経営企画 |
| 12月 | 予算原案の取りまとめ・経営会議承認 | 経営企画・経営層 |
| 1月 | 確定予算の全社展開 | 経営企画 |
| 毎月 | 予実差異分析・報告 | 経営企画・各部門 |
| 四半期 | 着地見込み（フォーキャスト）更新 | 経営企画 |

### 予算の種類

- **売上予算**: 製品別・顧客セグメント別・チャネル別の売上計画
- **原価予算**: 原材料費、外注費、労務費などの直接コスト
- **販管費予算**: 人件費、広告費、旅費交通費、地代家賃など
- **設備投資予算（CAPEX）**: 固定資産取得、IT投資、設備更新
- **資金予算**: 月次の資金繰り、借入・返済計画

## 従来の課題

1. **Excel地獄** — 複数部門のExcelを手作業で集約し、リンク切れやバージョン管理に苦しむ
2. **差異分析が形式的** — 「計画比-5%」の数字だけで、原因と対策が不十分なまま報告される
3. **予算のロック後に環境が激変** — 年初の前提条件が崩れても、修正予算の作成が追いつかない
4. **部門間の整合性チェックに時間がかかる** — 売上部門と原価部門の前提がずれていることに気づくのが遅い
5. **フォーキャスト精度が低い** — 実績のトレンドを反映した着地見込みの算出が属人的
6. **経営層向けサマリーの作成に時間を取られる** — データは揃っているのに、要約と解釈のドキュメント化に工数がかかる

## Claudeによる自動化・効率化

### 1. 予算テンプレートの生成
部門の特性（営業/開発/管理等）に応じた予算テンプレートをClaude で作成。勘定科目の抜け漏れを防ぐ。

### 2. 予実差異の自動コメント生成
予実データ（数値）を入力し、差異の原因仮説・推奨アクションを自動生成。担当者の分析工数を大幅削減。

### 3. フォーキャスト（着地見込み）の計算支援
実績トレンドと季節性を加味した残期間予測をClaude に依頼。複数シナリオの着地見込みを短時間で算出。

### 4. 部門間整合性チェック
複数部門の予算データを同時に読み込ませ、前提条件の矛盾（例: 営業が見込む受注数と開発のリソース計画の乖離）を検出。

### 5. 経営会議用サマリーの自動作成
予実データから、エグゼクティブ向けの1-2ページサマリーを生成。重要な差異と対策にフォーカスした報告書ドラフト。

### 6. 予算編成ガイドラインの作成
経営方針を入力し、各部門に配布するガイドライン文書を生成。KPI目標値の根拠説明も含む。

## 注意点

1. **Claudeに渡す数値データの正確性を確認** — 入力データが間違っていればアウトプットも間違う。特にExcelからのコピペ時に数値がずれやすい
2. **差異コメントは「仮説」として扱う** — Claudeが推定する差異原因はデータに基づく推測であり、現場の実情と異なる場合がある。部門担当者の確認を経ること
3. **機密性の高い財務データの取り扱い** — 未公開の売上・利益データをプロンプトに含める場合は、社内のAI利用ポリシーに従う
4. **予算策定は政治的プロセスでもある** — 数値の妥当性だけでなく、部門間のバランスや経営者の意志が反映される。Claudeの提案をそのまま使うのではなく、組織の文脈を加味する
5. **フォーキャストの過信に注意** — Claudeの予測はデータのトレンド延長であり、突発的なイベント（大型受注/解約、市場変動等）は織り込めない
6. **表計算との連携** — 最終的な予算管理はExcelやスプレッドシートで行うことが多い。Claudeの出力をそのまま貼れるMarkdown表形式が便利

## まとめ

予算策定・予実管理は定型性が高い反面、差異分析の「解釈」部分に人の判断力が求められる業務である。Claudeを活用することで以下の効果が見込める。

- **予算テンプレート作成**: 部門特性に合わせたテンプレートを数分で生成。勘定科目の漏れを防止
- **差異分析コメント**: 数値データから原因仮説・対策案を自動生成。月次報告の作成時間を従来の1/3に短縮
- **フォーキャスト**: 実績トレンドとシナリオ分析による着地見込みを迅速に算出
- **部門間整合性チェック**: 複数部門の予算案を横断的に確認し、矛盾点を早期発見
- **経営層向けサマリー**: エグゼクティブが1分で全体像を把握できる報告書を自動生成

予算管理の「作業」をClaude に任せることで、経営企画担当者は「分析の深掘り」と「経営層への戦略的提言」に集中できるようになる。

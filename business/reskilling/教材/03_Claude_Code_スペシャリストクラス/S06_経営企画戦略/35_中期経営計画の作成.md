# 35. 中期経営計画の作成
**Claude Code スペシャリストクラス — S06 経営企画・戦略**
**POST CABINETS Inc.**

---

## この講義で学ぶこと

- 中期経営計画（3-5年）の全体構成と策定プロセスを理解する
- ビジョン・戦略から数値目標・アクションプランへの体系的な落とし込み方法
- PEST分析、バリューチェーン分析、5Forces等のフレームワーク活用をClaudeで加速する
- 社内外のステークホルダーに伝わる中計ドキュメントの作成技法

---

## 業務の概要

中期経営計画（中計）は、企業の3-5年先のありたい姿（ビジョン）を定義し、そこに至る戦略・数値目標・アクションプランを体系化したドキュメントである。年度予算が「今年何をするか」であるのに対し、中計は「3-5年でどこに向かうか」を示す羅針盤である。

### 中計の構成要素

```
ミッション・ビジョン（不変の価値観と目指す姿）
  ↓
外部環境分析（PEST/5Forces/業界動向）
  ↓
内部環境分析（バリューチェーン/VRIO/財務分析）
  ↓
SWOT統合 → 戦略オプションの抽出
  ↓
基本戦略の決定（成長戦略/事業ポートフォリオ/投資方針）
  ↓
数値目標（売上/利益/ROE/ROIC等）
  ↓
事業別戦略 → アクションプラン → KPI
  ↓
資本政策・株主還元方針
  ↓
リスクと対応策
```

### 中計策定のタイムライン（典型例）

| フェーズ | 期間 | 主要タスク |
|---------|------|-----------|
| 1. 現状分析 | 2ヶ月 | 外部環境分析、内部環境分析、前中計の振り返り |
| 2. 戦略立案 | 2ヶ月 | ビジョン再定義、戦略オプション検討、事業ポートフォリオ設計 |
| 3. 数値計画 | 1.5ヶ月 | 財務モデル作成、KPI設計、投資計画 |
| 4. アクションプラン | 1ヶ月 | 部門別施策の具体化、ロードマップ作成 |
| 5. ドキュメント化 | 1ヶ月 | 中計書の作成、取締役会承認、社内外発表 |

---

## 従来の課題

1. **環境分析に膨大な時間** — PEST分析だけで数週間を要し、情報収集と整理に追われる
2. **戦略と数値の乖離** — 定性的な戦略ストーリーと定量的な財務計画が噛み合わない
3. **前中計の未達分析が形式的** — 「なぜ未達だったか」の深掘りが不十分なまま次の中計を作成
4. **部門間の温度差** — 経営企画が作る全社戦略と、各事業部の現場感覚にギャップがある
5. **陳腐化** — 策定に半年以上かけた結果、発表時には前提が変わっている
6. **投資家への説明力不足** — 戦略の独自性や実現可能性を定量的に説明できない

---

## Claudeによる自動化・効率化

### 1. 外部環境分析の高速化
PEST分析、5Forces分析を構造化された形式で短時間に生成。業界の主要トレンドを網羅的に整理。

### 2. 内部環境分析のフレームワーク適用
バリューチェーン分析、VRIO分析をClaudeに依頼し、自社の強み・弱みを体系的に整理。

### 3. SWOT統合と戦略オプション抽出
外部環境と内部環境の分析結果を統合し、クロスSWOTから戦略オプションを自動導出。

### 4. 財務目標のロジック構築
「売上CAGR 15%」「営業利益率20%」といった目標に対して、セグメント別・施策別のブレイクダウンをClaudeが支援。

### 5. ベンチマーク分析
同業他社の中計目標値と比較し、自社目標の妥当性や野心度を評価。

### 6. 中計ドキュメントの構成・表現
投資家・社員・取引先など読み手に応じた表現の調整、ストーリーラインの一貫性チェック。

### VRIO分析テンプレート

```markdown
## VRIO分析

| 経営資源/ケイパビリティ | V（価値） | R（希少性） | I（模倣困難性） | O（組織体制） | 競争優位性 |
|------------------------|-----------|------------|----------------|-------------|-----------|
| AI技術（独自アルゴリズム） | ○ | ○ | ○ | △ | 一時的優位 |
| 中小企業の業務知見 | ○ | ○ | ○ | ○ | 持続的優位 |
| ブランド認知度 | ○ | × | — | — | 競争均衡 |
| エンジニア組織 | ○ | △ | △ | ○ | 一時的優位 |
| [追加] | | | | | |

### 評価基準
- **V（Value）**: その資源は顧客価値の創出や脅威への対処に役立つか？
- **R（Rarity）**: その資源を保有する競合は少ないか？
- **I（Imitability）**: その資源を模倣するコストは高いか？
- **O（Organization）**: その資源を十分に活用する組織体制があるか？

### 競争優位性の判定
- V=×: 競争劣位
- V=○, R=×: 競争均衡
- V=○, R=○, I=×: 一時的競争優位
- V=○, R=○, I=○, O=×: 未活用の競争優位
- V=○, R=○, I=○, O=○: 持続的競争優位
```

## 注意点

1. **中計は「未来予測」ではなく「意志表明」** — Claudeが出す分析は客観的データに基づく推測であり、経営者の「こうしたい」という意志を反映させるのは人間の役割
2. **フレームワークの使い過ぎに注意** — PEST、SWOT、5Forces等を全て実施すると情報過多になる。自社の意思決定に必要なものを選択する
3. **数値の一人歩きを防ぐ** — Claudeが算出した「CAGR 36%」等の数字が社内で独り歩きしないよう、前提条件とセットで提示する
4. **競合情報の鮮度** — Claudeの学習データには時間的な制約がある。競合の最新動向は別途リサーチが必要
5. **社内の合意形成プロセスは人間の仕事** — 中計は各部門のコミットメントが必要。Claudeのドラフトをたたき台に、部門長との対話を重ねること
6. **IR・公開資料としての使用** — 上場企業の場合、中計の数値目標は投資家のコミットメントと見なされる。公開前に法務・IRの確認を必ず経る

## まとめ

中期経営計画は企業経営の「設計図」であり、策定には通常6-8ヶ月を要する大型プロジェクトである。Claudeを活用することで以下の効果が期待できる。

- **環境分析の高速化**: PEST/5Forces/バリューチェーン分析を数時間で初稿作成（従来: 数週間）
- **戦略オプションの網羅性向上**: クロスSWOTから体系的に戦略候補を導出
- **数値計画の精緻化**: 複数シナリオの財務モデルを短期間で作成・比較
- **前中計の振り返り強化**: 定量・定性両面での未達分析と次期への示唆抽出
- **ドキュメントの品質向上**: 論理の一貫性チェック、読み手に応じた表現調整

中計策定の「分析・文書化」工程をClaudeに任せることで、経営企画部門は「戦略の議論」と「組織の巻き込み」という本質的な活動に時間を振り向けられる。最終的に中計の価値を決めるのは、数字の精度ではなく、組織全体が同じ方向を向けるかどうかである。

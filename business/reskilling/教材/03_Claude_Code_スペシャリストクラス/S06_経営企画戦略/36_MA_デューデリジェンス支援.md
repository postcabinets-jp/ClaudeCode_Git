## 業務の概要

M&A（Mergers and Acquisitions）は、企業の合併・買収を通じて事業規模の拡大、新市場への参入、技術獲得などを実現する経営戦略である。経営企画部門はM&Aの初期検討からPMI（Post Merger Integration）まで、一連のプロセスを推進する中核的な役割を担う。

### M&Aプロセスの全体像

| フェーズ | 主要タスク | 期間目安 |
|---------|-----------|---------|
| 1. 戦略策定 | M&A方針の決定、ターゲット基準の定義 | 1-2ヶ月 |
| 2. ターゲット選定 | ロングリスト→ショートリスト、初期アプローチ | 2-3ヶ月 |
| 3. 初期検討 | NDA締結、IM受領/作成、初期バリュエーション | 1-2ヶ月 |
| 4. 基本合意 | LOI/MOU締結、独占交渉権の獲得 | 1ヶ月 |
| 5. デューデリジェンス | 財務/法務/ビジネス/IT/人事/環境DD | 1-3ヶ月 |
| 6. 最終交渉 | バリュエーション確定、SPA（株式譲渡契約）交渉 | 1-2ヶ月 |
| 7. クロージング | 契約締結、資金決済、届出 | 1ヶ月 |
| 8. PMI | 統合計画の実行、シナジー実現 | 6-24ヶ月 |

### 主要ドキュメント

- **ティーザー**: 匿名の概要資料（A4で1-2ページ）
- **IM（Information Memorandum）**: 対象企業の詳細情報をまとめた資料（30-80ページ）
- **バリュエーションレポート**: 企業価値の算定根拠
- **DDレポート**: 各領域の調査結果と発見事項
- **LOI/MOU**: 基本合意書
- **SPA**: 株式譲渡契約書

---

## 従来の課題

1. **IMの作成に膨大な工数** — 財務データ、事業概要、市場分析等を30-80ページにまとめるのに2-4週間
2. **DDチェックリストの属人化** — 過去案件の経験に依存し、チェック項目の漏れが発生するリスク
3. **バリュエーションの前提議論** — DCFのWACC、成長率、マルチプルの選定根拠を整理するのに時間がかかる
4. **大量の資料レビュー** — DDで提出される数百〜数千のドキュメントの読み込みが追いつかない
5. **交渉ポイントの整理** — DDの発見事項を交渉上のレバレッジに変換する作業が属人的
6. **PMI計画の策定** — 統合後の組織・システム・文化の融合計画を限られた時間で作成しなければならない

## Claudeによる自動化・効率化

### 1. IM（情報メモランダム）のドラフト作成
対象企業の基本情報を入力し、IMの各セクションのドラフトを生成。投資家が重視するポイントを網羅した構成。

### 2. DDチェックリストの生成
対象企業の業種・規模に応じた包括的なDDチェックリストを自動生成。財務/法務/ビジネス/IT/人事/環境の全領域をカバー。

### 3. バリュエーションモデルの構築支援
DCF法、マルチプル法の計算ロジックをClaude が整理。前提条件の感度分析も迅速に実施。

### 4. DDレポートの要約・リスク抽出
大量のDD資料から重要なリスク事項を抽出し、優先度付きで整理。

### 5. 交渉ポイントメモの作成
DDの発見事項を、価格調整・表明保証・補償条項に分類し、交渉戦略のドラフトを作成。

### 6. PMI計画のフレームワーク作成
Day1/30/60/100のマイルストーンに基づく統合計画のテンプレートを生成。

- 売上高: 4.2億円（2024年度）
- 営業利益: 6,300万円（営業利益率15%）
- ARR: 3.8億円、MRR: 3,170万円
- 顧客数: 350社（月次チャーン0.8%）
- ARPU: 月額9万円
- 主要株主: 創業者60%、VCファンド30%、役員持株会10%

【IM構成要件】
以下のセクション構成で、各セクションに記載すべき内容のアウトラインを箇条書きで作成:

4. 事業概要（プロダクト・ビジネスモデル・競争優位性）
5. 市場環境（市場規模・成長率・トレンド）

### プロンプト2: デューデリジェンスチェックリスト（包括版）

```
以下のM&A案件について、デューデリジェンスの包括的なチェックリストを作成してください。

【案件概要】
- 買収対象: BtoB SaaS企業（人事労務管理、従業員35名、売上4.2億円）
- 買収形態: 株式100%取得
- 想定取得価格: 15-20億円（ARR 3.8億円の4-5倍）

【チェックリスト要件】
以下の6領域について、確認項目・確認方法・リスクレベル（高/中/低）を表形式で作成:

**1. 財務DD**
- 売上の実在性・計上基準の妥当性
- ARR/MRRの定義と計算方法
- 解約率の算出方法と推移
- 繰延収益の処理
- 顧客集中度（トップ10顧客の売上比率）
- 未払債務・簿外債務の有無
- 税務リスク（移転価格、欠損金）
- 運転資本の分析

**2. 法務DD**
- 株主構成・種類株式の内容
- 知的財産権（特許/商標/著作権）
- 顧客契約の主要条項（自動更新/解約条件/料金改定）
- 従業員の雇用契約・競業避止義務
- 訴訟・紛争の有無
- 許認可・届出の状況
- 個人情報保護法・GDPRへの対応

**3. ビジネスDD**
- 市場規模・成長率の検証
- 競合ポジショニングの確認
- 顧客インタビュー（NPS・満足度）
- 営業パイプラインの精査
- プライシングの競争力
- ユニットエコノミクス（CAC/LTV）の検証

**4. IT/テクノロジーDD**
- システムアーキテクチャの評価
- 技術的負債の状況
- セキュリティ体制（SOC2/ISMS等）
- インフラコスト構造（AWS/GCP）
- コードベースの品質
- データ移行・統合の難易度

**5. 人事DD**
- 組織図・レポートライン
- キーパーソンの特定とリテンションリスク
- 給与水準の市場比較
- ストックオプション/RSUの未行使分
- 退職金・年金の積立状況
- 労務コンプライアンス

**6. 環境/コンプライアンスDD**
- 業界固有の規制対応
- 反社会的勢力との関係チェック
- 海外事業のコンプライアンス

各項目に「要請資料」列も追加し、対象会社に開示を求める資料名を具体的に記載してください。
```

### プロンプト3: DCFバリュエーション

```
以下の前提条件に基づき、DCF法による企業価値評価を実施してください。

【対象企業の財務データ】
- 直近年度売上高: 4.2億円
- 直近年度EBITDA: 8,400万円（EBITDAマージン20%）
- 直近年度設備投資: 1,200万円
- 直近年度運転資本増減: +500万円
- 実効税率: 30%
- 有利子負債: 5,000万円
- 現預金: 1.2億円

【予測の前提条件】
| 項目 | Year1 | Year2 | Year3 | Year4 | Year5 |
|------|-------|-------|-------|-------|-------|
| 売上成長率 | 25% | 22% | 18% | 15% | 12% |
| EBITDAマージン | 22% | 24% | 26% | 27% | 28% |
| 設備投資/売上比率 | 3% | 3% | 2.5% | 2.5% | 2% |
| 運転資本/売上比率 | 5% | 5% | 4% | 4% | 4% |

【WACC算出の前提】
- リスクフリーレート: 1.0%（日本10年国債）
- エクイティリスクプレミアム: 6.0%
- ベータ: 1.2（類似上場企業の平均）
- サイズプレミアム: 3.0%（小型株）
- 負債コスト: 2.0%
- D/E比率: 10%（時価ベース）

【出力要件】
1. WACC算出の計算過程
2. 5年間のFCF予測（詳細な計算過程付き）
3. ターミナルバリュー（永久成長率法: 成長率2%）
4. 事業価値（Enterprise Value）
5. 株主価値（Equity Value）= EV - 有利子負債 + 現預金
6. 感度分析: WACCとターミナル成長率の2軸マトリクス
   - WACC: 8%, 9%, 10%, 11%, 12%
   - 永久成長率: 1%, 1.5%, 2%, 2.5%, 3%
7. 評価額のレンジ（下限-中央値-上限）
```

## 注意点

1. **M&Aは守秘義務が極めて重要** — 案件名、対象企業名、取得価格等をClaudeに入力する場合は、社内のAI利用ポリシーとNDAの範囲を必ず確認する。コード名を使用するのも有効
2. **バリュエーションは「目安」** — Claudeが算出する企業価値は、前提条件に依存する理論値。実際の取得価格は交渉で決まる
3. **法的アドバイスの代替にはならない** — DDの法務パートや契約条項のドラフトは、必ず弁護士のレビューを経ること
4. **会計・税務の専門判断が必要** — 繰延税金資産、のれんの減損テスト、連結処理等は公認会計士に確認すること
5. **類似企業の選定が評価を左右する** — マルチプル法では、類似企業の選定基準（業種/規模/成長率/地域）を明確にし、恣意的な選定を避ける
6. **PMI計画は「統合前」に作る** — 買収の成否はPMIで決まるが、PMI計画はDD段階から並行して検討を始めるべきである

## まとめ

M&A・デューデリジェンスは、高度な専門知識と膨大な作業量が求められる業務である。Claudeを活用することで以下の効果が期待できる。

- **IM作成の効率化**: 各セクションのドラフトを数時間で生成（従来: 2-4週間）
- **DDチェックリストの網羅性向上**: 業種・規模に応じた包括的なチェックリストを自動生成
- **バリュエーションの迅速化**: DCF/マルチプルの計算と感度分析を短時間で実施
- **DD発見事項の構造化**: 膨大な発見事項を交渉ポイントとして体系的に整理
- **PMI計画のテンプレート化**: 100日計画のフレームワークを即座に生成

M&Aは1件あたり数億〜数百億円の意思決定であり、最終判断は必ず経営者と専門家（FA/弁護士/会計士）が行う。Claudeの役割は「分析の下準備」と「ドキュメント作成の加速」であり、人間の判断力・交渉力を補完するツールとして活用することが成功の鍵である。

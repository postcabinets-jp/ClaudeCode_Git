## 業務の概要

IR（Investor Relations）は、企業が株主・投資家に対して経営状況や今後の見通しを適時・適切に開示し、信頼関係を構築する活動である。経営企画部門はIR部門と連携し、各種資料の作成を担うことが多い。

### 主要なIR資料とスケジュール

| 資料 | 頻度 | 対象読者 | 主な内容 |
|------|------|---------|---------|
| 決算短信 | 四半期 | 投資家・アナリスト | 財務数値の速報 |
| 決算説明資料 | 四半期 | 機関投資家・アナリスト | 業績解説、戦略進捗、見通し |
| 有価証券報告書 | 年次 | 規制当局・投資家 | 法定開示（詳細な経営情報） |
| 株主通信 | 半期/年次 | 個人株主 | 業績ハイライト、事業トピック |
| 統合報告書 | 年次 | 機関投資家・ESG評価機関 | 財務+非財務情報の統合 |
| ESG/サステナビリティ報告書 | 年次 | ESG投資家・格付機関 | 環境・社会・ガバナンスの取り組み |
| IR説明会想定問答集 | 四半期 | 社内（経営陣向け） | 投資家からの想定Q&A |
| ファクトブック | 年次 | アナリスト | 長期の財務・非財務データ集 |

### IR開示のタイムライン（3月決算の場合）

| 時期 | イベント | 主要資料 |
|------|---------|---------|
| 4月末-5月中旬 | 本決算発表 | 決算短信、決算説明資料、通期業績予想 |
| 6月 | 株主総会 | 株主総会招集通知、事業報告 |
| 7月末-8月中旬 | Q1決算発表 | 四半期決算短信、説明資料 |
| 9月-10月 | 統合報告書発行 | 統合報告書、ESG報告書 |
| 10月末-11月中旬 | Q2決算発表 | 中間決算短信、説明資料 |
| 12月 | 株主通信（中間） | 株主通信 |
| 1月末-2月中旬 | Q3決算発表 | 四半期決算短信、説明資料 |

## 従来の課題

1. **決算説明資料の作成に追われる** — 四半期ごとに30-50ページの資料を数日で仕上げる必要がある
2. **定型的な記述の繰り返し** — セグメント解説や事業環境の説明が毎期似た内容になるが、微修正が必要
3. **想定問答の網羅性** — アナリストの質問は多岐にわたり、準備が追いつかないことがある
4. **非財務情報の開示要求の増大** — TCFD、ISSB、人的資本等の開示要件が年々増加し、対応コストが急増
5. **株主通信のトーン調整** — 個人株主向けは専門用語を避け、わかりやすく伝える必要があるが、正確性との両立が難しい
6. **グローバル投資家への英語対応** — 日本語で作成した資料の英訳に時間がかかる

## Claudeによる自動化・効率化

### 1. 決算説明資料のドラフト作成
財務データと主要トピックを入力し、各スライドの文言ドラフトを生成。前期からの変化点を自動抽出。

### 2. 株主通信の執筆
個人株主向けのわかりやすい文章で、業績ハイライトと今後の見通しを記述。

### 3. ESG/サステナビリティ報告書の構成
TCFD提言、ISSB基準に沿った開示フレームワークの構成を作成。各項目の記載例を提示。

### 4. 想定問答集の作成
決算数値と事業トピックから、投資家・アナリストが質問しそうなポイントを予測し、回答案を生成。

### 5. ファクトブックのデータ整理
過去5-10年分の財務・非財務KPIを体系的に整理するテンプレートを作成。

### 6. 英語IR資料のドラフト
日本語の決算説明資料を、投資家向けの自然な英語に翻訳。

## 注意点

1. **適時開示規制の遵守** — 決算数値や業績予想は金融商品取引法の適時開示ルールに従って開示する。Claudeのドラフトは法務・IR担当のレビューを必ず経ること
2. **インサイダー情報の取り扱い** — 未公表の業績データをClaudeに入力する場合、社内のAI利用ポリシーに従う。決算発表前の数値は特に慎重に扱う
3. **フォワードルッキングステートメント** — 将来の見通しに関する記述には「将来予測に関する注意事項」の免責文言を付ける
4. **ESG開示基準の変化** — ISSB、CSRD等の基準は急速に進化中。Claudeの知識が最新でない場合があるため、最新基準は別途確認する
5. **機関投資家と個人株主のトーンの違い** — 機関投資家向けはデータ重視、個人株主向けはわかりやすさ重視。読み手を明確にしてから作成する
6. **英語IRの品質** — Claudeの英語翻訳は自然だが、財務用語の慣行（IFRS vs US GAAP等）は専門家のチェックが必要

## まとめ

IR資料・株主向け報告書の作成は、経営企画・IR部門にとって四半期ごとに繰り返される高負荷業務である。Claudeを活用することで以下の効果が期待できる。

- **決算説明資料**: 財務データと主要トピックから各スライドの文言を自動生成。作成時間を従来の1/3に短縮
- **株主通信**: 個人株主向けのわかりやすい文章を自動生成。専門用語の平易化も自動対応
- **想定問答集**: 決算数値と事業トピックから25-50問の想定Q&Aを網羅的に作成
- **ESG報告書**: TCFD/ISSB準拠の構成テンプレートと記載ガイドを即座に生成
- **英語IR**: 投資家向けの自然な英語に翻訳。日本特有の会計用語の変換もカバー

IR活動の本質は「企業の価値を正しく市場に伝え、適正な評価を得ること」である。Claudeで定型的な資料作成を効率化し、経営陣のメッセージの磨き込みや投資家との対話の質向上に時間を振り向けることが、IR活動全体の価値を高める。

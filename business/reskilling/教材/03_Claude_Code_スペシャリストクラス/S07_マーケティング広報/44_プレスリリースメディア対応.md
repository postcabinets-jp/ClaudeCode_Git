## 業務の概要

広報（PR: Public Relations）は、メディアや社会に対して企業・サービスの情報を発信し、認知・信頼を獲得する業務である。広告と異なり、メディアの第三者的な報道を通じて情報が伝わるため、信頼性が高い一方、コントロールが難しい。

主な業務フロー:

1. **プレスリリース作成** — ニュースバリューのある情報を定型フォーマットで配信
2. **メディアリスト管理** — ターゲットとなる記者・編集者のリストを構築・更新
3. **ピッチ活動** — 個別の記者に対して取材を提案するメール・電話
4. **記者会見・発表会** — 大型発表時のイベント運営とQ&A準備
5. **メディアモニタリング** — 掲載記事の収集・分析
6. **危機管理広報** — ネガティブな報道やSNS炎上への対応

## 従来の課題

| 課題 | 詳細 |
|------|------|
| プレスリリースの作成に時間がかかる | 1本あたり2〜4時間、レビュー含めると半日〜1日 |
| ニュースバリューの判断が属人的 | 何がメディアに取り上げられるかの判断基準がない |
| ピッチメールの開封率が低い | テンプレート的なメールで記者の興味を引けない |
| Q&A準備が不十分 | 想定外の質問に対応できず、メッセージがブレる |
| 危機対応が後手に回る | 初動対応文の作成に時間がかかり、状況が悪化する |
| メディアリストが古い | 記者の異動・退職を追えず、無効なコンタクトが増える |

## Claudeによる自動化・効率化

### 1. プレスリリースの高速ドラフト作成
- 基本情報を入力するだけで、業界標準フォーマットのリリース文を生成
- 複数の切り口（技術寄り、ビジネス寄り、消費者寄り）でバリエーション作成

### 2. ニュースバリューの評価
- リリース候補のネタを入力し、メディアに取り上げられやすさを評価
- 「ニュース性」を高めるための切り口・角度の提案

### 3. ピッチメールのパーソナライズ
- 記者の担当領域・過去記事に応じたカスタマイズメール生成
- 件名のバリエーション作成（開封率向上）

### 4. Q&A想定問答の自動生成
- 発表内容をもとに、記者から想定される質問を50問以上生成
- 好意的な質問、批判的な質問、予想外の質問をカテゴリ別に

### 5. 危機管理広報文書の迅速作成
- インシデント情報を入力し、初動対応文・プレスステートメントのドラフトを生成
- Q&A、SNS対応文、社内通知文を一括作成

## プロンプト例（5つ以上、コピペで使えるレベル）

### プロンプト1: プレスリリース作成

```
あなたは企業広報の専門家です。以下の情報をもとに、プレスリリースを作成してください。

【発表内容】
- 企業名: 株式会社Green Table
- サービス名: オーガニック野菜の法人向け定期便「Green Office」
- 内容: 従業員の健康経営支援を目的とした法人向けオーガニック野菜定期便サービスを2024年7月1日より開始
- 価格: 1オフィスあたり月額15,000円〜（従業員20名分の週1回配送）
- 特徴: 契約農家から直送、農薬不使用、カット済みで社内キッチンですぐ使える
- 背景: 健康経営優良法人の認定取得を目指す企業が増加、食の福利厚生ニーズが拡大
- 実績: テスト導入10社で従業員満足度92%、野菜摂取量が平均1.5倍に
- 目標: 初年度100社導入

プレスリリースの構成:
1. タイトル（30字以内 + サブタイトル50字以内）
2. リード文（5W1Hを含む3〜4行）
3. 背景・市場環境（なぜ今このサービスが必要か）
4. サービス概要（特徴を3つ、箇条書き）
5. テスト導入の実績・データ
6. 代表者コメント（150字程度）
7. 今後の展望
8. サービス詳細（料金・申込方法・URL）
9. 会社概要

文体: 「である」調、客観的・簡潔。数字を効果的に使用。
文字数: 本文1,200〜1,500字
```

### プロンプト2: ニュースバリュー評価・角度提案

```
以下の5つのプレスリリース候補について、ニュースバリューを評価し、メディアに取り上げられやすい切り口を提案してください。

【企業】株式会社Green Table（オーガニック食品D2C、創業3年）

候補1: 法人向け定期便サービスの開始
候補2: 累計会員数10,000人突破
候補3: 新しい契約農家5軒と提携開始
候補4: 食品ロス削減の取り組みで廃棄率を前年比50%削減
候補5: 代表が「Forbes 30 Under 30 Japan」にノミネート

各候補について:
1. ニュースバリュー評価（5段階）
2. 取り上げそうなメディアカテゴリ（全国紙 / 業界紙 / Webメディア / テレビ / 地方紙）
3. メディアが興味を持つ角度（切り口）を3つ提案
4. リリースのタイミング提案（いつ出すと効果的か）
5. 組み合わせ提案（複数のネタを1つのリリースにまとめるべきか）

評価基準:
- 新規性（初めて/業界初）
- 社会性（社会課題への貢献）
- 数字のインパクト
- トレンドとの関連性
- 読者/視聴者の共感可能性
```

### プロンプト3: ピッチメール作成

```
以下の情報をもとに、記者向けのピッチメールを5パターン作成してください。

【送付先の記者像】
A: 全国紙の経済部記者（健康経営・福利厚生の取材歴あり）
B: 業界専門メディアの編集者（食品・農業・サステナビリティ）
C: Webメディアのライター（スタートアップ・D2C・EC）
D: テレビ局の情報番組ディレクター（生活・健康系企画）
E: 地方紙の記者（契約農家の地元紙）

【発表内容】法人向けオーガニック野菜定期便「Green Office」の開始

各ピッチメールに含めるもの:
1. 件名（開封したくなる、20字以内×3パターン）
2. 本文（300〜500字）
   - 冒頭: なぜこの記者に送っているか（パーソナライズ要素）
   - ニュースのポイント（3行以内）
   - 記者にとってのメリット（取材すると何が得られるか）
   - 具体的な提案（取材、試食会招待、代表インタビュー等）
3. 添付資料の案内

トーン: 丁寧だが簡潔。記者の時間を奪わない配慮。売り込み感を出しすぎない。
```

```

- 発表: 法人向けオーガニック野菜定期便「Green Office」の開始

- 会見形式: 記者会見（メディア20社招待）

■ ビジネスに関する質問（10問）
- 収益モデル、成長戦略、資金調達、目標、KPI 等

■ 社会的意義に関する質問（5問）
- 食品ロス、地域農業支援、健康経営推進 等

■ 批判的・挑戦的な質問（10問）
- 価格の妥当性、オーガニックの科学的根拠、スケーラビリティの課題、食中毒リスク 等

■ 想定外のカーブボール質問（5問）
- 代表の個人的な話、業界の問題、政策への意見 等

各Q&Aには:
- 質問文
- 模範回答（200字以内、キーメッセージを盛り込む）
- 避けるべき表現・NGワード
- 回答のポイント（なぜこう答えるか）
```

### プロンプト5: 危機管理広報文書

```
以下のインシデントに対する危機管理広報の文書一式を作成してください。

【インシデント概要】
- 企業: 株式会社Green Table
- 事象: 定期便で配送された有機トマトの一部ロットに、基準値を超える残留農薬が検出された（外部検査機関の定期検査で判明）
- 影響範囲: 6月10日〜17日に配送された約200世帯分
- 健康被害: 現時点で報告なし
- 原因: 調査中（契約農家の一部で使用された隣接農地からのドリフト（飛散）の可能性）
- 現在の対応: 該当ロットの配送停止済み、全ロットの緊急検査を実施中

以下の文書を作成してください:

1. 第一報プレスリリース（事実の公表、お詫び、対応状況）
   - 構成: 事象 → 影響範囲 → 現在の対応 → 今後の方針 → 問い合わせ先
   - 文体: 「です・ます」調、誠実・簡潔

2. 顧客向けお知らせメール
   - 該当する200世帯への個別連絡文
   - 返金・代替品の案内を含む

3. SNS（X）での公式アナウンス（140文字以内 × 3投稿）
   - 第1報: 事実の公表
   - 第2報: 対応状況の報告
   - 第3報: 再発防止策の概要

4. 記者対応Q&A（15問）
   - 特に厳しい質問（「オーガニックと言いながら農薬検出とはどういうことか」等）への回答

5. 社内向け通知文
   - 全従業員への周知、外部からの問い合わせ対応マニュアル

各文書の冒頭に【目的】【トーン】【配信タイミング】を記載してください。
```

## 注意点

1. **プレスリリースはファクトベース** — Claudeに「盛った」表現を書かせない。数字、日付、固有名詞は必ず人が事実確認する
2. **記者への連絡は人が行う** — Claudeで文面を作成しても、送信の判断と実行は必ず広報担当者が行う
3. **危機管理は速度と正確性の両立** — Claudeで素早くドラフトを作成しつつ、事実確認は省略しない
4. **法務チェック** — 特に危機対応文書は法務部門の確認を経てから公開する
5. **インサイダー情報の取り扱い** — 上場企業の場合、未公表の経営情報をClaudeに入力しない（情報漏洩リスク）
6. **記者の個人情報** — メディアリストの記者情報（個人メールアドレス等）はClaudeに入力する際に注意
7. **トーンの一貫性** — 危機時に「普段と違う」トーンを使うと不信感を招く。平時の広報トーンとの整合性を保つ

---

## まとめ

- プレスリリースの作成はClaudeで1〜2時間の作業を15分に短縮できる。ただし事実確認は省略しない
- ピッチメールは記者ごとにパーソナライズすることで開封率・返信率が大幅に向上する
- 記者会見のQ&A準備は、批判的な質問・想定外の質問まで含めた網羅的な想定問答をClaudeで生成し、登壇者のトレーニングに活用
- 危機管理広報では、Claudeの高速ドラフト生成能力が初動対応のスピードを大きく改善する
- すべての広報文書は「事実の正確性」「法的リスク」「ブランドトーンの一貫性」を人がチェックした上で公開する

## 業務の概要

### 株主総会

株主総会は、株式会社の最高意思決定機関です（会社法第295条）。

| 区分 | 定時株主総会 | 臨時株主総会 |
|------|-------------|-------------|
| **開催時期** | 事業年度終了後一定の時期（通常3か月以内） | 必要がある場合に随時 |
| **法的根拠** | 会社法第296条第1項 | 会社法第296条第2項 |
| **主な議題** | 計算書類の承認、剰余金の配当、役員選任 | 定款変更、合併、解散等 |

### 取締役会

取締役会設置会社では、取締役会が業務執行の決定を行います（会社法第362条）。

**取締役会の専決事項（会社法第362条第4項）：**
1. 重要な財産の処分及び譲受け
2. 多額の借財
3. 支配人その他の重要な使用人の選任及び解任
4. 支店その他の重要な組織の設置・変更・廃止
5. 社債の発行に関する事項
6. 内部統制システムの整備
7. 定款の定めに基づく役員等の責任の一部免除

### 書面決議（みなし決議）

- **取締役会の書面決議** — 定款の定めがある場合、取締役全員が書面等により同意し、監査役が異議を述べないときは、取締役会の決議があったものとみなされる（会社法第370条）
- **株主総会の書面決議** — 株主全員が書面等により同意したときは、株主総会の決議があったものとみなされる（会社法第319条第1項）

## 従来の課題

| 課題 | 詳細 |
|------|------|
| スケジュール管理の複雑さ | 招集通知の発送期限、基準日、届出期限など多数の期日管理 |
| 議事録作成の負担 | 法定記載事項の漏れ、決議要件の確認に時間がかかる |
| 法定要件の見落とし | 特別決議と普通決議の混同、特殊利害関係人の議決権排除漏れ |
| 決議事項と報告事項の整理 | 何を総会で決議し、何を取締役会で決議するかの判断 |
| 書面決議の手続き不備 | 全員の同意書の取得漏れ、監査役への通知忘れ |

## Claudeによる自動化・効率化

### 1. スケジュール逆算表の自動生成
事業年度末と総会希望日を入力するだけで、法定期限を逆算したスケジュール表を生成。

### 2. 招集通知のドラフト作成
会社情報と議案を入力すれば、会社法の要件を満たした招集通知のドラフトを生成。

### 3. 議事録の自動ドラフト
議事メモ・決議事項を入力すれば、法定記載事項を網羅した議事録のドラフトを生成。

### 4. 決議要件の自動判定
議案内容を入力すると、普通決議・特別決議・特殊決議のいずれに該当するかを判定。

### 5. 書面決議の文書セット生成
提案書・同意書・みなし決議の議事録を一括で生成。

## 注意点

1. **招集通知の期限** — 非公開会社の株主総会は1週間前まで（会社法第299条第1項）。ただし、取締役会設置会社では書面投票・電子投票を定めた場合は2週間前（同条第1項）。公開会社は2週間前が原則。

2. **議事録の作成義務** — 株主総会議事録は総会の日から10年間本店に備え置く義務がある（会社法第318条第2項）。取締役会議事録も同様に10年間の備置義務（会社法第371条第1項）。

3. **登記申請との関連** — 取締役の選任・代表取締役の変更等は、変更があった日から2週間以内に本店所在地で変更登記が必要（会社法第915条第1項）。議事録は登記申請の添付書類となる。

4. **特別利害関係人の議決権** — 取締役会では特別の利害関係を有する取締役は議決に加わることができない（会社法第369条第2項）。株主総会では原則として制限はないが、特別決議の一部に例外あり。

5. **Claudeの出力は下書き** — 議事録は法的文書であり、登記申請の添付書類にもなります。Claudeの出力は必ず弁護士または司法書士に確認を依頼してください。

6. **電子署名の活用** — 取締役会議事録への電子署名は、法務省令で定める署名または記名押印に代わる措置として認められています（会社法第369条第4項、会社法施行規則第225条）。

## まとめ

- 株主総会・取締役会は会社法で厳格に手続きが定められた **法定機関** — 手続き不備は決議取消しのリスクがある
- **非公開会社・少人数株主** の場合、書面決議（みなし決議）を活用して運営を効率化できる
- Claudeを活用すれば、**招集通知・議事録・書面決議文書のドラフト作成** を大幅に短縮できる
- スケジュール管理は **法定期限からの逆算** が基本 — Claudeに逆算表を生成させると漏れを防げる
- 最終的な文書は **弁護士・司法書士のレビュー** を経てから使用すること

- Claudeを活用した届出一覧の整理・必要書類チェック・期限管理

## 業務の概要

### 1. 法人登記

法人登記は、会社の基本情報を法務局に届け出て公示する制度です。登記すべき事項に変更が生じた場合、原則として **変更の日から2週間以内** に変更登記を申請する義務があります（会社法第915条第1項）。

| 登記事項 | 主な発生場面 | 法的根拠 |
|---------|-------------|---------|
| 役員変更 | 取締役の選任・辞任・重任 | 会社法第911条第3項第13号 |
| 本店移転 | オフィス移転 | 会社法第911条第3項第3号 |
| 目的変更 | 事業内容の追加・変更 | 会社法第911条第3項第1号 |
| 増資 | 資本金の増加 | 会社法第911条第3項第5号 |
| 商号変更 | 会社名の変更 | 会社法第911条第3項第2号 |
| 支店設置・廃止 | 支店の開設・閉鎖 | 会社法第930条 |

### 2. 各種届出

会社運営に伴い、行政機関への届出が随時発生します。

**税務関連（税務署・都道府県税事務所・市区町村）：**
- 法人設立届出書
- 給与支払事務所等の開設届出書
- 青色申告の承認申請書
- 消費税の届出書（課税事業者選択、簡易課税等）
- 異動届出書（本店移転、代表者変更等）

**労務関連（労働基準監督署・ハローワーク）：**
- 就業規則の届出（常時10人以上の労働者を使用する場合）
- 36協定届（時間外・休日労働に関する協定）
- 雇用保険の適用事業所設置届
- 労働保険の成立届

**社会保険関連（年金事務所）：**
- 新規適用届
- 被保険者資格取得届・喪失届
- 算定基礎届・月額変更届
- 賞与支払届

### 3. 許認可管理

業種によって必要な許認可は異なります。許認可には有効期限があるものが多く、更新を怠ると事業が継続できなくなるリスクがあります。

| 許認可の例 | 根拠法 | 有効期間 |
|-----------|--------|---------|
| 建設業許可 | 建設業法第3条 | 5年 |
| 宅地建物取引業免許 | 宅建業法第3条 | 5年 |
| 飲食店営業許可 | 食品衛生法第55条 | 都道府県による（5〜8年） |
| 労働者派遣事業許可 | 労働者派遣法第5条 | 初回3年、更新後5年 |
| 有料職業紹介事業許可 | 職業安定法第30条 | 初回3年、更新後5年 |
| 古物商許可 | 古物営業法第3条 | 期限なし（届出変更あり） |

## 従来の課題

| 課題 | 詳細 |
|------|------|
| 届出漏れ・期限超過 | 登記懈怠は過料（会社法第976条第1号、100万円以下）の対象 |
| 必要書類の確認に時間がかかる | 登記事項ごとに添付書類が異なり、法務局のサイトが分かりにくい |
| 届出先が多岐にわたる | 1つの変更（例：本店移転）で税務署・都道府県・市町村・年金事務所等に複数届出 |
| 許認可の更新漏れ | 更新期限の管理が属人的で、担当者の異動で引き継がれない |
| 手続きが複雑で外注コストが高い | 司法書士・税理士への依頼費用が積み重なる |

## Claudeによる自動化・効率化

### 1. 届出チェックリストの自動生成
変更内容（本店移転、役員変更等）を入力するだけで、必要な届出先・届出書類・期限を一覧化。

### 2. 登記申請書のドラフト作成
法務局の書式に準拠した申請書のドラフトを生成。

### 3. 届出期限管理台帳の設計・更新
全届出・許認可の期限を一元管理する台帳を設計し、アラート基準を設定。

### 4. 届出書の記載例作成
各届出書の記入方法を具体的な記載例付きで解説。

### 5. 許認可更新の必要書類チェック
更新に必要な書類・手数料・スケジュールを一覧化。

## 注意点

1. **登記懈怠の過料** — 登記すべき事項の変更後2週間以内に登記しない場合、代表者個人に100万円以下の過料が科される可能性がある（会社法第976条第1号）。実務上は数か月〜1年以上の遅延で裁判所から過料通知が届くケースが多い。

2. **みなし解散** — 最後の登記から12年経過した株式会社は、法務大臣による官報公告を経て、みなし解散となる（会社法第472条）。役員の任期管理を怠るとこの事態に陥る。

3. **届出先の確認** — 同じ変更でも届出先が複数にわたることが多い（例：本店移転なら法務局・税務署・都道府県・市町村・年金事務所等）。Claudeに漏れチェックさせると有効。

4. **電子申請の活用** — 法人登記はオンライン申請が可能（登記・供託オンライン申請システム）。社会保険・税務もe-Gov、eLTAX等で電子申請可能。

5. **Claudeの限界** — 登記申請書の最終確認は法務局の相談窓口または司法書士に依頼すること。手数料や書式は改定されることがあるため、最新情報は各官庁のWebサイトで確認してください。

6. **許認可の業種確認** — Webマーケティング事業でも、広告代理業務の内容次第では特定の届出が必要になる場合がある。事業内容と許認可の要否は、行政書士や所管官庁に確認すること。

---

## まとめ

- 法人登記は **変更後2週間以内** が原則 — 懈怠すると過料のリスクがある
- 1つの変更で **複数の届出先** に手続きが必要になるため、チェックリストでの漏れ防止が重要
- 許認可は **有効期限の管理** が生命線 — 管理台帳を作成し、6か月前からアラートを設定する
- Claudeを活用すれば、**必要手続きの洗い出し・申請書ドラフト・期限管理台帳の設計** を効率化できる
- 最終的な申請は **司法書士・行政書士・税理士** 等の専門家と連携して進めること

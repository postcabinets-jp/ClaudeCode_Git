## 業務の概要

競合分析と市場調査は、営業戦略の根幹を支える活動である。自社の立ち位置を客観的に理解し、顧客に対して「なぜ当社を選ぶべきか」を論理的に説明するための土台となる。

**競合分析・市場調査の主な業務:**
1. 競合企業のリストアップと基本情報収集
2. 競合のサービス・価格・強み弱みの整理
3. 市場規模・成長率の推計
4. 業界トレンドの把握
5. 自社のポジショニング分析
6. SWOT分析・3C分析等のフレームワーク適用
7. 分析結果の営業資料・戦略への反映

---

## 従来の課題

| 課題 | 詳細 |
|------|------|
| **情報が散在** | 競合情報がSlack、メール、営業の頭の中に散らばっている。一元管理されていない |
| **分析に時間がかかる** | 情報収集から分析まで1つの競合で半日〜1日。複数競合だと数日かかる |
| **属人的な市場観** | 「あの業界は伸びている」という感覚はあるが、数字で裏付けられない |
| **フレームワークの形骸化** | SWOTを作ってはみるが、「で、どうするの？」が出てこない |
| **鮮度の問題** | 半年前に作った競合分析が更新されず、古い情報で商談に臨んでいる |
| **営業トークへの転換が弱い** | 分析結果が報告書止まりで、実際の商談での切り返しに使えていない |

## Claudeによる自動化・効率化

### 1. 競合情報の構造化
断片的な競合情報をClaudeに入力し、一定のフォーマットで構造化・整理する。

### 2. SWOT分析の自動生成
自社と競合の情報をインプットすれば、SWOT分析のドラフトを即座に生成する。

### 3. 市場規模推定（フェルミ推定）
限られたデータから合理的に市場規模を推計するロジックを構築する。

### 4. ポジショニングマップの設計
2軸のポジショニングマップを設計し、自社と競合の位置関係を可視化する。

### 5. 競合対策トークの生成
「顧客が競合と比較している」場面で使える切り返しトークを作成する。

### プロンプト2: SWOT分析の自動生成

```
あなたは戦略コンサルタントです。
以下の情報をもとに、当社のSWOT分析を作成してください。
「分析して終わり」ではなく、具体的な戦略アクションまで落とし込んでください。

【当社情報】
・会社名: POST CABINETS Inc.
・事業: Webマーケティング支援（サイト制作・SEO・SNS運用・広告運用）
・従業員数: [例: 5名]
・強み: SNSマーケとの統合力、データ分析、意思決定スピード
・弱み: 人員が少ない、ブランド認知度が低い、大規模案件の実績が少ない
・主要顧客: 中小企業（年商1〜10億円）

【市場環境】
・Webマーケティング市場は年率10%成長
・AI活用のマーケティングツールが急増
・中小企業のDX意識が高まっている
・大手代理店の下請け単価は下落傾向
・SNS（特にショート動画）のマーケ活用需要が急増

【主要競合の動向】
・大手A社: AI搭載の自動化ツールをリリース。価格攻勢
・フリーランス集団B: 低価格でスピード対応。品質はばらつきあり
・地域密着型C社: 地元企業の口コミ紹介が強い

以下を作成してください:
1. SWOT分析マトリクス
   - Strengths（強み）: 5つ以上
   - Weaknesses（弱み）: 5つ以上
   - Opportunities（機会）: 5つ以上
   - Threats（脅威）: 5つ以上

2. クロスSWOT戦略
   - SO戦略（強み×機会）: 攻めの戦略
   - WO戦略（弱み×機会）: 改善の戦略
   - ST戦略（強み×脅威）: 差別化の戦略
   - WT戦略（弱み×脅威）: 防衛の戦略

3. 各戦略から導かれる具体的なアクション（合計10個以上）
   - 優先度（高/中/低）
   - 実行難易度（高/中/低）
   - 期待効果
```

### プロンプト3: 市場規模推定（フェルミ推定）

```
あなたは市場分析の専門家です。
以下の条件で、ターゲット市場の市場規模を推定してください。

【推定対象】
・市場: [例: 大阪府内の中小企業（年商1〜10億円）向けSNSマーケティング運用代行市場]

【前提条件（わかっている情報）】
・大阪府の企業数: 約42万社（経済センサス）
・うち中小企業比率: 約99.7%
・年商1〜10億円の中小企業の割合: 推定[例: 5%程度]
・SNSマーケティングを外部委託している/検討中の割合: 推定[例: 15%]
・月額の平均委託単価: 推定[例: 25万円/月]

以下を作成してください:
1. ボトムアップ推定
   - TAM（Total Addressable Market）: 理論上の最大市場
   - SAM（Serviceable Addressable Market）: 当社がアプローチ可能な市場
   - SOM（Serviceable Obtainable Market）: 現実的に獲得可能な市場

2. 推定のロジック（計算過程を明示）

3. 感度分析（前提条件が±20%変動した場合の影響）

4. 市場の成長予測（3年間）

5. 推定の信頼度と、精度を上げるために追加で調べるべき情報

6. この市場規模から導かれる事業計画への示唆
   - 現実的なシェア目標
   - 必要な営業リソース
```

## 注意点

1. **Claudeの知識には限界がある**: Claudeの情報はトレーニングデータに基づくため、最新の競合動向や市場データは一次ソースで確認が必要
2. **公開情報ベースが原則**: 競合の内部情報を不正に入手して分析に使うのは倫理的・法的にNG
3. **分析バイアスに注意**: 自社に都合の良い分析結果になりがち。意識的に「自社に不利な視点」も含める
4. **鮮度管理**: 競合カードは最低でも四半期に1回更新。古い情報で判断すると逆効果
5. **分析→アクションを必ずセットに**: 「分析しました。以上。」では営業の役に立たない。必ず「だから何をするか」まで落とし込む

---

## まとめ

- 競合情報を構造化し、営業チーム全員がアクセスできる「競合カード」として管理する
- SWOT分析はクロスSWOT（SO/WO/ST/WT戦略）まで落とし込んで初めて実用的になる
- 市場規模推定（フェルミ推定）は、事業計画の根拠として経営層への説得力を高める
- ポジショニングマップで「自社の空白領域」を見つけ、差別化メッセージに転換する
- 競合分析の最終目的は「営業の現場で使える切り返しトーク」と「提案の説得力向上」である

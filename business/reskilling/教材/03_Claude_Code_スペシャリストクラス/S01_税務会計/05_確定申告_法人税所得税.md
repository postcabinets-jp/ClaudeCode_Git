## この講義で学ぶこと

1. 法人税申告書の別表作成をClaudeで支援し、申告実務の効率化を図る方法
2. 所得税確定申告の計算確認・控除適用判定をAIで行うテクニック
3. 税額控除・特別償却の適用判定と申告書チェックリストの活用

---

## 業務の概要

**確定申告**は、法人税（法人が対象）と所得税（個人が対象）の年間の税額を確定し、税務署に申告・納付する手続きです。法人税は事業年度終了後2ヶ月以内（申告期限の延長特例あり）、所得税は翌年2月16日〜3月15日が申告期間です。

- **誰がやるか**: 税理士（法人税は税理士関与率90%以上）、経理担当者、個人事業主本人
- **なぜ重要か**: 申告書の誤りは過少申告加算税（10〜15%）、延滞税、場合によっては重加算税（35〜40%）のペナルティに直結します。一方で、適用可能な税額控除・特別償却を見逃すと「払わなくてもよい税金を払う」ことになります。

---

## 従来の課題

| 課題 | 具体的な影響 |
|------|------------|
| **別表の作成が複雑** | 法人税申告書の別表は数十種類あり、相互参照関係が複雑 |
| **税額控除の見落とし** | 中小企業向けの優遇税制（投資促進税制、賃上げ税制等）の適用漏れ |
| **申告調整の理解不足** | 会計上の利益と税務上の所得の差異（加算・減算項目）の把握が困難 |
| **所得税の控除判定** | 医療費控除、寄附金控除、住宅ローン控除等の適用判定と計算が煩雑 |
| **税制改正の追従** | 毎年の税制改正で適用要件や税率が変更される |
| **チェック体制の不足** | ダブルチェックの人員不足で計算ミスが見逃される |

---

## Claudeによる自動化・効率化

### ステップ1: 決算書データの投入と申告調整項目の洗い出し

会計上の決算書（損益計算書・貸借対照表）のデータをClaudeに投入し、税務上の加算・減算が必要な項目を洗い出します。

### ステップ2: 別表四（所得の金額の計算に関する明細書）の作成支援

加算項目（損金不算入）と減算項目（益金不算入）を整理し、課税所得金額を算出します。

### ステップ3: 税額控除・特別償却の適用判定

自社が適用可能な税額控除・特別償却制度を網羅的にチェックします。

### ステップ4: 申告書の整合性チェック

各別表間の数値の整合性、決算書との突合を行います。

### ステップ5: 所得税確定申告の計算支援

個人事業主や給与所得者の確定申告について、控除計算と税額の検算を行います。

### プロンプト4: 申告書の整合性チェックリスト

```
あなたは法人税申告書のレビュー担当（品質管理部門）の税理士です。以下の法人税申告書の主要データについて、整合性チェックを行ってください。

【チェック対象データ】
- 決算書（確定決算の損益計算書・貸借対照表）
- 別表一（法人税額の計算）
- 別表四（所得の金額の計算）
- 別表五（一）（利益積立金額の計算）
- 別表五（二）（租税公課の納付状況）
- 別表六（所得税額の控除）
- 別表十一（貸倒引当金）
- 別表十五（交際費等）
- 別表十六（減価償却）

【整合性チェック項目】
1. 別表四の「当期利益」= 損益計算書の「税引前当期純利益」であるか
2. 別表四の「所得金額」= 別表一の「所得金額」であるか
3. 別表五（一）の期首残高 = 前期末の別表五（一）の期末残高であるか
4. 別表五（一）の当期の増減 = 別表四の留保欄の合計と整合しているか
5. 別表五（二）の未納法人税等 = 貸借対照表の未払法人税等と整合しているか
6. 別表十五の交際費等支出額 = 別表四の加算額と整合しているか
7. 別表十六の減価償却費 = 決算書の減価償却費と整合しているか（超過額は別表四で加算）
8. 別表六の控除所得税額 = 源泉徴収税額の集計と一致しているか
9. 別表十一の貸倒引当金 = 貸借対照表の貸倒引当金と整合しているか

各チェック項目について【チェック方法】【不一致の場合の原因候補】【修正手順】を出力してください。チェック用のExcel風テンプレートも作成してください。
```

## 注意点・法的留意事項

### AIの限界

- **申告書の最終作成・提出は税理士が行うべきです。** 特に法人税申告書は別表間の関係が複雑であり、Claudeの出力を鵜呑みにせず、必ず税理士による最終チェックを受けてください。
- **税額控除・特別償却の適用要件は毎年変更されます。** 2026年時点の最新情報は国税庁の通達・パンフレットで確認してください。
- **数値の正確性はClaudeが保証できません。** 特に税率の適用区分、端数処理、速算表の適用など、計算の各ステップを人間が検算してください。

### 税法上の注意

| 項目 | 注意点 |
|------|--------|
| **法人税の申告期限** | 原則は事業年度終了後2ヶ月以内。申告期限の延長特例の届出で1ヶ月延長可 |
| **青色申告の取消し** | 帳簿書類の備付け・記録・保存の不備で取り消される可能性あり |
| **繰越欠損金の控除限度** | 中小法人以外は所得金額の50%が上限 |
| **外形標準課税** | 資本金1億円超の法人は法人事業税に外形標準課税が適用される |
| **電子申告の義務化** | 大法人（資本金1億円超）は法人税・消費税の電子申告が義務 |

### 実務上のベストプラクティス

1. **申告書作成前に税理士と打合せを行い、当期の特殊事項（設備投資、組織再編、特別損失等）を共有する**
2. **税額控除の適用漏れを防ぐため、毎期「適用可能な優遇税制チェックリスト」を確認する**
3. **申告書は2名以上でクロスチェックする**（Claudeを「2人目のチェッカー」として活用）
4. **前期の申告書を必ず参照し、継続性を確認する**
5. **中間申告の仮決算を検討する**（業績悪化時は中間で還付を受けられる場合がある）

## まとめ

確定申告は税務実務の集大成であり、法人税の別表作成、所得税の控除計算、消費税の申告など、複雑な計算と判断が求められます。Claudeは、計算の検算、税額控除の適用判定、申告書間の整合性チェックなど、「ダブルチェックの相手」として非常に有用です。

特に中小企業向けの税額控除・特別償却は種類が多く、適用漏れが起きやすい分野です。Claudeに自社の情報を投入し、適用可能な制度を網羅的にチェックさせることで、「知らなかったために使えなかった」という事態を防ぐことができます。

ただし、申告書の最終作成・提出は必ず税理士が行ってください。特に法人税の別表は相互参照関係が複雑であり、1箇所の誤りが連鎖的に波及します。Claudeはあくまで「補助」であり、最終責任は人間にあることを常に意識してください。

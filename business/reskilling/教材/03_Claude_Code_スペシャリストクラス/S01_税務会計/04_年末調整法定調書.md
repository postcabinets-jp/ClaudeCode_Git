## 業務の概要

**年末調整**とは、給与所得者の1年間の所得税を正確に計算し、毎月の源泉徴収税額との差額を精算する手続きです。**法定調書**は、税務署に提出が義務付けられた書類（給与所得の源泉徴収票、報酬等の支払調書など）の総称です。

- **誰がやるか**: 給与計算担当者、経理担当者、税理士事務所、社会保険労務士
- **なぜ重要か**: 年末調整は従業員にとって「確定申告の代替」であり、正確な処理は従業員の税負担に直結します。法定調書の提出漏れや記載誤りは税務署からの指摘対象となり、場合によっては罰則（1年以下の懲役または50万円以下の罰金）もあります。

---

## 従来の課題

| 課題 | 具体的な影響 |
|------|------------|
| **扶養控除等申告書の記載ミス** | 従業員が記入した内容の誤りを経理がチェックする負担が大きい |
| **各種控除の適用判定が複雑** | 配偶者控除、配偶者特別控除、扶養控除、保険料控除、住宅借入金等特別控除の要件が複雑 |
| **所得金額調整控除の見落とし** | 給与収入850万円超の従業員に適用される比較的新しい控除の見落とし |
| **法定調書の作成が手作業** | 源泉徴収票・支払調書を1件ずつ作成する手間 |
| **マイナンバーの管理リスク** | 特定個人情報の取扱いに関する厳格なルールへの対応 |
| **年々の税制改正への対応** | 毎年のように控除額や適用要件が変更される |

## Claudeによる自動化・効率化

### ステップ1: 扶養控除等申告書の記載チェック

従業員から提出された申告書の内容をClaudeに入力し、記載内容の妥当性をチェックします。

### ステップ2: 各種控除額の計算確認

配偶者控除・配偶者特別控除・扶養控除・保険料控除等の計算をClaudeで検算します。

### ステップ3: 年末調整計算の検証

給与計算ソフトの計算結果をClaudeに入力し、計算過程を検証します。

### ステップ4: 法定調書の記載内容チェック

源泉徴収票・支払調書の記載内容が正確かどうかをチェックします。

### ステップ5: 従業員向け説明文の生成

年末調整の結果を従業員にわかりやすく説明する文書をClaudeに生成させます。

## 注意点・法的留意事項

### AIの限界（特に重要）

- **マイナンバー（個人番号）は絶対にClaudeに入力しないでください。** マイナンバーは特定個人情報であり、外部のAIサービスに送信することは個人情報保護法・マイナンバー法に違反する可能性があります。
- **従業員の個人情報（氏名、生年月日、住所、給与額等）をClaudeに入力する際は、匿名化・仮名化するか、自社のセキュリティポリシーに照らして判断してください。**
- **税制改正は毎年行われます。** 2026年分の計算をする際は、最新の税制改正大綱・施行された法律を必ず確認してください。Claudeの学習データが最新でない可能性があります。

### 税法上の注意

| 項目 | 注意点 |
|------|--------|
| **扶養控除の所得要件** | 合計所得金額48万円以下（給与収入のみなら103万円以下） |
| **配偶者控除の適用除外** | 本人の合計所得金額が1,000万円超の場合、配偶者控除は適用不可 |
| **16歳未満の扶養親族** | 扶養控除の対象外（住民税の非課税判定には影響するため申告書への記載は必要） |
| **源泉徴収票の電子交付** | 従業員の同意があれば電子交付可。ただし書面での交付を求められた場合は応じる必要あり |
| **年末調整の対象外** | 給与収入2,000万円超、災害減免法適用者、2箇所以上から給与を受ける者（従たる給与）等 |

### マイナンバーの取扱い

| 原則 | 内容 |
|------|------|
| **利用目的の限定** | 税務・社会保険の手続きにのみ使用可 |
| **安全管理措置** | 物理的・技術的安全管理措置を講じる義務 |
| **収集・保管の制限** | 必要がなくなった場合は速やかに廃棄 |
| **提供の制限** | 法令に基づく場合を除き第三者への提供禁止 |
| **AIサービスへの入力禁止** | 外部クラウドサービスへの送信はマイナンバー法違反のリスクあり |

## まとめ

年末調整と法定調書は、年に1度の大きな実務イベントです。扶養控除等申告書のチェック、各種控除額の計算確認、源泉徴収票・支払調書の作成支援など、Claudeを活用できるポイントは多数あります。

特に計算の検算や、複雑な控除判定のフローチェックにClaudeは威力を発揮します。「配偶者の所得が48万円を超えるか超えないかで控除額が変わる」「扶養親族の年齢で控除区分が変わる」といった判定ロジックのチェックに最適です。

ただし、マイナンバーは絶対にClaudeに入力しないでください。また、実際の計算は給与計算ソフトで行い、Claudeは検算・チェックの補助として活用してください。最終確認は必ず税理士が行ってください。

# 02. 月次決算試算表作成
**Claude Code スペシャリストクラス — S01 税務・会計**
**POST CABINETS Inc.**

---

## この講義で学ぶこと

1. 月次決算の締め処理をClaudeで効率化し、試算表の作成スピードを向上させる方法
2. 試算表データの異常値検出・前月比分析をAIに任せるテクニック
3. 経営者が意思決定に使える月次報告レポートの自動生成

---

## 業務の概要

**月次決算**とは、毎月末に帳簿を締め、その月の収益・費用・資産・負債の状況を「試算表（Trial Balance）」として集計する業務です。年次決算と異なり法的義務はありませんが、経営の現状をタイムリーに把握するために不可欠です。

- **誰がやるか**: 経理部門、税理士事務所、管理会計担当者
- **なぜ重要か**: 月次決算が遅れると経営判断が遅れます。「先月の数字がわからない」状態で投資判断や資金繰りの意思決定はできません。上場企業では月次決算の早期化（翌月5営業日以内等）が求められ、中小企業でも金融機関への報告や経営改善のために月次試算表の作成は重要です。

---

## 従来の課題

| 課題 | 具体的な影響 |
|------|------------|
| **締め処理の遅延** | 未計上の請求書、経費精算の遅れで月次が翌月15日以降にずれ込む |
| **異常値の見落とし** | 前月比で大きく変動した科目を見逃し、誤計上が決算まで放置される |
| **経営者向け説明の負担** | 数字の羅列では経営者が理解できず、毎月「説明資料」を手作業で作成 |
| **比較分析の手間** | 前年同月比、予算対比、部門別比較を手作業でExcelに展開 |
| **決算整理仕訳の漏れ** | 減価償却費、前払費用の按分、引当金の計上漏れ |

---

## Claudeによる自動化・効率化

### ステップ1: 月次締めチェックリストの実行

Claudeに月次締めの標準チェックリストを実行させ、漏れがないか確認します。

### ステップ2: 試算表データの投入と異常値検出

会計ソフトから出力した試算表データ（CSV/テキスト）をClaudeに投入し、異常値を自動検出します。

### ステップ3: 前月比・前年同月比分析

増減が大きい科目をピックアップし、原因の仮説を提示します。

### ステップ4: 経営者向けサマリーレポートの生成

数字を「経営者が読めるストーリー」に変換し、重要ポイントを強調したレポートを生成します。

### ステップ5: 決算整理仕訳の提案

月次で計上すべき決算整理仕訳（減価償却、前払費用按分、賞与引当金等）を提案します。

給料手当: 2,800 / 法定福利費: 420 / 福利厚生費: 280

## 注意点・法的留意事項

### AIの限界

- **試算表の「数字そのもの」は会計ソフトから正確に出力する必要があります。** Claudeに数字を手入力する際の転記ミスに注意してください。
- **異常値の「原因」はAIの推測です。** 実際の原因確認は必ず担当者へのヒアリングや証憑の確認で行ってください。
- **将来予測（来月の見通し等）は参考情報です。** 経営判断の材料にする場合は、複数のシナリオを検討してください。

### 法的留意事項

| 項目 | 注意点 |
|------|--------|
| **会社法上の計算書類** | 月次試算表は法定書類ではないが、年次決算書類（貸借対照表・損益計算書等）の基礎となる |
| **金融機関への提出** | 融資先の銀行から月次試算表の提出を求められることがある。正確性が求められる |
| **税務申告との整合性** | 月次の累計が年次決算と整合するよう、決算整理仕訳の月次按分を適切に行う |
| **管理会計と制度会計の区別** | 部門別損益等の管理会計データと、制度会計（税務申告用）のデータは区別する |

### 実務上のベストプラクティス

1. **月次決算は翌月10営業日以内を目標に** — スピードと正確性のバランスを取る
2. **異常値の閾値を事前に設定** — 前月比20%以上の変動は自動アラート
3. **経営者レポートは定型フォーマットで** — 毎月同じ形式だと変化が見えやすい
4. **決算整理仕訳は年間スケジュールで管理** — 毎月やるもの・四半期のもの・年1回のものを分ける

## まとめ

月次決算試算表の作成は、経理実務の中核であり、経営判断の基盤です。Claudeを活用することで、試算表データの異常値検出、前月比・前年比分析、経営者向けレポートの生成を大幅に効率化できます。

特に「数字を経営者にわかる言葉に翻訳する」という作業は、従来は経理担当者の大きな負担でしたが、Claudeに定型フォーマットで出力させることで、品質を保ちながら時間を短縮できます。

ただし、数字の正確性は会計ソフトのデータに依存します。Claudeへの入力時の転記ミスには十分注意し、最終確認は必ず人間が行ってください。

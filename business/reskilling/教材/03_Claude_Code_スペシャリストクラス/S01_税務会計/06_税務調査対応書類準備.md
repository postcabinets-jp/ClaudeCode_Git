## 業務の概要

**税務調査**とは、税務署が納税者の申告内容が正しいかどうかを確認するために行う調査です。法人は概ね3〜5年に1回の頻度で調査を受けます。調査には「任意調査」（事前通知あり）と「強制調査（査察）」がありますが、一般的な中小企業が受けるのは任意調査です。

- **誰がやるか**: 経営者、経理担当者、顧問税理士（税務代理人）
- **なぜ重要か**: 調査で誤りが指摘されると、修正申告による追加納税に加え、加算税（過少申告加算税10〜15%、重加算税35〜40%）や延滞税が課されます。適切な事前準備により、調査のスムーズな進行と不利益の最小化が可能です。

---

## 従来の課題

| 課題 | 具体的な影響 |
|------|------------|
| **事前準備に時間がかかる** | 過去3〜5年分の書類・データの整理に膨大な時間を要する |
| **想定質問が読めない** | 何を聞かれるかわからない不安から過剰に準備するか、逆に準備不足になる |
| **書類の所在が不明** | 「あの時の契約書はどこ？」で調査当日にパニックになる |
| **調査官への回答が不適切** | 余計なことを言ったり、質問の意図を誤解して不正確な回答をする |
| **修正申告の要否判断が遅い** | 指摘事項が修正申告すべきものか、争えるものかの判断に時間がかかる |
| **社内の情報共有不足** | 経理と営業で取引の認識が異なり、調査官に不信感を与える |

### プロンプト2: 業種別の想定質問と回答準備

```
あなたは税務調査対応の専門コンサルタントです。以下のWebマーケティング会社に対する税務調査で想定される質問リストを作成し、各質問への模範的な回答方針を提案してください。

【業種特性と調査官が注目しやすいポイント】
- Webマーケティング・広告代理業
- 無形のサービス提供（成果物が目に見えにくい）
- フリーランスへの外注が多い（偽装外注のリスク）
- 広告費の先払い・後払いが混在（期間帰属の問題）
- 接待交際費が多い業種

【想定質問カテゴリ】
1. 売上の計上基準と時期
2. 外注費と給与の区分
3. 交際費と会議費の区分
4. 代表者の個人的支出の混入
5. 消費税の課税区分
6. 在庫（仕掛品）の計上
7. 関連当事者取引
8. 海外取引（クラウドサービス等）
9. 売掛金の回収状況と貸倒れ
10. 帳簿書類の保存状況

【出力要件】
各カテゴリについて:
1. 想定される具体的な質問（3〜5個ずつ）
2. 質問の裏にある調査官の意図（何を確認したいのか）
3. 適切な回答方針（「こう答えるべき」「これは言ってはいけない」）
4. 回答時に手元に準備しておくべき資料
5. 「グレーゾーン」の判断（争点になりうるポイント）

【重要な注意】
- 回答は事実に基づくこと（虚偽の回答は絶対にしない）
- 不確かな場合は「確認して後日回答します」と回答する方針
- 顧問税理士が同席する前提
```

```
あなたは国税局OBの税理士です。以下の3期分の申告データを分析し、税務調査で指摘される可能性が高いポイントを洗い出してください。

【3期分の主要データ】

■ 2024年3月期
売上高: 80,000,000円 / 売上総利益率: 62% / 営業利益: 5,200,000円
交際費支出: 3,800,000円 / 外注費: 18,000,000円 / 広告宣伝費: 8,000,000円
役員報酬: 12,000,000円（代表1名） / 旅費交通費: 2,400,000円

■ 2025年3月期
売上高: 95,000,000円 / 売上総利益率: 58% / 営業利益: 4,800,000円
交際費支出: 5,200,000円 / 外注費: 25,000,000円 / 広告宣伝費: 12,000,000円
役員報酬: 15,000,000円（代表1名） / 旅費交通費: 3,600,000円

■ 2026年3月期
売上高: 120,000,000円 / 売上総利益率: 44% / 営業利益: -975,000円
交際費支出: 6,500,000円 / 外注費: 35,000,000円 / 広告宣伝費: 15,000,000円
役員報酬: 18,000,000円（代表1名） / 旅費交通費: 5,400,000円

【分析要件】
1. 3期比較で異常な変動がある項目の特定（増加率、対売上比率の変動）
2. 業種平均との比較で乖離がある項目（Webマーケティング業の一般的な費用構造との比較）
3. 調査官が「掘り下げて調べたい」と感じるであろうポイントのランキング（Top10）
4. 各ポイントについて、合理的な説明ができるかどうかの事前チェック項目

【リスクレベル】
各指摘ポイントについて:
- 🔴 高リスク（高確率で指摘される。修正申告の可能性大）

- 🟢 低リスク（可能性は低いが準備はしておくべき）
```

### プロンプト4: 修正申告の要否判断フレームワーク

```
あなたは税務調査での修正申告の要否判断に精通した税理士です。以下の調査官からの指摘事項について、修正申告すべきかどうかの判断フレームワークを提供してください。

【調査官からの指摘事項（仮定）】

指摘1: 外注費のうち500万円分について、「これは実質的に給与ではないか」との指摘
- 対象: フリーランスAさんへの月額報酬（月40万円 × 12ヶ月 + 追加作業2件）
- 調査官の論拠: 出勤時間の管理あり、専属的、自社PCを使用
- 当社の認識: 業務委託契約に基づく外注費

指摘2: 交際費のうち80万円分について、「代表者の個人的な飲食ではないか」との指摘
- 対象: 代表者名義のクレジットカードで支払った飲食代
- 調査官の論拠: 接待記録がない、相手先が記載されていない
- 当社の認識: 全て取引先との会食

指摘3: 売上の計上時期について、「3月に納品完了した案件が翌期の売上になっている」との指摘
- 対象: 3月25日に検収書を受領した案件（800万円）の計上が4月
- 調査官の論拠: 検収書の日付が3月
- 当社の認識: 請求書発行が4月のため4月に計上

【判断フレームワークの要件】
各指摘事項について:
1. 法的根拠の分析（関連する税法の条文・通達・判例）
2. 指摘が正当かどうかの客観的評価（調査官の主張と当社の主張のどちらに理があるか）
3. 修正申告した場合の追加税額の概算
4. 修正申告しない場合（更正処分を受ける場合）のリスクとメリット
5. 交渉の余地がある項目の特定
6. 今後の再発防止策の提案
7. 判断に迷う場合の「判断基準チェックリスト」

【重要な注意】
- これはあくまでシミュレーションであり、実際の調査対応は顧問税理士と相談の上で行ってください
- 事実を隠蔽・仮装する対応は絶対にしてはいけません
```

```markdown
- □ 税理士が同席する場合は、税理士の指示に従う
- □ メモを取る（調査官が何を質問したか、何を見たか）
```

## 注意点・法的留意事項

### AIの限界

- **税務調査への対応は、必ず顧問税理士と一緒に行ってください。** Claudeで準備した想定質問・回答は「練習」であり、実際の調査では税理士の判断に従ってください。
- **調査官の質問の意図は文脈によって異なります。** 同じ質問でも調査官の真意はその場の状況次第であり、AIには判断できません。
- **修正申告の要否は重大な判断です。** 税額への影響が大きいため、必ず税理士と協議の上で決定してください。

### 法的留意事項

| 項目 | 注意点 |
|------|--------|
| **質問検査権** | 調査官は法律に基づく質問検査権を持つ（国税通則法第74条の2）。正当な理由なく拒否すると罰則あり |
| **事前通知** | 任意調査は原則として事前通知が必要（国税通則法第74条の9）。ただし事前通知なしの調査もあり得る |
| **守秘義務** | 調査官には守秘義務がある。調査で知り得た情報は目的外に使用されない |
| **修正申告と更正の違い** | 修正申告は納税者の自主的な行為、更正は税務署の処分。更正処分には不服申立てが可能 |
| **帳簿書類の提示拒否** | 正当な理由なく帳簿の提示を拒否すると、青色申告の取消しリスクあり |

### 実務上のベストプラクティス

1. **日頃から書類を整理しておく** — 調査通知を受けてから慌てない
2. **調査当日は冷静に対応する** — 感情的にならない
3. **調査官を「敵」と思わない** — 協力的な姿勢が調査をスムーズにする
4. **指摘事項は全てメモする** — 後で税理士と検討するため
5. **修正申告は早めに判断する** — 調査完了前の自主的な修正は加算税が軽減される場合がある

## まとめ

税務調査は、多くの経営者にとって心理的な負担が大きいイベントですが、適切な準備によりスムーズに乗り切ることができます。Claudeを活用して、事前準備チェックリストの作成、想定質問と回答の準備、書類の棚卸し、申告内容のセルフチェックを行うことで、準備の質と効率を大幅に向上させることができます。

特に「自社の申告内容のどこが指摘されやすいか」というセルフチェックは、Claudeに過去のデータを分析させることで客観的な視点を得られます。

ただし、実際の調査対応は必ず顧問税理士と共に行ってください。税務調査は法的な手続きであり、対応を誤ると大きな不利益を被る可能性があります。Claudeはあくまで「事前準備の補助」として活用してください。

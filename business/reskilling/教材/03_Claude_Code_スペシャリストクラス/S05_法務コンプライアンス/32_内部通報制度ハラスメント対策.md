# 32. 内部通報制度・ハラスメント対策
**Claude Code スペシャリストクラス — S05 法務・コンプライアンス**
**POST CABINETS Inc.**

> **重要な前提**: AIによる内部通報制度設計・ハラスメント対策の支援は、あくまで「設計補助・ドラフト支援」です。法的助言は弁護士法第72条により弁護士のみが行えます。制度設計・運用にあたっては、弁護士・社労士等の専門家の関与を推奨します。また、個別の通報案件や相談案件の判断にAIを使用しないでください。

## この講義で学ぶこと

- 公益通報者保護法に対応した内部通報制度の設計方法
- 通報窓口の運用規程・調査手順書の作成手法
- ハラスメント防止研修資料のClaude活用による作成
- 相談対応マニュアルの整備と実務ポイント

## 業務の概要

### 内部通報制度
企業内の不正行為を早期に発見・是正するための仕組みです。

**法的背景:**
- **公益通報者保護法**（2022年6月施行の改正法）
  - 従業員300人超の企業: 内部通報体制の整備が**義務**
  - 従業員300人以下の企業: 体制整備は**努力義務**
  - 従事者の守秘義務違反に対する**刑事罰**が新設

### ハラスメント対策
職場におけるハラスメントを防止し、発生した場合に適切に対応するための仕組みです。

**法的背景:**
- **パワーハラスメント防止法**（労働施策総合推進法、2022年4月〜全企業義務化）
- **セクシュアルハラスメント防止**（男女雇用機会均等法）
- **マタニティハラスメント防止**（男女雇用機会均等法、育児・介護休業法）
- **カスタマーハラスメント対策**（2024年以降、対策強化の動き）

---

## 従来の課題

| 課題 | 詳細 |
|------|------|
| **通報制度の形骸化** | 制度はあるが実際に利用されない、信頼されていない |
| **通報者保護の不徹底** | 通報者が不利益取扱いを受けるケースが後を絶たない |
| **調査手順の未整備** | 通報を受けた後の対応が場当たり的になる |
| **ハラスメントの判断基準の曖昧さ** | 何がハラスメントに該当するかの基準が社内で統一されていない |
| **相談対応のスキル不足** | 相談担当者のトレーニングが不十分 |
| **研修のマンネリ化** | 毎年同じ内容で社員の意識が変わらない |

## Claudeによる自動化・効率化

### できること
- 内部通報制度の運用規程ドラフト作成
- 調査手順書・マニュアルの作成
- ハラスメント防止研修資料の作成（ケーススタディ付き）
- 相談対応マニュアルの作成
- ハラスメント判断基準のチェックリスト作成
- 通報制度の周知用資料（ポスター、メール文面等）の作成
- カスタマーハラスメント対応マニュアルの作成

### できないこと（絶対にAIに委ねてはいけないこと）
- **個別の通報案件の判断**（事実認定、処分の決定）
- **個別のハラスメント相談への対応**（被害者・行為者への面談）
- **懲戒処分の決定**
- 法的助言・法的判断の最終決定
- 被害者のメンタルケア（専門家に委ねること）

### プロンプト2: 調査手順書の作成

```
内部通報を受けた後の調査手順書を作成してください。

【手順書の構成】
1. 受付段階
   - 通報内容の記録方法
   - 初期評価（受理/不受理の判断基準）
   - 通報者への受理通知
   - 利益相反の確認（調査担当者が関係者でないか）

2. 調査計画の策定
   - 調査チームの編成
   - 調査範囲・方法の決定
   - スケジュールの策定
   - 調査中の情報管理ルール

3. 事実調査
   - 書類・データの収集・保全
   - 関係者へのヒアリング手順
     - ヒアリングの順序（通報者→関係者→対象者）
     - ヒアリングの記録方法
     - ヒアリング時の注意事項
   - 証拠の評価基準

4. 事実認定と判断
   - 事実認定の基準（証拠の優越）
   - 調査報告書の作成フォーマット

5. 是正措置と処分
   - 是正措置の種類と選定基準
   - 懲戒処分との関係
   - 通報者へのフィードバック

6. フォローアップ
   - 再発防止策の実施確認
   - 通報者への不利益がないかの確認（定期的）
   - 調査記録の保管（保管期間、アクセス制限）

【出力】
各ステップの詳細手順と、使用するフォーム・テンプレートを含めてください。
```

### プロンプト3: ハラスメント防止研修資料

```
以下の条件でハラスメント防止研修資料を作成してください。

【研修種別】:
□ 全社員向け基礎研修
□ 管理職向け研修（相談対応を含む）
□ 新入社員向け研修

【研修テーマ】:
□ パワーハラスメント
□ セクシュアルハラスメント
□ マタニティハラスメント（含むパタハラ、ケアハラ）
□ カスタマーハラスメント
□ 全テーマ包括

【所要時間】: ○分

【研修資料の構成】
1. ハラスメントの定義（法律上の定義、3要素）
   - パワハラの6類型
   - セクハラの2類型（対価型・環境型）
   - マタハラの類型
2. 「グレーゾーン」の判断基準
   - 指導とパワハラの境界線
   - コミュニケーションとセクハラの境界線
   - 「受け手の感じ方」と「客観的基準」の関係
3. ケーススタディ（5問以上）
   各ケースについて:
   - シナリオ（具体的な場面設定）
   - 「これはハラスメント?」（Yes/No/グレー）
   - 解説（判断の根拠、裁判例の参照）
   - より良い対応例
4. ハラスメントを見聞きした場合の対応
   - 当事者としての対応
   - 第三者（バイスタンダー）としての対応
   - 管理職としての対応
5. 相談窓口の案内
6. 確認テスト

【出力形式】
- スライド構成案（ページごとの内容、ビジュアルの指示）
- 講師用ガイド（発問例、ディスカッションの進め方）
- 受講者配布資料（相談窓口カード、チェックリスト）
```

### プロンプト4: 相談対応マニュアル

```
ハラスメント相談窓口担当者向けの相談対応マニュアルを作成してください。

【マニュアルの構成】
1. 相談担当者の心構え
   - 中立性の維持
   - 傾聴の姿勢
   - 二次被害の防止
   - 守秘義務の徹底

2. 相談受付の手順
   - 初期対応（最初の接触〜面談設定）
   - 面談の進め方
     - 面談の場所・環境（プライバシーの確保）
     - 面談の時間配分
     - ヒアリング項目（5W1H）
     - 記録の取り方
   - 相談者の意向の確認（調査希望の有無）

3. ヒアリングで使うフレーズ集
   - 冒頭の説明（「この相談の目的は...」「秘密は守ります」等）
   - 傾聴のフレーズ（「それは辛かったですね」「もう少し詳しく教えてください」等）
   - 事実確認のフレーズ（「いつ頃のことですか」「他に見ていた方は」等）
   - 避けるべきフレーズ（「あなたにも問題が...」「考えすぎでは」等）

4. 相談後の対応
   - 調査への移行手順
   - 緊急対応が必要な場合（配置転換、接触禁止等）
   - メンタルヘルスケアへの橋渡し（産業医、EAP）
   - フォローアップ面談

5. 記録・報告
   - 相談記録フォーマット
   - 報告先と報告内容
   - 記録の保管ルール

6. 困難なケースへの対応
   - 匿名の相談
   - 相談者が加害者である可能性がある場合
   - 相談者が退職を希望している場合
   - 事実関係が不明確な場合

【出力】
各セクションを詳細に記載し、実際に使えるフォームのテンプレートも含めてください。
```

   - メンタルヘルスケア
   - 対応好事例の共有

6. 組織としての対応方針
   - 「毅然とした対応」の社内合意形成
   - 対応困難顧客リストの管理（個人情報保護法との関係）
   - 法的措置の判断基準

   - 連絡先（電話/メール/Web）

### パワーハラスメント判定チェックリスト

```markdown
## パワーハラスメント該当性チェックリスト

### 3要素の確認（すべて該当でパワハラに該当）

#### 要素1: 優越的な関係を背景とした言動
- [ ] 行為者は相談者より職位が上である
- [ ] 行為者は業務上の指揮命令権を持つ
- [ ] 行為者は専門的知識・経験において優位にある
- [ ] 集団で行われており、抵抗・拒否が困難である

#### 要素2: 業務上必要かつ相当な範囲を超えた言動
- [ ] 業務上の必要性がない（私的な感情に基づく等）
- [ ] 業務上の必要性があっても、態様が相当でない
- [ ] 繰り返し行われている
- [ ] 行為者の目的と手段が不均衡である

#### 要素3: 労働者の就業環境が害される
- [ ] 身体的な苦痛を与えている
- [ ] 精神的な苦痛を与えている
- [ ] 相談者が通常業務を遂行できない状態になっている
- [ ] 「平均的な労働者」の感じ方として看過できないレベルである

### 6類型の確認
- [ ] 身体的な攻撃（暴行・傷害）
- [ ] 精神的な攻撃（脅迫・名誉毀損・侮辱・暴言）
- [ ] 人間関係からの切り離し（隔離・仲間外し・無視）
- [ ] 過大な要求（業務上不要なこと、遂行不可能なことの強制）
- [ ] 過小な要求（能力や経験とかけ離れた程度の低い仕事、仕事を与えない）
- [ ] 個の侵害（私的なことに過度に立ち入る）

### 判定
- [ ] パワーハラスメントに該当する可能性が高い → 調査へ
- [ ] グレーゾーン → 詳細調査が必要
- [ ] パワーハラスメントに該当しない可能性が高い → 指導として対応

※ 本チェックリストは初期判断の参考であり、最終判定は調査結果に基づき行う。
```

## 注意点・法的留意事項

### 弁護士法との関係
- AIによる制度設計支援は**フレームワークの提示**であり、法的助言ではありません
- 個別の通報案件・ハラスメント案件の法的判断は弁護士に確認してください
- 懲戒処分の決定にあたっては、労働法に精通した弁護士・社労士への相談を推奨します

### 公益通報者保護法の重要ポイント
- **従事者の守秘義務違反は刑事罰**（30万円以下の罰金）の対象です（2022年改正）
- 通報者への**不利益取扱い**は法律で禁止されています
- **通報者を探索する行為**も禁止です
- 内部通報体制の整備義務に違反した場合、行政指導・公表の対象となります

### ハラスメント対策の法的義務
- パワハラ防止措置は**全企業に義務化**されています（2022年4月〜）
- 事業主が講ずべき措置（厚労省指針）:
  1. 方針の明確化と周知・啓発
  2. 相談窓口の設置と適切な対応
  3. 事後の迅速かつ適切な対応
  4. プライバシー保護、不利益取扱いの禁止

### AIを使ってはいけない場面（厳守）
- **個別の通報案件の事実認定や処分判断にAIを使わないでください**
- **ハラスメント相談の対応にAIを使わないでください**（相談者の心理的ケアは人間が行うべき）
- 通報内容・相談内容をAIに入力する場合、秘密保持義務との関係を必ず確認してください

## まとめ

- 内部通報制度は**公益通報者保護法（2022年改正）**により整備が義務化（300人超）/努力義務化（300人以下）されている
- ハラスメント防止措置は**全企業に義務化**されている
- Claudeを活用して、**運用規程・調査手順書・研修資料・相談対応マニュアル**のドラフトを効率的に作成可能
- ただし、**個別案件の判断にAIを使ってはいけない**（事実認定、処分決定、相談対応は人間が行う）
- 制度の実効性を確保するために、**通報者保護の徹底**と**従事者の守秘義務**の意識付けが重要
- AIの出力は**ドラフト・素材**として活用し、**最終的な制度設計は弁護士・社労士と連携**すること

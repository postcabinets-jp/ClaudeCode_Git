# 30. 知的財産管理
**Claude Code スペシャリストクラス — S05 法務・コンプライアンス**
**POST CABINETS Inc.**

> **重要な前提**: AIによる知的財産関連業務の支援は、あくまで「調査補助・ドラフト支援」です。特許出願・商標出願等の代理は弁理士法により弁理士のみが行えます（弁護士を含む）。また、法的助言は弁護士法第72条の制約があります。最終判断は必ず弁理士・弁護士に確認してください。

## この講義で学ぶこと

- 商標出願の事前調査・出願書類作成をClaudeで支援する方法
- 特許明細書のドラフト支援と先行技術調査の効率化
- 著作権管理の基礎と実務上のポイント
- ライセンス管理台帳の設計・運用方法

## 業務の概要

知的財産管理は、企業の無形資産を守り、活用するための業務です。

| 知財の種類 | 保護対象 | 権利期間 | 管轄 |
|-----------|---------|---------|------|
| **特許権** | 発明（技術的アイデア） | 出願から20年 | 特許庁 |
| **商標権** | ブランド名・ロゴ | 登録から10年（更新可） | 特許庁 |
| **著作権** | 創作的表現 | 著作者の死後70年 | 無方式主義（登録不要） |
| **意匠権** | デザイン | 出願から25年 | 特許庁 |
| **営業秘密** | ノウハウ等 | 秘密管理が続く限り | 不正競争防止法 |

---

## 従来の課題

| 課題 | 詳細 |
|------|------|
| **調査に膨大な時間** | 商標・特許の先行調査に数日〜数週間かかる |
| **明細書作成の専門性** | 特許明細書の作成には高度な技術知識と法律知識が必要 |
| **管理の煩雑さ** | 多数の知財の期限管理・ライセンス管理が複雑 |
| **コストの高さ** | 弁理士費用が高額で、中小企業にはハードルが高い |
| **社内の知財意識の低さ** | 知財の重要性が社内で認識されていない |

## Claudeによる自動化・効率化

### できること
- 商標の類似調査のための検索キーワード・分類整理
- 特許明細書の初稿ドラフト（請求項、明細書、要約書）
- 先行技術文献の要約・比較分析
- 著作権の帰属判断に関する論点整理
- ライセンス管理台帳のテンプレート作成
- 知財ポートフォリオの整理・分析支援
- 知財研修資料の作成

### できないこと（弁理士・弁護士に委ねるべき事項）
- 特許・商標の出願代理
- 特許性（新規性・進歩性）の最終判断
- 商標の類似性の最終判断
- 知財侵害の法的判断
- 知財訴訟の代理

## プロンプト例（5つ以上、コピペで使えるレベル）

### プロンプト1: 商標出願の事前調査支援

```
あなたは知的財産の調査アシスタントです。
以下の商標について、出願前の事前調査を支援してください。

【出願予定商標】
- 商標名: （文字商標の場合はテキスト）
- 商標の種類: 文字 / 図形 / 結合 / 立体 / 音 / 色彩
- 指定商品・役務: （具体的に記載）
- 希望する区分: （第○類）

【調査項目】
1. 類似商標の調査戦略
   - J-PlatPatでの検索キーワード案（称呼検索・外観検索）
   - 類似群コードの特定
   - 類似する可能性のある商標のパターン
2. 識別力の評価
   - 普通名称・慣用商標に該当しないか
   - 記述的商標（品質等表示）に該当しないか
   - 商標法第3条・第4条の拒絶理由に該当するリスク
3. 出願戦略の提案
   - 指定商品・役務の範囲（広めに取るか絞るか）
   - 区分の選択（コスト vs 保護範囲のバランス）
   - 標準文字出願 vs ロゴ出願の使い分け

【出力フォーマット】
各項目について具体的なアドバイスを出力し、
最後にリスク評価（登録可能性: 高/中/低）をまとめてください。
```

### プロンプト2: 特許明細書のドラフト支援

```
以下の発明について、特許明細書の初稿ドラフトを作成してください。

【発明の概要】
- 発明の名称:
- 技術分野:
- 解決しようとする課題:
- 課題を解決する手段: （技術的な説明）
- 発明の効果:
- 実施形態の説明: （具体的な構成・動作）

【出力する書類】
1. 特許請求の範囲
   - 独立項（請求項1）: 最も広い権利範囲
   - 従属項（請求項2〜5程度）: 具体的な実施形態に対応
2. 明細書
   - 技術分野
   - 背景技術
   - 発明が解決しようとする課題
   - 課題を解決するための手段
   - 発明の効果
   - 発明を実施するための形態
3. 要約書（400字以内）

【注意事項】
- 請求項は「〜を備えることを特徴とする○○。」の形式で
- 明確性要件（第36条第6項第2号）を意識した記載
- サポート要件（第36条第6項第1号）を満たす記載

※ これはドラフトであり、最終的な出願書類は弁理士の確認・修正が必要です。
```

```
以下のケースについて、著作権の帰属に関する論点を整理してください。

【ケース情報】
- 著作物の種類: （プログラム/デザイン/文章/映像/音楽 等）
- 作成者: （社員/フリーランス/外注先 等）
- 作成の経緯: （業務命令/自発的/委託契約に基づく 等）

【分析してほしい論点】
```

## 注意点・法的留意事項

### 弁理士法との関係
- 特許出願・商標出願の**代理**は弁理士（または弁護士）のみが行えます（弁理士法第75条）
- AIによる明細書ドラフトや調査結果は、あくまで弁理士に提出する**素材**として位置づけてください
- 出願戦略の最終決定は弁理士と相談の上で行ってください

### 弁護士法との関係
- 知財侵害の判断、ライセンス交渉における法的助言は弁護士の業務です
- AIの分析結果を法的判断の根拠として直接使用することは避けてください

### AI生成物に関する注意
- AI生成物の著作物性については法的な議論が進行中です。最新の裁判例・ガイドラインを確認してください
- AI生成物を商標として出願する場合、識別力の要件に留意してください
- AI開発にあたり、学習データの著作権処理（著作権法第30条の4）の適用範囲に注意してください

### 実務上のポイント
- 商標調査は**J-PlatPat**（特許情報プラットフォーム）での確認が必須です。AIの回答のみで判断しないでください
- 特許の新規性喪失の例外適用（公開から1年以内の出願）には厳格な手続要件があります
- 営業秘密として保護するには、「秘密管理性」「有用性」「非公知性」の3要件を満たす管理が必要です

## まとめ

- 知的財産管理は企業の競争力の源泉であり、Claudeを使って**調査・ドラフト作成・管理業務**を効率化できる
- 商標調査のキーワード設計、特許明細書の初稿作成、ライセンス管理台帳の設計などにAIが活用可能
- **AI生成物の知財**は新しい論点であり、文化庁のガイドライン等を注視すること
- 出願代理は**弁理士**の独占業務、法的助言は**弁護士**の業務であり、AIの出力は**素材・ドラフト**にとどめること
- 知財の**期限管理**は失権に直結するため、確実な管理体制を構築すること

# 29. 利用規約・プライバシーポリシー
**Claude Code スペシャリストクラス — S05 法務・コンプライアンス**
**POST CABINETS Inc.**

> **重要な前提**: AIによる利用規約・プライバシーポリシーの作成は、あくまで「ドラフト支援」です。法的助言は弁護士法第72条により弁護士のみが行えます。公開前に必ず弁護士の確認を受けてください。

## この講義で学ぶこと

- Cookie同意バナーの設計と関連規制の理解
- 特定商取引法に基づく表記の作成方法

## 業務の概要

Webサービスやアプリを公開する際には、以下の法的文書の整備が必要です。

| 文書 | 目的 | 根拠法令 |
|------|------|----------|
| **利用規約** | サービス利用条件の定義、トラブル防止 | 民法（定型約款）、消費者契約法 |
| **プライバシーポリシー** | 個人情報の取扱い方針の公表 | 個人情報保護法 |
| **Cookie ポリシー** | Cookie利用の透明性確保 | 電気通信事業法（外部送信規律）、GDPR/ePrivacy |
| **特定商取引法表記** | 通信販売における事業者情報の表示 | 特定商取引法 |

---

## 従来の課題

| 課題 | 詳細 |
|------|------|
| **法改正への追従が困難** | 個人情報保護法、電気通信事業法等の改正に対応できていない |
| **テンプレの流用リスク** | 他社のコピペで自社サービスに合わない条項が混在 |
| **グローバル対応の複雑さ** | GDPR、CCPA等の海外規制への対応が必要な場合の難しさ |
| **専門用語の壁** | 法律用語と技術用語の両方を理解する必要がある |
| **改定作業の手間** | サービス変更のたびに規約改定が必要 |

```

1. 総則（目的、定義、適用範囲）
2. アカウント管理
3. サービス内容・利用条件
4. 禁止事項
```

## 注意点・法的留意事項

### 弁護士法との関係
- AIが生成した利用規約・プライバシーポリシーは**ドラフト（下書き）**です
- 公開前に必ず弁護士のレビューを受けてください
- 特にB2Cサービスの場合、消費者契約法との整合性は弁護士の確認が必須です

### 個人情報保護法の注意点
- 2022年改正で**安全管理措置の公表義務**が追加されました。対応漏れに注意してください
- **漏えい等報告義務**が個人情報保護委員会への報告と本人通知の両方に拡大されています
- **仮名加工情報**制度の新設により、データ利活用の選択肢が広がっています

### 電気通信事業法（外部送信規律）
- 2023年6月施行の改正で、Cookie等の外部送信について**通知・公表義務**が課されました
- Google Analytics、広告タグ等を使用している場合は対応が必要です
- 対応方法は「通知又は公表」「同意取得」「オプトアウト措置」の3択です

### 定型約款に関する注意
- 利用規約は民法の「定型約款」に該当する場合が多く、変更手続に制約があります
- 変更が利用者の利益に適合する場合、または合理的な場合にのみ一方的変更が可能です
- 変更の効力発生時期の相当期間前までにインターネット等で周知する必要があります

### 消費者契約法との関係
- 事業者の故意・重過失による損害賠償責任の全部免除は無効です
- 不当に高額なキャンセル料は無効になります
- 一方的に不利な条項は「不当条項」として無効となる可能性があります

## まとめ

- 利用規約・プライバシーポリシーは、Webサービス運営に**必須の法的文書**
- Claudeを使えば、サービス条件を入力するだけで法令対応のドラフトを効率的に作成可能
- **個人情報保護法2022年改正**（安全管理措置の公表、漏えい報告等）への対応は必須
- **電気通信事業法の外部送信規律**（2023年施行）への対応も忘れずに
- 特に消費者向けサービスでは**消費者契約法**との整合性に注意
- AIの出力は**ドラフト**であり、**公開前に必ず弁護士の確認**を受けること

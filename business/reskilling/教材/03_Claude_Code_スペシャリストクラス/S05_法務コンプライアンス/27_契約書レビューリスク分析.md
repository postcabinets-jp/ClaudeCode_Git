# 27. 契約書レビュー・リスク分析
**Claude Code スペシャリストクラス — S05 法務・コンプライアンス**
**POST CABINETS Inc.**

> **重要な前提**: AIによる契約書レビューは、あくまで「ドラフト支援・チェック補助」です。法的助言は弁護士法第72条により弁護士のみが行えます。最終判断は必ず弁護士に確認してください。

---

## この講義で学ぶこと

- 契約書の条文別チェックポイントを体系的に理解する
- Claudeを使ったリスク条項の自動検出手法を習得する
- 修正提案・交渉ポイントの整理をAIで効率化する方法を学ぶ
- 契約類型別（売買/業務委託/ライセンス/賃貸借）のレビュー観点を身につける

---

## 業務の概要

契約書レビューとは、取引先から提示された契約書や自社で作成した契約書について、法的リスク・商業的リスクを洗い出し、修正すべき点を特定する業務です。

**主な業務フロー:**
1. 契約書の受領・初期確認（契約類型の特定）
2. 条文ごとの精査（リスク条項の特定）
3. リスク評価（高/中/低の分類）
4. 修正案の作成
5. 交渉ポイントの整理
6. 社内承認・弁護士確認

---

## 従来の課題

| 課題 | 詳細 |
|------|------|
| **時間がかかる** | 1件の契約書レビューに数時間〜数日を要する |
| **属人化** | レビューの品質が担当者の経験・知識に依存する |
| **見落としリスク** | 大量の条文から重要なリスク条項を見落とす可能性 |
| **標準化の困難** | チェック観点が統一されていない |
| **交渉準備の負担** | 修正提案と根拠の整理に時間がかかる |

## Claudeによる自動化・効率化

### できること
- 契約書全文の条文別リスクスコアリング
- 一般的に不利とされる条項の自動検出
- 修正案のドラフト生成
- 交渉ポイントと優先度の整理
- 契約類型に応じたチェックリストの自動生成
- 類似条項の比較分析

### できないこと（弁護士に委ねるべき事項）
- 法的助言・法的判断の最終決定
- 訴訟リスクの最終評価
- 規制対応の最終確認
- 契約交渉における法的代理

### プロンプト1: 契約書全体のリスク分析

```

以下の契約書を条文ごとに分析し、リスク評価を行ってください。

【分析フォーマット】
各条文について以下の形式で出力してください：

■ 第○条（条文タイトル）
- 要約: （条文の要旨を1-2文で）
- リスクレベル: 🔴高 / 🟡中 / 🟢低
- リスク内容: （具体的なリスクの説明）
- 修正提案: （リスクを軽減する修正案）
- 交渉優先度: A（必須）/ B（推奨）/ C（あれば望ましい）

最後に「総合リスクサマリー」として、特に注意すべき上位3条項をまとめてください。

【契約書本文】
（ここに契約書テキストを貼り付け）
```

### プロンプト3: 契約類型別チェックリスト生成

```
以下の契約類型に応じた包括的なレビューチェックリストを作成してください。

【契約類型】: （売買契約 / 業務委託契約 / ライセンス契約 / 賃貸借契約 / その他）

【チェックリストに含める項目】
1. 必須条項の有無チェック
2. 各条項の標準的な記載内容との比較
3. 当該契約類型特有のリスクポイント
4. 関連法令との整合性確認ポイント
5. 実務上よく問題になる条項

各項目について「チェック済み □ / 要修正 □ / 該当なし □」の形式で出力してください。
```

### プロンプト4: 損害賠償・免責条項の分析

```
以下の損害賠償条項・免責条項を分析してください。

【分析フレームワーク】
1. 賠償範囲の妥当性（直接損害/間接損害/逸失利益）
2. 賠償上限額の妥当性（契約金額との比率）
3. 免責事由の範囲（過度に広くないか）
4. 故意・重過失の場合の扱い
5. 消費者契約法との整合性（B2Cの場合）
6. 同業他社の標準的な条項との比較
7. 修正提案（当社にとってバランスの取れた案）

【対象条項】
（ここに条項テキストを貼り付け）
```

### プロンプト5: 交渉シナリオの作成

```
以下の契約書について、修正交渉のシナリオを作成してください。

【交渉ポイント一覧】

【出力フォーマット】

■ 交渉ポイント○: （条項名）
```

## 注意点・法的留意事項

### 弁護士法との関係
- **AIによる契約書レビューは「法的助言」には該当しません**が、その結果を法的判断の根拠として直接使用することは避けてください
- AIの出力はあくまで「ドラフト」「チェック補助」として位置づけ、最終判断は弁護士に確認してください
- クライアントに対してAIの分析結果をそのまま「法的助言」として提供することは、弁護士法第72条に抵触する可能性があります

### AI利用上の注意
- **機密情報の取り扱い**: 契約書には営業秘密が含まれる場合があります。AIサービスの利用規約・データ取扱いポリシーを確認してください
- **最新法令の確認**: AIの学習データには最新の法改正が反映されていない場合があります。直近の法改正は別途確認してください
- **業界固有の規制**: 特定業界（金融、医療、建設等）には固有の規制があります。業界専門の弁護士への相談を推奨します

### 実務上のポイント
- レビュー結果は必ず人間の法務担当者がダブルチェックする運用を徹底してください
- AIレビューの精度は入力する契約書テキストの品質に依存します。OCR等で読み取った場合は誤読に注意してください
- 契約書のレビュー履歴・修正履歴は適切に管理してください

## まとめ

- Claudeを活用することで、契約書レビューの**初期スクリーニング**を大幅に効率化できる
- 条文別のリスクスコアリング、修正提案の自動生成、交渉シナリオの作成など、実務の各フェーズでAIを活用可能
- ただし、AIの出力は**ドラフト・チェック補助**であり、**法的助言ではない**
- 最終判断は必ず**弁護士に確認**し、社内の法務承認フローに従うこと
- 契約類型ごとにチェックポイントが異なるため、類型別のプロンプトを使い分けることが効果的

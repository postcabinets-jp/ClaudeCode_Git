# AI導入成果報告書 テンプレート
**POST CABINETS — Claude Code AI組織導入スターターキット**
**Version 1.0 / 2026-03**

---

## 目的

AI導入プログラムの成果を経営層に報告するためのプレゼン資料テンプレート。
「やったこと」ではなく「得られた成果と投資対効果」にフォーカスする構成。

---

## PPT作成プロンプト

```
以下の仕様でPowerPointファイル（.pptx）を作成してください。

■ 全体設定:
- スライドサイズ: 16:9
- フォント: 見出し=游ゴシック Bold、本文=游ゴシック Regular
- 配色:
  - メイン: #1B3A5C（紺）
  - アクセント: #2E86DE（青）
  - 成功: #27AE60（緑）
  - 数字ハイライト: #E74C3C（赤）※注意を引く数字に使用
  - 背景: #FFFFFF
  - テキスト: #2D3436
- ロゴ: 全スライド左上にロゴエリア（3cm×1.5cm）
- ページ番号: 右下
- フッター: 「Confidential — [会社名]」

---

■ スライド1: 表紙

中央揃え:
- タイトル: 「AI導入 成果報告書」フォント36pt、#1B3A5C
- サブタイトル: 「[プログラム名/プロジェクト名]」フォント20pt
- 期間: 「[YYYY年MM月 〜 YYYY年MM月]」フォント16pt
- 報告日: 「[YYYY年MM月DD日]」フォント14pt
- 報告者: 「[部署名] [氏名]」フォント14pt
- 背景装飾: 右下に紺→青のグラデーション帯

---

■ スライド2: エグゼクティブサマリー

KPIカード（横一列、4つ）:
各カード: 幅6cm × 高さ5cm、角丸、影あり

カード1:
- ラベル: 「対象業務数」（12pt、グレー）
- 数値: 「[○]業務」（36pt、太字、#1B3A5C）

■ スライド3: 導入背景

タイトル: 「導入の背景と目的」

2カラムレイアウト:

左カラム（背景）:
見出し: 「なぜAI導入が必要だったか」（16pt、太字）
```

## プレゼンのポイント

### 経営層が知りたいこと（優先順位）

1. **投資対効果（ROI）**: いくら投資して、いくら返ってきたか
2. **再現性**: 他の部署・業務にも展開できるか
3. **リスク**: セキュリティ・法的リスクは管理できているか
4. **今後の展望**: 次は何をするのか、追加投資は必要か

### プレゼン時間の目安

| 所要時間 | 構成 |
|----------|------|
| 5分（エレベーターピッチ） | スライド1, 2, 6のみ |
| 15分（定例報告） | スライド1〜8 |
| 30分（詳細報告） | 全スライド + 質疑応答 |

### 差し替え箇所

すべての[角括弧]内をクライアント企業の実データに差し替えてください。数値は「効果測定レポートテンプレート」のExcelから転記することで一貫性を保てます。

# 今後のAI活用ロードマップ テンプレート
**POST CABINETS — Claude Code AI組織導入スターターキット**
**Version 1.0 / 2026-03**

---

## 目的

AI導入プログラムの「卒業時」に、クライアント企業が経営層に提出するためのロードマップ資料。
「プログラムで得た成果」と「これからの自走計画」を1つの資料にまとめ、継続的なAI活用への経営コミットメントを引き出す。

---

## 想定シーン

- AI導入プログラム（3〜6ヶ月）の最終報告会
- 経営会議でのAI戦略提案
- 次年度予算申請の添付資料

## カスタマイズのポイント

### クライアント企業の規模別

| 規模 | 調整ポイント |
|------|-------------|
| 中小企業（〜100名） | AI推進チーム→推進担当者1名。予算表はシンプルに。「兼務でも回る」安心感を出す |
| 中堅企業（100〜500名） | 部署横断の推進体制を強調。Wave展開の計画が重要 |
| 大企業（500名〜） | ガバナンス・セキュリティの体制を厚く。全社方針との整合性を示す |

### 業種別の強調ポイント

| 業種 | 強調すべき効果 |
|------|---------------|
| 製造業 | 品質向上、不良率低減、予防保全 |
| 金融 | コンプライアンス、リスク管理、顧客対応速度 |
| 小売・サービス | 顧客満足度、在庫最適化、パーソナライゼーション |
| IT | 開発効率、コードレビュー、ドキュメント自動化 |

## 次のステップ

1. プログラム最終月に、このテンプレートに実データを入力
2. クライアント企業のAI推進チームと内容をレビュー
3. 経営層向けプレゼンのリハーサル（推進チームが発表する想定）
4. プレゼン後、承認された項目をNotionのTasksに登録して実行フェーズへ

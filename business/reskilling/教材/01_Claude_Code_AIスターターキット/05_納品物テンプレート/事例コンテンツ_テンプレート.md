# 事例コンテンツ テンプレート
**POST CABINETS — Claude Code AI組織導入スターターキット**
**Version 1.0 / 2026-03**

---

## 目的

クライアント企業のAI活用実績を「事例記事」として資産化するためのテンプレート。
Web記事として公開する場合と、PDF資料として配布する場合の両方に対応。

---

## 活用シーン

- クライアント企業の採用ブランディング（「AI活用先進企業」としてのアピール）
- コンサルタント（POST CABINETS）の実績紹介
- 業界メディアへの寄稿素材
- 社内報での成果共有

## フォーマット別の調整ガイド

### Web記事として公開する場合
- 全体で2,000〜3,000文字が目安
- H2/H3の見出しを使い、スキャンしやすい構成にする
- OGP画像（1200×630px）を用意: 「[会社名]×AI」のビジュアル
- SEOキーワード: 「AI導入 事例」「[業種] AI活用」「業務効率化 AI」
- 末尾にCTAを配置

### PDF資料として配布する場合
- A4縦で6〜8ページが目安
- 表紙（タイトル+企業プロフィール）+ 本文4〜6ページ + 裏表紙（CTA）
- 図表を多用し、ビジュアルで伝える
- ヘッダー/フッターにロゴと「Confidential」表記（非公開の場合）

### 社内報・社内ポータル向け
- 500〜1,000文字に圧縮
- 「課題→解決策→効果（数字）→担当者コメント」の4部構成で十分
- 「あなたの部署でも使えるかも？ 相談は[AI推進チーム]まで」で締める

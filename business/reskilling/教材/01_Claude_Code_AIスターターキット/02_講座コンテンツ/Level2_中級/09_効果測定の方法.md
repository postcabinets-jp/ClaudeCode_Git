### パート1: 何を測るか — 4つの測定カテゴリ（5分）

---

皆さん、こんにちは。今日は「効果測定」の話です。

ここまでで、業務の棚卸し、AI適用ポイントの特定、プロンプト設計、業務フローへの組み込みと進めてきました。あとは「本当に効果が出たのか」を数字で証明するステップです。

なぜ効果測定が必要か。理由は3つあります。

1つ目は、**経営層への説明**。「AIを導入しました。なんか便利になりました」では予算は通りません。「月間60時間削減、年間コスト360万円の削減効果」と言えれば、次の投資の承認が得られます。

2つ目は、**継続的な改善のため**。測定しないと、改善の方向がわかりません。「良くなった気がする」では改善のしようがない。

3つ目は、**社内展開の根拠**。「営業部でこれだけの効果が出ました」という実績があれば、他部署への展開がスムーズになります。

では、何を測ればいいか。4つのカテゴリがあります。

**カテゴリ1: 工数削減**
最もわかりやすい指標です。Before/Afterで、同じ業務にかかる時間がどれだけ減ったか。
- 単位: 時間/月、時間/年
- 例: 見積書作成が月間90時間→20時間に（70時間削減）

**カテゴリ2: コスト削減**
工数削減を金額に換算します。
- 計算式: 削減時間 × 時間単価 = コスト削減額
- 時間単価の目安: 従業員の人件費÷月間稼働時間（目安: 3,000〜5,000円/時間）
- 例: 70時間 × 3,500円 = 245,000円/月（年間294万円）

**カテゴリ3: 品質向上**
数値化しにくいですが、できる範囲で測ります。
- ミス率の変化: Before「月5件のミス」→ After「月1件」
- 手戻り率の変化: Before「30%が差し戻し」→ After「10%」
- 顧客満足度（アンケートがあれば）

**カテゴリ4: 売上・機会への貢献**
間接的な効果ですが、重要です。
- 浮いた時間で新規営業に充てた結果、商談数が増えた
- 提案書の質が上がり、受注率が改善した
- 対応スピードが上がり、顧客の満足度が向上した

ただし、カテゴリ4は因果関係が曖昧になりがちです。最初はカテゴリ1と2に絞って測定し、確実な数字で勝負することをお勧めします。

### パート2: 測定方法 — 具体的な手順（5分）

---

では、どうやって測るか。具体的な手順です。

**手順1: Beforeデータの取得（AI導入前に必ずやる）**

これが一番重要です。導入してしまってからでは遅い。

測定するもの:
- 対象業務の1回あたりの所要時間（3回測定して平均を取る）
- 月間の発生回数
- 担当者数
- ミス・差し戻しの件数（1ヶ月分）

測定のコツ:
- ストップウォッチで計る必要はありません。業務の開始時刻と終了時刻をメモするだけでOK
- 1回だけだとブレるので、最低3回は測ってください。特に早い回と遅い回を含めて平均を取る

**手順2: Afterデータの取得（導入2週間後〜1ヶ月後）**

導入直後は慣れていないので正確な数字が出ません。最低2週間は慣らし期間を置いてから測定してください。

同じ方法で測定します:
- 1回あたりの所要時間（3回測定の平均）
- 月間の発生回数（変わっていない場合が多い）
- 担当者数
- ミス・差し戻しの件数

**手順3: 差分の計算**

| 指標 | Before | After | 差分 | 削減率 |
|------|--------|-------|------|--------|
| 1回あたり時間 | 55分 | 12分 | -43分 | -78% |
| 月間総工数 | 90時間 | 20時間 | -70時間 | -78% |
| 月間コスト | 315,000円 | 70,000円 | -245,000円 | -78% |
| ミス件数 | 5件/月 | 1件/月 | -4件 | -80% |

**手順4: 投資対効果（ROI）の計算**

AIツールにはコストがかかります。ChatGPTの有料版、専用ツールのライセンス費用、導入にかかった人件費など。

```
ROI = (年間削減額 - 年間AI費用) ÷ 年間AI費用 × 100%
```

例:
- 年間削減額: 294万円
- ChatGPT Team: 月額$30×5名 = 約27万円/年
- 導入にかかった人件費: 約30万円（初年度のみ）
- ROI = (294 - 27 - 30) ÷ 57 × 100% = **416%**

ROIが100%を超えていれば「投資した以上のリターンがある」ということです。416%なら「1円投資したら4.16円返ってきた」という状態。これを経営層に見せれば、次の投資はほぼ確実に承認されます。

### パート3: Before/After比較のまとめ方（5分）

---

数字が揃ったら、それを「見せる形」にまとめます。経営層は忙しいので、一目で効果がわかるフォーマットが必要です。

**報告書の構成（A4で1〜2枚）:**

```
■ AI導入効果報告書

1. 概要
   - 対象業務: 見積書の作成（営業部）
   - 導入期間: 2025年4月1日〜5月31日
   - 使用ツール: ChatGPT Team
   - 導入パターン: 半自動（パターン2）

2. 効果サマリー
   ┌────────────────────────────────────┐
   │  工数削減: 月間70時間（年間840時間）   │
   │  コスト削減: 月間24.5万円（年間294万円）│
   │  ミス削減: 月5件 → 月1件（-80%）      │
   │  ROI: 416%                          │
   └────────────────────────────────────┘

3. Before/After比較
   （比較表を掲載）

4. 定性的な効果
   - 営業担当者の残業時間が平均2時間/月減少
   - 顧客への見積提出リードタイムが3日→1日に短縮
   - 営業事務担当が新規顧客フォロー業務に時間を充当

5. 課題と改善点
   - 特殊な商品の見積には手動対応が必要（月3-4件）
   - プロンプトの微調整が月1回程度発生

6. 次のステップ
   - 提案書作成への横展開（7月〜）
   - 受注データ入力の自動化検討（8月〜）
```

このフォーマットで大事なのは「効果サマリー」を一番上に持ってくることです。結論ファーストです。経営層が最初の3行で「おっ、いいじゃん」と思ってくれたら、その先も読んでくれます。

もう一つのコツは、**定性的な効果も必ず書く**こと。「残業が減った」「顧客への対応が早くなった」といった現場の声は、数字だけでは伝わらない臨場感を加えてくれます。

### パート4: 報告書の書き方のコツ（5分）

---

報告書を書くときの実践的なコツを5つお伝えします。

**コツ1: 数字は3つまでに絞る**

効果を示す数字がたくさんあると、かえって印象がぼやけます。最もインパクトのある数字を3つ選んで、大きく見せてください。「年間294万円削減」「工数78%削減」「ROI 416%」この3つで十分です。

**コツ2: 比較は「絵」で見せる**

数字の羅列より、棒グラフや円グラフのほうが一目で伝わります。Before/Afterの棒グラフを横並びにするのが最もシンプルで効果的です。

**コツ3: 控えめな数字で報告する**

AIの効果を多少控えめに報告するくらいがちょうどいいです。「78%削減」を「約75%削減」と書く。なぜなら、次の報告で「予想以上の効果が出ています」と言えるからです。逆に、盛った数字で報告して後から下方修正するのは最悪です。

**コツ4: 課題も正直に書く**

全部うまくいきましたという報告は信用されません。「特殊ケースは手動対応が必要」「月1回のメンテナンスが発生」といった課題を正直に書くことで、報告全体の信頼性が上がります。

**コツ5: 次のアクションで締める**

報告書は「現状報告」で終わらせない。「次はこの業務に横展開します」「この課題を解決するために、こういう検討を始めます」と、次のアクションを明示してください。これがあると、経営層は「この人に任せておけば進む」と感じます。

以上で今日の講座は終了です。次回は総合演習として、ここまで学んだことを全部使って、1つの業務をAI化し、効果を数値化するところまでやります。お疲れさまでした！

## 効果測定シート — Excel作成プロンプト

```
以下の仕様でAI導入効果測定シートのExcelファイルを作成してください。

【ファイル名】
AI導入効果測定シート.xlsx

【シート構成】
シート1: 「測定入力」（Before/Afterデータの入力シート）
シート2: 「効果サマリー」（自動集計されたダッシュボード）
シート3: 「ROI計算」
シート4: 「報告書テンプレート」
シート5: 「記入ガイド」

【シート1: 測定入力 のレイアウト】

セクションA: 基本情報（行1-8）
- 対象業務名（B2）
- 対象部署（B3）
- 導入ツール（B4）
- 測定期間 Before: 開始日（B5）〜終了日（D5）
- 測定期間 After: 開始日（B6）〜終了日（D6）
- 担当者数（B7）
- 時間単価（円/時間）（B8）— デフォルト値3,500円

セクションB: 工数測定（行10-25）

Before測定テーブル:
| 列 | 列名 |
|----|------|
| A | 測定回 |
| B | 日付 |
| C | 開始時刻 |
| D | 終了時刻 |
| E | 所要時間（分）— 自動計算 |
| F | メモ |

3行分（測定1-3）＋平均行（自動計算）

After測定テーブル（同じ構成で下に配置）

セクションC: 頻度・ボリューム（行27-32）
- 月間発生回数 Before（B28）
- 月間発生回数 After（B29）
- 月間総工数 Before（自動計算: 平均所要時間×回数×担当者数）
- 月間総工数 After（自動計算）

セクションD: 品質測定（行34-40）
- ミス件数/月 Before（B35）
- ミス件数/月 After（B36）
- 差し戻し率 Before（B38、パーセント）
- 差し戻し率 After（B39、パーセント）

セクションE: 定性的効果（行42-48）
- 3行分の自由記述エリア（担当者の声、顧客の反応など）

【シート2: 効果サマリー のレイアウト】

上段: KPIカード（4つ横並び）
- 工数削減（時間/月）— 大きなフォント
- コスト削減（円/月）
- ミス削減率（%）
- ROI（%）

中段: Before/After比較チャート
- 横棒グラフ: 工数のBefore/After
- 横棒グラフ: ミス件数のBefore/After

下段: 比較テーブル
| 指標 | Before | After | 差分 | 削減率 |
（シート1から自動参照）

【シート3: ROI計算 のレイアウト】

収益側:
- 年間工数削減額（自動計算: 月間削減額×12）
- その他の削減効果（手入力）
- 合計収益（自動計算）

費用側:
- AIツール費用/月（手入力）
- AIツール費用/年（自動計算）
- 導入時人件費（手入力）— 設計・テスト・トレーニングにかかった工数×単価
- その他初期費用（手入力）
- 合計費用（自動計算）

ROI:
- 初年度ROI（自動計算）
- 2年目以降ROI（自動計算 — 初期費用除外）
- 投資回収期間（月数）（自動計算）

【シート4: 報告書テンプレート】
- A4印刷を想定したレイアウト
- シート1-3のデータを自動参照して報告書形式に整形
- 見出し: 概要 / 効果サマリー / Before/After比較 / 定性的効果 / 課題 / 次のステップ
- 課題と次のステップは手入力エリア

【シート5: 記入ガイド】
- 各項目の入力ルール説明
- 時間単価の算出方法の解説
- よくある質問Q&A

【条件付き書式】
- シート2のKPIカード:
  - 工数削減: 正の値→緑背景、0以下→赤背景
  - ROI: 100%以上→緑太字、50-100%→黄色、50%未満→赤

- シート3のROI:
  - 100%以上→緑背景、0-100%→黄色背景、マイナス→赤背景

【書式設定】
- ヘッダー行: 背景色 #00B894、文字色 白、太字
- セクション見出し: 背景色 #DFE6E9、太字
- 入力セル: 薄い青背景（#EBF5FB）で入力箇所を明示
- フォント: 游ゴシック、サイズ10
- シート2のKPIカード: フォントサイズ24pt、太字
- A4印刷に最適化された余白設定（シート4）
```

### ファシリテーションのコツ

- 「ROI 416%」の数字は非常にインパクトがある。スライドで大きく見せて「これが経営層を動かす数字です」と強調
- Beforeデータの重要性は何度でも繰り返す。「導入してから測ればいいや」では手遅れ
- 報告書テンプレートは実際にスクリーンに映して、「この1枚を経営会議に出すだけでいい」と見せる
- 「控えめな数字で報告する」コツは管理職受講者に特に刺さる。「盛ると後が辛い」は共感される

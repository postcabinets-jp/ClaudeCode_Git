# 演習: 自分の業務で試してみる
**Claude Code AI組織導入スターターキット — Level 1: 初級（全社員向け）**
**POST CABINETS Inc.**

---

## 演習情報

| 項目 | 内容 |
|------|------|
| 対象 | 全社員（Level 1 講座を全4回受講済み） |
| 所要時間 | 約30〜45分（個人ワーク + 振り返り） |
| 形式 | ワークシート形式（自分のペースで進める） |
| 到達目標 | 自分の業務でAIを実際に活用し、効果と改善点を言語化できる |
| 準備物 | PC（ChatGPTまたはClaudeにログイン済み）、本ワークシート |

---

## はじめに

この演習は、Level 1 の総仕上げです。

これまで学んだことを振り返ると:
- 第1回: AIは「道具」であること、できること・できないこと
- 第2回: AIツールの基本操作（ログイン・チャット・3つの実演）
- 第3回: プロンプトの5つの基本テクニック
- 第4回: 業務での具体的な活用シーン10選

今日は、**あなた自身の業務** にAIを適用してみます。
正解はありません。「やってみた」ことが一番大事です。

日常的に行っている業務の中から、3つを書き出してください。
大きな仕事でなくてOKです。「毎週やっている」「毎日やっている」くらいの粒度が理想です。

| # | 業務内容 | 頻度（毎日/毎週/毎月） | 1回あたりの所要時間 |
|---|---------|---------------------|-------------------|
| 1 | | | |
| 2 | | | |
| 3 | | | |

**書き出しのヒント:**
- 毎朝やっていることは?
- 毎週の定例作業は?
- 「面倒だな」と思いながらやっていることは?
- 「誰かに代わりにやってほしい」と思ったことは?

**例:**
| # | 業務内容 | 頻度 | 所要時間 |

## 提出について

- **提出先**: 各社のAI推進担当者またはチームリーダー
- **提出期限**: 研修日から1週間以内
- **提出物**: 本ワークシート（Step 1〜5 記入済み）
- **提出方法**: メール添付 or 社内共有フォルダにアップロード

※ 提出は任意ですが、提出いただいた内容をもとに、Level 2（中級）の内容をカスタマイズします。皆さんの「ここをもっと知りたい」という声が、次の講座に直接反映されます。

## 付録: プロンプトのテンプレート集

すぐに使えるプロンプトのテンプレートです。[ ] の部分を自分の業務に合わせて書き換えてください。

### テンプレート1: メール作成
```
あなたはビジネスメールの専門家です。

以下の条件でメールを書いてください。
宛先: [会社名] [氏名]様
用件: [用件を具体的に]
トーン: [丁寧/カジュアル/フォーマル]
含めたい内容:
- [ポイント1]
- [ポイント2]

件名も考えてください。
```

### テンプレート2: 議事録作成
```
以下の会議メモを議事録にまとめてください。

出力形式:
- 会議名・日時・参加者
- 議題ごとの要点（箇条書き）
- 決定事項
- 次回アクション（担当者・期限付き）

[会議メモを貼り付け]
```

### テンプレート3: 文章の要約
```
以下の文章を[3つ/5つ]のポイントに要約してください。

条件:
- [対象者: 経営層/現場スタッフ/新入社員]にわかるレベルで
- 各ポイントは[1行/2行]以内
- 重要度の高い順に並べる

[文章を貼り付け]
```

```

...
```

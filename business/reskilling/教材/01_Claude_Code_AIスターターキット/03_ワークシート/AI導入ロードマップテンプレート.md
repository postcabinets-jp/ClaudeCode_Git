# AI導入ロードマップテンプレート
**POST CABINETS — Claude Code AI組織導入スターターキット**
**Version 1.0 / 2026-03**

---

## 目的

AI導入を「思いつき」ではなく「計画」として進めるためのロードマップ。
経営層への承認取得、社内共有、進捗管理のすべてに使える標準フォーマット。

## カスタマイズガイド

### 差し替え箇所一覧

| プレースホルダ | 説明 | 記入例 |
|---------------|------|--------|
| [会社名] | クライアント企業名 | 株式会社○○ |
| [部署名/全社] | 対象範囲 | 営業部 or 全社 |
| [YYYY年MM月] | 策定日 | 2026年4月 |
| [業務名] | 対象業務 | 日報作成、請求書処理 |
| [○○時間] | 現状工数 | 500時間 |
| [○○万円] | 金額 | 150万円 |

### 期間の調整

- **スピード型**（中小企業・少人数部署）: Phase 1を1ヶ月に圧縮可能
- **慎重型**（大企業・規制産業）: 各Phaseを1.5〜2倍に延長。Phase 1にセキュリティ審査を追加
- **段階型**（複数部署横断）: 部署ごとにPhaseをずらしてリソースを平準化

## 次のステップ

1. 現状データを埋める（業務棚卸しシートの結果を使用）
2. 経営層にPhase 1の承認を取る（この資料をそのまま使用）
3. Phase 1完了後、実績データでスライド2と6を更新
4. Phase 2以降は「効果測定レポート」の結果を基にリソース再配分

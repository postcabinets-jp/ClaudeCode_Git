# 効果測定レポートテンプレート
**POST CABINETS — Claude Code AI組織導入スターターキット**
**Version 1.0 / 2026-03**

---

## 目的

AI導入の効果を定量的に測定し、継続・拡大・中止の判断を行うための標準レポート。
「なんとなく便利になった」ではなく、数値で効果を証明する。

---

## Excel作成プロンプト

```
以下の仕様でExcelファイル（.xlsx）を作成してください。3つのシートで構成します。

■ 全体設定:
- フォント: 游ゴシック
- ヘッダー行: 背景 #1B3A5C（紺）、文字 白、太字
- 交互行: 薄いグレー（#F2F2F2）

---

■ シート1: Before/After比較（シート名: 効果比較）

列構成:
| 列 | 項目名 | 幅 |
|----|--------|-----|
| A | 測定項目 | 25 |
| B | 単位 | 10 |
| C | Before（導入前） | 15 |
| D | After（導入後） | 15 |
| E | 差分 | 15 |
| F | 改善率（%） | 15 |
| G | 備考 | 30 |

E列の数式: =D-C
F列の数式: =IF(C=0, "", (C-D)/C*100)
※ コスト系は逆（削減がプラス）、品質系はD-C/C*100

測定項目のプリセット行（A列に初期値として入力）:
1. 1件あたり処理時間（分）
2. 月間処理件数
3. 月間総工数（時間）
4. エラー発生件数（月）
5. エラー率（%）
6. 手戻り件数（月）
7. 残業時間（月間・時間）
8. 担当者数
9. 月間コスト_人件費（円）
10. 月間コスト_ツール費（円）
11. 月間コスト_合計（円）
12. 顧客対応速度（時間）
13. 従業員満足度（5段階）
14. 品質スコア（100点満点）

条件付き書式:
- F列:
  - 正の改善（コスト削減・時間短縮）: 背景 緑（#C6EFCE）
  - 改善なし or 悪化: 背景 赤（#FFC7CE）

行15に太い罫線で区切り、以下のサマリー行を追加:
- A16: 「年間コスト削減額（推定）」
- D16: =E11*12（月間コスト差分×12）
- A17: 「ROI（投資回収率）」
- D17: 数式入力エリア（ツール年間費用をE17に入れると自動計算）
  =D16/E17*100

---

■ シート2: 月次推移（シート名: 月次推移）

列構成:
| 列 | 項目名 |
|----|--------|
| A | 月 |
| B | 処理件数 |
| C | 1件あたり時間（分） |
| D | 月間総工数（時間）=B*C/60 |
| E | エラー件数 |
| F | エラー率（%）=E/B*100 |
| G | ツール利用率（%） |
| H | 従業員満足度 |
| I | コスト（万円） |
| J | 累計削減額（万円） |

A列にプリセット:
導入前3ヶ月、導入月、導入後1〜6ヶ月の計10行
（例: 2026/01, 2026/02, 2026/03, 2026/04[導入], 2026/05, ..., 2026/10）

J列: 導入月以降の累計削減額を自動計算
=IF(A列>=導入月, SUM(導入前平均コスト-I列の累計), "")

グラフ作成指示:
1. 折れ線グラフ1: 月間総工数の推移（D列）
   - 導入月に縦の点線を入れて「AI導入」と注記
   - 線の色: #2E86DE

2. 折れ線グラフ2: エラー率の推移（F列）
   - 線の色: #E74C3C

3. 棒グラフ: 累計削減額（J列）
   - 色: #27AE60のグラデーション

4. 複合グラフ: ツール利用率（棒）× 満足度（折れ線）
   - 利用率が上がるにつれ満足度も上がる相関を可視化

---

■ シート3: コスト換算（シート名: コスト分析）

列構成:
| 列 | 項目名 |
|----|--------|
| A | 費目 |
| B | 月額（円） |
| C | 年額（円）=B*12 |
| D | 備考 |

セクション1: 導入コスト（イニシャル）
- ツール初期設定費
- 研修・教育費
- カスタマイズ費
- その他初期費用
- 小計

セクション2: 運用コスト（ランニング）
- ツール月額利用料
- 保守・サポート費
- 追加研修費
- その他月額費用
- 小計

セクション3: 効果（削減額）
- 人件費削減
- 外注費削減
- 残業代削減
- その他削減
- 小計

セクション4: サマリー
- 年間総コスト（セクション1 + セクション2年額）
- 年間総効果（セクション3年額）
- 年間純効果（効果 - コスト）
- 投資回収期間（月）=セクション1小計 / セクション3月額小計
- 3年間累計効果

条件付き書式:
- 年間純効果がプラス → 背景 緑 + 太字
- 年間純効果がマイナス → 背景 赤 + 太字
```

## 記入例（日報作成AI導入の効果測定）

### Before/After比較

| 測定項目 | 単位 | Before | After | 差分 | 改善率 | 備考 |
|----------|------|--------|-------|------|--------|------|
| 1件あたり処理時間 | 分 | 20 | 3 | -17 | 85% | 音声入力→AI要約 |
| 月間処理件数 | 件 | 440 | 440 | 0 | — | 20人×22日 |
| 月間総工数 | 時間 | 147 | 22 | -125 | 85% | |
| エラー発生件数 | 件 | 15 | 3 | -12 | 80% | 記入漏れ・誤記 |
| 従業員満足度 | 5段階 | 2.1 | 4.3 | +2.2 | — | アンケート結果 |

### コスト分析サマリー

| 項目 | 金額 |
|------|------|
| ツール年間費用 | 60万円（月5万円） |
| 年間人件費削減額 | 450万円（125h×3,000円×12ヶ月） |
| 年間純効果 | +390万円 |
| 投資回収期間 | 1.6ヶ月 |

## レポート作成のポイント

1. **Before のデータは導入前に必ず取得**: 後から「多分このくらいだった」は説得力がない
2. **最低3ヶ月は計測**: 1ヶ月目は慣れの問題で効果が低く出る。3ヶ月で安定値を見る
3. **定性効果も記録**: 「ストレスが減った」「創造的な業務に時間を使えるようになった」等
4. **ツール利用率を追跡**: 導入しても使われなければ効果はゼロ。利用率が下がったら原因を調査
5. **経営層には ROI と投資回収期間を重点的に報告**: 技術的詳細より投資対効果

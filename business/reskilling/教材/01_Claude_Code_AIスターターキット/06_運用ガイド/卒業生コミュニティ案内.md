# 卒業生コミュニティ案内
**POST CABINETS — Claude Code AI組織導入スターターキット**
**Version 1.0 / 2026-03**

---

## 目的

AI導入プログラムを修了した企業同士が継続的に学び合い、AI活用を進化させ続けるためのコミュニティ。
プログラム終了後の「その後」を支える仕組みです。

---

## コミュニティの概要

### コミュニティ名

**POST CABINETS AI実践者ネットワーク**

### ビジョン

「AI導入を経験した企業が、互いの知見を共有し、日本の中小企業のAI活用を底上げする」

### 対象者

- POST CABINETSのAI導入プログラムを修了した企業の担当者
- AI推進チームのメンバー、チャンピオンユーザー
- 経営者・マネージャー（希望者）

### 参加費

- **基本参加: 無料**（プログラム修了者特典）
- プレミアムサポート（POST CABINETSによる月次個別相談付き）: 別途有料プラン

---

## 参加方法

### Step 1: 参加申請
プログラム修了時に、AI推進チームの代表者にコミュニティ参加の案内をお送りします。

### Step 2: オンボーディング
1. Slackワークスペースへの招待メールが届きます
2. 自己紹介チャンネルで「会社名・業種・導入した業務・今後やりたいこと」を投稿
3. 歓迎メッセージとコミュニティガイドが届きます

### Step 3: 活動開始
- 気になるチャンネルに参加
- 月1回の勉強会に出席（任意）
- 質問・事例共有をいつでも投稿

## 活動内容

### 1. 月次オンライン勉強会（月1回 / 60分）

| 時間 | 内容 |
|------|------|
| 0〜5分 | オープニング・今月のAIニュースダイジェスト |
| 5〜30分 | **メインセッション**: 参加企業による事例発表（1〜2社） |
| 30〜50分 | **ディスカッション**: 参加者のQ&A・意見交換 |
| 50〜60分 | **POST CABINETSからの情報提供**: 新ツール紹介、業界動向 |

**開催**: 毎月第[○][曜日] [時間]（例: 毎月第3木曜 18:00〜19:00）
**形式**: Zoom（録画あり。後日アーカイブ配信）
**発表者**: 参加企業から募集。発表者には特典あり（後述）

- 「営業日報AI化で月100時間削減した方法」（製造業A社）

- 「ChatGPT vs Claude — 業務別使い分けの実践知」（POST CABINETS）
- 「AIのセキュリティガイドライン策定で苦労したこと」（金融D社）

## コミュニティルール

### 基本ルール

1. **守秘義務**: コミュニティ内で共有された企業情報・数字は外部に漏らさない
2. **建設的な姿勢**: 批判ではなく改善提案を。失敗談の共有は大歓迎
3. **営業禁止**: コミュニティ内での直接的な営業活動・勧誘は禁止
4. **実名参加**: 企業名・氏名を明示（信頼関係の構築のため）
5. **相互尊重**: 企業規模・業種・AI習熟度に関わらず、全員が対等な学びの仲間

### 投稿ガイドライン

- **OK**: 事例共有、質問、情報提供、感想、アドバイス
- **NG**: 営業メッセージ、誹謗中傷、機密情報の投稿、政治的・宗教的な話題
- **グレー**: ツールの紹介（自社製品は事前にPOST CABINETSに相談）

### 退会

- 退会はいつでも可能（Slackを退出するだけ）
- 再参加も可能（POST CABINETSに連絡）

## 今後の発展構想

### Phase 1（開始〜6ヶ月）: 基盤構築
- 初期メンバー10〜20社でスタート
- 月次勉強会の定着
- 事例の蓄積

### Phase 2（6ヶ月〜1年）: コンテンツ充実
- 事例集の公開（メンバー限定）
- プロンプトライブラリの構築
- 業種別の分科会設置

### Phase 3（1年〜）: エコシステム化
- メンバー企業間の業務提携支援
- AI人材の紹介・マッチング
- 共同研究・共同開発プロジェクト
- 外部への発信力強化（メディア露出、カンファレンス登壇）

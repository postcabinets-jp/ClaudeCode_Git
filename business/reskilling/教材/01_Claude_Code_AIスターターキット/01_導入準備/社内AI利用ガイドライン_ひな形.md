## 第1条（目的）

本ガイドラインは、[貴社名]（以下「当社」）の役員および従業員（以下「社員」）が業務においてAI（人工知能）ツールおよびサービスを利用する際の基本的なルールを定め、以下を実現することを目的とする。

1. AIの適切な活用による**業務効率・生産性の向上**
2. **情報セキュリティ・個人情報保護**の確保
3. **法令遵守およびコンプライアンス**の維持
4. AIに関する**リスクの最小化**と適切な管理

> **カスタマイズポイント**: 自社の経営方針やデジタル戦略に合わせて目的を追加・修正してください。

## 第2条（適用範囲）

1. 本ガイドラインは、当社の全ての役員、正社員、契約社員、派遣社員、アルバイト・パート、および業務委託先のうち当社の情報資産にアクセスする者に適用する。
2. 業務時間内外を問わず、**当社の業務に関連してAIツールを利用する場合**に適用する。
3. 個人的な利用（業務と無関係な私的利用）は本ガイドラインの対象外とするが、当社のデバイス・ネットワークを使用する場合は本ガイドラインに準ずる。

> **カスタマイズポイント**: 業務委託先・子会社への適用範囲を自社の契約形態に合わせて調整してください。

## 第3条（定義）

本ガイドラインにおける用語の定義は以下のとおりとする。

| 用語 | 定義 |
|------|------|
| **AIツール** | テキスト生成AI、画像生成AI、音声認識AI、コード補完AI等、人工知能技術を利用したソフトウェア・サービスの総称 |
| **生成AI** | ユーザーの指示（プロンプト）に基づいてテキスト、画像、音声、コード等を生成するAI |
| **プロンプト** | AIツールに対する指示・入力情報 |
| **機密情報** | 当社の営業秘密、技術情報、財務情報、未公表の経営情報等、社外への開示が制限される情報 |
| **個人情報** | 個人情報保護法第2条に定める個人情報 |

## 第5条（禁止事項）

社員は、AIツールの利用において以下の行為を行ってはならない。

### 5.1 入力してはならない情報

| 分類 | 具体例 |
|------|--------|
| **機密情報** | 未公表の財務情報、M&A情報、新製品の開発情報、営業戦略、取引先との契約条件 |
| **個人情報** | 氏名＋住所、マイナンバー、クレジットカード番号、医療情報、従業員の評価情報 |
| **認証情報** | パスワード、APIキー、アクセストークン、秘密鍵 |
| **第三者の秘密情報** | NDA対象の取引先情報、顧客から預かった非公開データ |

### 5.2 禁止される利用方法

1. AIの出力を**人間のレビューなしに**そのまま社外に公開・送信すること
2. AIを使って**虚偽の情報を意図的に生成・流布**すること
3. AIを使って**他者を差別・誹謗中傷する**コンテンツを生成すること
4. AIの出力を**自社のオリジナル成果物として偽る**こと（AI利用の事実を隠す必要がある場合は上長に確認）
5. **法令・契約に違反する目的**でAIを利用すること
6. AIを使って当社の**情報セキュリティポリシーを回避**すること

> **カスタマイズポイント**: 業界固有の規制（金融、医療、法律等）がある場合は、業界ガイドラインに基づく禁止事項を追加してください。

## 第6条（情報の取り扱い）

### 6.1 情報分類に基づく利用基準

| 情報レベル | 定義 | AIツールへの入力 |
|:---:|------|:---:|
| **極秘** | 漏洩時に経営に重大な影響を与える情報 | **禁止** |
| **秘密** | 社外秘として管理される情報 | **原則禁止**（匿名化・抽象化した場合のみ可） |
| **社内限定** | 社内で広く共有可能だが社外には非公開 | **承認済みツール（法人契約）のみ可** |
| **公開** | 社外に公開されている・公開予定の情報 | **可** |

### 6.2 データ学習への配慮

1. **法人契約（Team/Enterprise）のAIツール**は、入力データがモデルの学習に使用されない契約であることを確認すること。
2. **無料版・個人版**は入力データが学習に使用される可能性があるため、業務情報の入力を禁止する。
3. APIを利用する場合は、利用規約のデータ取り扱い条項を確認し、[担当部署] の承認を得ること。

### 6.3 個人情報の取り扱い

1. 個人情報をAIツールに入力する場合は、**個人情報保護法**および当社の個人情報保護方針に従うこと。
2. やむを得ず個人情報を含むデータを処理する場合は、事前に**匿名化・仮名化**を行うこと。
3. AIツールの利用が個人情報の**第三者提供**に該当しないか、[担当部署 / 法務] に確認すること。

> **カスタマイズポイント**: 自社の情報分類基準が既にある場合は、そちらに合わせてください。個人情報の取り扱いは法務と連携して最終確認してください。

## 第7条（著作権・知的財産権）

### 7.1 AIの出力物の著作権

1. AIが生成した出力物の著作権の帰属については、利用するAIサービスの利用規約を確認すること。
2. AI生成物を当社の成果物として利用する場合は、**人間が実質的な創作的寄与**（編集、修正、選択、構成等）を行うこと。
3. AI生成物をそのまま（人間の関与なしに）公表する場合、著作権が発生しない可能性があることを認識すること。

### 7.2 第三者の著作権の尊重

1. 他者の著作物をAIに入力する場合は、**著作権法上の許諾**があることを確認すること。
2. AIが生成した出力物が**既存の著作物と類似していないか**を確認すること（特に画像生成AIの利用時）。
3. AIの出力物が第三者の著作権を侵害している疑いがある場合は、使用を中止し、[担当部署 / 法務] に報告すること。

### 7.3 AI生成物の表示

1. 社外向けの成果物にAI生成物を含む場合は、**社内の品質基準**に基づきレビューを行うこと。
2. 取引先・顧客からAI利用の有無を問われた場合は、**事実に基づき誠実に回答**すること。

> **カスタマイズポイント**: AI生成物の社外公表時にAI利用を明示するかどうかのポリシーを、業界慣行と自社方針に基づき決定してください。

## 第8条（正確性の確保と責任）

### 8.1 AIの出力に対する責任

1. AIの出力には**誤り（ハルシネーション）が含まれる可能性**があることを常に認識すること。
2. AIの出力を業務に利用する場合は、**必ず人間がレビュー**し、正確性を確認すること。
3. AIの出力に基づく判断・行為の**最終的な責任は利用者本人および承認者**にある。

### 8.2 重要度に応じたレビュー基準

| 重要度 | 例 | レビュー基準 |
|:---:|------|------|
| **高** | 契約書、プレスリリース、法的文書、財務報告 | **複数名のレビュー必須**。原典との照合を行う |
| **中** | 社外向けメール、提案資料、分析レポート | **上長レビュー必須** |
| **低** | 社内メモ、議事録ドラフト、アイデア出し | **本人の確認で可** |

> **カスタマイズポイント**: 自社の決裁フローに合わせてレビュー基準を調整してください。

## 第9条（モニタリングと記録）

1. 当社は、AIツールの適切な利用を確保するため、利用状況の**モニタリング**を行うことがある。
2. モニタリングは、法人契約のAIツールが提供する**管理機能（利用ログ、利用統計等）** を用いて実施する。
3. モニタリングの実施にあたっては、社員の**プライバシーに配慮**し、必要最小限の範囲で行う。
4. 各部署は、AI活用による**業務効率化の効果**を[頻度: 四半期ごと等]に[担当部署]へ報告すること。

> **カスタマイズポイント**: モニタリングの実施主体、頻度、報告先を自社の体制に合わせてください。従業員への事前通知方法も検討してください。

## 第10条（インシデント対応）

### 10.1 報告義務

以下の事象が発生した場合、社員は速やかに[担当部署 / 報告先]に報告しなければならない。

1. 機密情報・個人情報をAIツールに**誤って入力**した場合
2. AIの出力に基づく行為により**損害が発生**した、または発生のおそれがある場合
3. AIツールのアカウントへの**不正アクセス**が疑われる場合
4. AIの出力が**第三者の権利を侵害**している可能性が判明した場合

### 10.2 対応フロー

```
事象発生 → 即時報告（[担当部署]） → 影響範囲の調査 → 対策実施 → 再発防止策策定 → 記録
```

> **カスタマイズポイント**: 自社のインシデント対応フロー、報告先（情報セキュリティ委員会等）に合わせてください。

## 第11条（違反時の対応）

1. 本ガイドラインに違反した場合は、当社の**就業規則に基づく懲戒処分**の対象となることがある。
2. 違反の程度に応じて、以下の措置を講じることがある。
   - 口頭注意
   - 書面による警告
   - AIツールの利用権限の停止
   - 就業規則に基づく懲戒処分
3. 意図的でない違反（過失）の場合は、**再発防止のための教育**を優先する。

> **カスタマイズポイント**: 自社の就業規則・懲戒規程との整合性を確認してください。

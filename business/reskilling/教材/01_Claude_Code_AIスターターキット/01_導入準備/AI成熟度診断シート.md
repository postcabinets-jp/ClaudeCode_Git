# AI成熟度診断シート
**POST CABINETS — Claude Code AI組織導入スターターキット**
**Version 1.0 / 2026-03**

---

## 本資料の使い方

1. **診断質問**（全15問）に回答してスコアを集計する（所要時間: 5〜10分）
2. スコアから **成熟度レベル（0〜4）** を判定する
3. 現在のレベルに応じた **次のアクション** を確認する

---

## AI成熟度モデル（5段階）

```
Level 0          Level 1          Level 2          Level 3          Level 4
未着手     →    個人利用    →    部署導入    →    全社展開    →    自走・最適化
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### Level 0: 未着手

| 項目 | 内容 |
|------|------|
| **状態** | AIツールを業務で使っている社員がほぼいない |
| **典型例** | 「ChatGPTは聞いたことあるが使ったことない」「会社としてAIの方針がない」 |
| **課題** | AIへの理解不足、何から始めればいいかわからない |
| **次のステップ** | 経営層・管理職向けのAI体験ワークショップを実施。1つの部署で小さく試す |

### Level 1: 個人利用

| 項目 | 内容 |
|------|------|
| **状態** | 一部の社員が個人的にAIツールを使っている |
| **典型例** | 「詳しい人が勝手にChatGPTを使っている」「会社のルールはない」 |
| **課題** | 情報漏洩リスク、利用ルール未整備、効果が属人的 |
| **次のステップ** | 社内AI利用ガイドラインを策定。公式に使えるツールを選定・契約する |

### Level 2: 部署導入

| 項目 | 内容 |
|------|------|
| **状態** | 特定部署でAIツールが公式に導入・運用されている |
| **典型例** | 「マーケ部門でClaude Teamを契約」「カスタマーサポートにチャットボット導入」 |
| **課題** | 部署間の温度差、ナレッジの横展開ができていない |
| **次のステップ** | 成功事例を社内共有。AI推進担当（またはチーム）を設置し、全社展開の計画を策定 |

### Level 3: 全社展開

| 項目 | 内容 |
|------|------|
| **状態** | 全社的にAIツールが導入され、業務プロセスに組み込まれている |
| **典型例** | 「全社員がAIアシスタントを利用」「AIを使った業務フローが標準化」 |
| **課題** | ROI測定、セキュリティガバナンス、継続的な教育 |
| **次のステップ** | KPI設定と効果測定の仕組みを構築。API連携による自動化を推進 |

### Level 4: 自走・最適化

| 項目 | 内容 |
|------|------|
| **状態** | AIが経営戦略に組み込まれ、自律的に改善サイクルが回っている |
| **典型例** | 「AI活用のROIを四半期ごとに測定」「社員が自発的に新しいAI活用法を提案」 |
| **課題** | 最新技術のキャッチアップ、組織全体のリスキリング |
| **次のステップ** | AI人材の育成プログラム運用。自社独自のAIモデル・ワークフロー構築 |

## レベル判定基準

| 合計スコア | 成熟度レベル | 状態 |
|:---:|:---:|------|
| **0〜8点** | **Level 0: 未着手** | AIの業務活用がほぼ始まっていない |
| **9〜17点** | **Level 1: 個人利用** | 一部社員が使い始めているが、組織的な取り組みではない |
| **18〜26点** | **Level 2: 部署導入** | 特定部署で成果が出始めている |
| **27〜35点** | **Level 3: 全社展開** | 全社的に活用が進み、業務に定着している |
| **36〜45点** | **Level 4: 自走・最適化** | AIが経営に組み込まれ、継続的に改善している |

## カテゴリ別の傾向分析

スコアが偏っている場合、以下を確認してください。

| パターン | 傾向 | 対策 |
|----------|------|------|
| **Aが低い（体制不足）** | ツールは使えるが組織の仕組みがない | ガイドライン策定、推進担当の任命が急務 |
| **Bが低い（技術不足）** | 方針はあるがツール選定・導入が遅れている | ツール比較表を参照し、まず1つ契約して試す |
| **Cが低い（文化不足）** | 導入したが成果が見えず定着しない | 成功事例の共有、効果測定の仕組みを構築 |

## レベル別アクションプラン

### Level 0 → Level 1 へ（目安: 1〜2ヶ月）

- [ ] 経営層・管理職向けAI体験ワークショップを実施
- [ ] 1つの部署を選び、パイロット導入を開始
- [ ] 社内AI利用ガイドライン（ひな形）を策定
- [ ] 無料版でもいいので、まず全員が1回使ってみる機会を作る

### Level 1 → Level 2 へ（目安: 2〜3ヶ月）

- [ ] 社内ガイドラインを正式に制定・周知
- [ ] パイロット部署でAIツールの法人契約を開始
- [ ] 月1回のAI活用勉強会を開始
- [ ] 効果が出た業務の事例をまとめる

### Level 2 → Level 3 へ（目安: 3〜6ヶ月）

- [ ] AI推進チームを正式に設置
- [ ] 全社向け研修プログラムを開始
- [ ] パイロット部署の成功事例を他部署に横展開
- [ ] AI活用のKPIを設定し、月次で測定開始

### Level 3 → Level 4 へ（目安: 6〜12ヶ月）

- [ ] AIを業務プロセスに正式に組み込み（SOP化）
- [ ] API連携による自動化ワークフローの構築
- [ ] 四半期ごとのROI測定・改善サイクルの確立
- [ ] AI人材の採用・育成プログラムの整備

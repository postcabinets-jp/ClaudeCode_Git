# AI導入チェックリスト
**POST CABINETS — Claude Code AI組織導入スターターキット**
**Version 1.0 / 2026-03**

---

## 本資料の使い方

1. 下記マトリクスの各セルを確認し、自社で該当する業務に **チェック（ ）** を入れてください
2. AI適用可能性が **高** の業務から優先的に導入を検討します
3. 所要時間の目安: **5〜10分**

> **判定基準**
> - **高**: 定型的・反復的な業務。すぐにAIツール導入で効果が出る
> - **中**: 半定型。人の判断が一部必要だが、AIで補助・加速できる
> - **低**: 非定型・対人・現場判断が中心。AI活用には工夫が必要

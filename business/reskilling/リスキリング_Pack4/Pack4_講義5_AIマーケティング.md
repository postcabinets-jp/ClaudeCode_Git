## PART 3: 顧客分析＆A/Bテスト設計（15分）

### AIによる顧客セグメント分析

**プロンプト例:**
```
以下の顧客データ（CSV）を分析し、顧客セグメントを作成してください。

## データカラム:
- customer_id, age, gender, region
- first_purchase_date, last_purchase_date
- total_orders, total_amount, avg_order_value
- product_category（最頻カテゴリ）

## 分析内容:
1. RFM分析（Recency / Frequency / Monetary）でスコアリング
2. 5つのセグメントに分類:
   - VIP顧客（上位10%）
   - ロイヤル顧客（上位30%）
   - 成長顧客（購入頻度が増加傾向）
   - 休眠リスク顧客（最終購入が3ヶ月以上前）
   - 新規顧客（初回購入から1ヶ月以内）

3. 各セグメントについて:
   - 人数と売上構成比
   - 特徴（年齢、地域、購入カテゴリの傾向）
   - 推奨アプローチ（具体的な施策3つ）
   - KPI（追うべき指標）

4. 可視化:
   - セグメント別の売上構成比（円グラフ）
   - RFMスコアの散布図
   - セグメント別の購入頻度分布
```

### A/Bテスト設計をAIに任せる

**プロンプト例:**
```
以下のランディングページのA/Bテストを設計してください。

## 現状:
- LP URL: ○○
- 月間PV: 10,000
- 現在のCVR: 2.3%
- 目標CVR: 3.0%

## テスト設計の依頼:
1. テストすべき要素を優先順位付きで5つ提案
2. 各テストについて:
   - 仮説（「○○を△△に変更すると、□□が改善する。なぜなら〜」）
   - コントロール版（A）とテスト版（B）の具体的な違い
   - 必要なサンプルサイズと推定テスト期間
   - 成功基準（統計的有意差の条件）
3. テスト実施スケジュール（優先度順に3ヶ月分）
4. テスト結果の分析テンプレート

## 注意:
- 1度に1要素のみテスト（多変量テストは避ける）
- 統計的有意差（p < 0.05）を確保できるサンプルサイズで設計
```

### A/Bテスト結果の分析プロンプト

```
以下のA/Bテスト結果を分析し、次のアクションを提案してください。

## テスト内容:
- テスト対象: CTAボタンの色
- A（コントロール）: 青色ボタン
- B（テスト）: オレンジ色ボタン
- テスト期間: 2週間

## 結果:
| 指標 | A（青） | B（オレンジ） |
|------|---------|-------------|
| 表示回数 | 5,200 | 5,150 |
| クリック数 | 156 | 206 |
| CVR | 3.0% | 4.0% |
| 平均注文額 | ¥8,500 | ¥8,200 |

## 分析してほしいこと:
1. 統計的有意差はあるか（カイ二乗検定）
2. 実質的な効果量（Effect Size）
3. 収益インパクト（年換算）
4. 次にテストすべき要素の提案
5. 注意点・追加調査が必要な点
```

### 演習3: A/Bテストを1つ設計しよう（5分）

自社のWebサイト、LP、メールマガジン等で試してみたいA/Bテストを1つ設計してください。

| 項目 | あなたの設計 |
|------|-------------|
| テスト対象 | （例: メルマガの件名） |
| A（現行） | |
| B（テスト案） | |
| 仮説 | 「○○を△△にすると、□□が改善する」 |
| 計測指標 | （例: 開封率、CTR、CVR） |
| 成功基準 | （例: 開封率が5%以上向上） |

> **💡 A/Bテストの落とし穴**
>
> - テスト期間が短すぎる → 曜日効果で偏る（最低2週間）
> - 同時に複数要素を変える → 何が効いたかわからない
> - 「勝った」と判断する前に統計的有意差を確認
> - 勝利パターンが常に正しいとは限らない（季節変動、外部要因）

> **💡 AI生成コンテンツとSEOの注意点**
>
> - Googleは「AI生成かどうか」ではなく「ユーザーにとって有益か」で評価
> - AIの下書きをそのまま公開するのはNG（独自性・正確性の担保）
> - **E-E-A-T**: 経験（Experience）に基づく独自の知見が差別化のカギ
> - 事実確認（ファクトチェック）は必ず人間が行う
> - 大量の低品質記事より、少数の高品質記事が効果的

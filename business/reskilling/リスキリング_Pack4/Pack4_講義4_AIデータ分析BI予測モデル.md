## PART 1: データ分析の4レベルとAIの活用ポイント（15分）

### データ分析の4つのレベル

```
Level 4: 処方的分析（Prescriptive）  ← 最も高度
  「どうすべきか？」→ 最適なアクションを提案

Level 3: 予測的分析（Predictive）
  「これからどうなるか？」→ 将来を予測

Level 2: 診断的分析（Diagnostic）
  「なぜそうなったか？」→ 原因を特定

Level 1: 記述的分析（Descriptive）  ← 最も基本
  「何が起きたか？」→ 現状を可視化
```

### 各レベルのAI活用例

| レベル | 質問 | 従来の方法 | AI活用 |
|--------|------|-----------|--------|
| 記述的 | 先月の売上は？ | Excelでピボットテーブル | 「このCSVの月別売上推移を可視化して」 |
| 診断的 | なぜ売上が下がった？ | 手動でクロス分析 | 「売上減少の要因を多角的に分析して」 |
| 予測的 | 来月の売上は？ | 統計ソフトで回帰分析 | 「過去データから来月の売上を予測して」 |
| 処方的 | どうすれば改善？ | コンサルに依頼 | 「売上改善のアクションプランを提案して」 |

### AIが生成するコードのイメージ

```python
import streamlit as st
import pandas as pd
import plotly.express as px

st.set_page_config(page_title="売上ダッシュボード", layout="wide")
st.title("📊 売上ダッシュボード")

# データ読み込み
df = pd.read_csv("sales_data.csv", parse_dates=["date"])

# サイドバーフィルター
with st.sidebar:
    st.header("フィルター")
    date_range = st.date_input("期間", [df["date"].min(), df["date"].max()])
    products = st.multiselect("商品", df["product"].unique())
    regions = st.multiselect("地域", df["region"].unique())

# フィルタ適用
filtered = df.copy()
if products:
    filtered = filtered[filtered["product"].isin(products)]
if regions:
    filtered = filtered[filtered["region"].isin(regions)]

# KPIカード
col1, col2, col3, col4 = st.columns(4)
col1.metric("月間売上", f"¥{filtered['amount'].sum():,.0f}")
col2.metric("前月比", "+8.3%")  # 実際は計算で算出
col3.metric("粗利率", "42.1%")
col4.metric("注文件数", f"{len(filtered):,}")

# グラフ
fig = px.line(filtered.groupby(filtered["date"].dt.to_period("M"))
              ["amount"].sum().reset_index(),
              x="date", y="amount", title="月別売上推移")
st.plotly_chart(fig, use_container_width=True)
```

## PART 3: 簡易予測モデルをAIで作る（15分）

### 予測モデルとは

過去のデータのパターンから**未来を推定**する数理モデル。AIを使えば、統計の専門知識がなくてもプロトタイプが作れます。

### 予測モデルの種類と用途

| モデル種類 | 用途例 | 難易度 |
|-----------|--------|--------|
| **線形回帰** | 広告費と売上の関係予測 | 入門 |
| **時系列予測** | 来月の売上予測、在庫需要予測 | 中級 |
| **分類モデル** | 顧客の離脱予測、与信判定 | 中級 |
| **クラスタリング** | 顧客セグメント分析 | 中級 |

### 実践: AIに予測モデルを作らせる

**プロンプト例（ChatGPT Code Interpreter / Claude Code）:**
```
添付のCSV（月別売上データ2021-2024年）を使って、
2025年1〜6月の売上を予測してください。

## 要件:
1. まずデータの傾向を可視化（折れ線グラフ）
2. 季節性（月ごとの傾向）があるか確認
3. 以下の3つの方法で予測:
   a. 移動平均（直近6ヶ月）
   b. 線形回帰（トレンド + 季節性）
   c. Prophet（Facebookの時系列予測ライブラリ）
4. 3つの方法の予測結果を比較するグラフ
5. 各方法の精度（MAPE）を算出
6. 推奨する予測値と信頼区間を提示

## 出力:
- Pythonコード（再実行可能）
- グラフ3枚（傾向、季節性、予測比較）
- 予測結果テーブル（月, 移動平均, 線形回帰, Prophet, 推奨値）
- 分析レポート（500字以内）
```

### AIが生成する予測コードの概要

```python
import pandas as pd
from prophet import Prophet
import matplotlib.pyplot as plt

# データ準備
df = pd.read_csv("monthly_sales.csv")
df["ds"] = pd.to_datetime(df["date"])
df["y"] = df["amount"]

# Prophet モデル
model = Prophet(yearly_seasonality=True, weekly_seasonality=False)
model.fit(df[["ds", "y"]])

# 6ヶ月先まで予測
future = model.make_future_dataframe(periods=6, freq="MS")
forecast = model.predict(future)

# 可視化
fig = model.plot(forecast)
plt.title("売上予測（2025年1-6月）")
plt.savefig("forecast.png")

# 結果テーブル
result = forecast[["ds", "yhat", "yhat_lower", "yhat_upper"]].tail(6)
print(result)
```

### 異常検知とは

| 活用シーン | 検知対象 | ビジネス価値 |

### 実践プロンプト: 異常検知

```
以下の日別売上データ（過去1年分）から異常値を検出してください。

## 分析方法:
1. 基本統計量（平均、標準偏差、四分位数）を算出
2. IQR法（四分位範囲）で外れ値を検出
3. 移動平均からの乖離率で異常値を検出
4. 異常値をグラフ上でハイライト表示

## 出力:
- 異常値の日付・値・種類（異常に高い/低い）のテーブル
- 可視化グラフ（正常値=青、異常値=赤）
- 各異常値について考えられる原因の仮説

## データ特性:
- 土日は売上が通常の60%程度（曜日効果あり）
- 12月と3月は繁忙期（季節効果あり）
- これらの効果を考慮した上で「真の異常」を検出してください
```

> **💡 AI活用データ分析の注意点**
>
> - **データの品質がすべて**: ゴミデータからは良い分析は生まれない（Garbage In, Garbage Out）
> - **因果と相関を混同しない**: 「アイスの売上と水難事故が相関」→ 因果関係ではない
> - **外れ値の扱いに注意**: 除外するか含めるかで結論が大きく変わる
> - **結果を鵜呑みにしない**: AIの分析結果は「仮説」であり「事実」ではない

# セットアップ手順

## 前提条件

- Googleアカウントがあること
- Google スプレッドシートにアクセスできること
- 所要時間: 約5分

---

## STEP 1: スプレッドシートを作成する

1. [Google スプレッドシート](https://sheets.google.com) を開く
2. 「空白のスプレッドシート」を新規作成
3. スプレッドシートの名前を変更する（例: 「帳票自動化システム」）

---

## STEP 2: Apps Script エディタを開く

1. メニューバーの **「拡張機能」** をクリック
2. **「Apps Script」** を選択
3. 新しいタブで Apps Script エディタが開く

---

## STEP 3: コードをコピーする

Apps Script エディタには最初から `Code.gs` というファイルがあります。
以下の手順で4つのファイルを作成します。

### 3-1. config.gs を作成

1. エディタ左側の「ファイル」欄の横にある **「＋」ボタン** をクリック
2. **「スクリプト」** を選択
3. ファイル名を `config` に変更（`.gs` は自動で付きます）
4. `gas/config.gs` の内容を全てコピーして貼り付け

### 3-2. utils.gs を作成

1. 同じく **「＋」→「スクリプト」**
2. ファイル名を `utils` に変更
3. `gas/utils.gs` の内容を全てコピーして貼り付け

### 3-3. templates.gs を作成

1. 同じく **「＋」→「スクリプト」**
2. ファイル名を `templates` に変更
3. `gas/templates.gs` の内容を全てコピーして貼り付け

### 3-4. Code.gs を更新

1. 左側のファイル一覧で **「Code.gs」** をクリック
2. 元のコード（`function myFunction() {...}`）を**全て削除**
3. `gas/Code.gs` の内容を全てコピーして貼り付け

### 3-5. 保存

1. **Ctrl+S** (Mac: Cmd+S) で保存
2. または、ツールバーの **フロッピーディスクアイコン** をクリック

> **完成後のファイル構成**:
> ```
> Code.gs
> config.gs
> utils.gs
> templates.gs
> ```

---

## STEP 4: 初期設定を実行する

1. Apps Script エディタを閉じて、**スプレッドシートに戻る**
2. ページを **リロード** する（F5 または Cmd+R）
3. 数秒待つと、メニューバーに **「帳票自動化」** が表示される
4. **「帳票自動化」→「初期設定（シート作成）」** をクリック

### 初回の権限承認

初めて実行するとき、Googleから権限の確認が求められます。

1. 「承認が必要です」→ **「権限を確認」** をクリック
2. 自分のGoogleアカウントを選択
3. 「このアプリはGoogleで確認されていません」と出る場合:
   - **「詳細」** をクリック
   - **「（安全ではないページ）に移動」** をクリック
   - これは自分で作ったスクリプトなので安全です
4. **「許可」** をクリック

### 実行結果の確認

権限承認後、自動的に以下のシートが作成されます:

- **商品マスタ**: サンプルの商品が3件入っている
- **取引先マスタ**: サンプルの取引先が2件入っている
- **案件データ**: ヘッダーと計算式のテンプレートが入っている

---

## STEP 5: 動作を確認する

### 5-1. 案件を入力する

1. **「案件データ」** シートを開く
2. 2行目に以下を入力:

| 列 | 入力内容 |
|----|---------|
| A列（案件番号） | `EST-20260325-001` |
| B列（案件日付） | 今日の日付 |
| C列（取引先名） | プルダウンから「大阪機械工業株式会社」を選択 |
| D列（ステータス） | プルダウンから「見積作成」を選択 |
| E列（商品名） | プルダウンから「SUS304 丸棒」を選択 |
| F列（規格） | `φ10 × 1000mm` |
| G列（数量） | `100` |
| H列（単価） | `2500` |

I列（小計）・J列（消費税）・K列（合計）は**自動計算**されます。

### 5-2. 見積書を生成する

1. 2行目のどこかのセルを選択した状態で
2. メニュー **「帳票自動化」→「見積書を生成」** をクリック
3. **「見積書」** シートが自動作成される

### 5-3. ステータス連動を確認する

1. **「案件データ」** シートに戻る
2. D列（ステータス）を **「注文確定」** に変更
3. → **「納品書」** シートが自動生成される
4. D列（ステータス）を **「納品済み」** に変更
5. → **「請求書」** シートが自動生成される + L列（納品日）が自動入力される

---

## STEP 6: 自社情報をカスタマイズする

1. Apps Script エディタを開く
2. `config.gs` を開く
3. `COMPANY` オブジェクトの値を自社の情報に変更:

```javascript
const COMPANY = {
  name: '（自社名を入力）',
  zip: '（郵便番号）',
  address: '（住所）',
  tel: '（電話番号）',
  fax: '（FAX番号）',
  bank: '（振込先情報）',
  representative: '（代表者名）',
};
```

4. 保存（Ctrl+S）

---

## トラブルシューティング

### 「帳票自動化」メニューが表示されない

- ページをリロード（F5）してください
- 数秒待ってから確認してください
- Apps Script エディタでエラーがないか確認してください

### 「シートが見つかりません」エラー

- `config.gs` の `SHEET_NAMES` とスプレッドシートのタブ名が一致しているか確認
- 初期設定を再実行してみてください

### 計算式が動かない

- I列・J列・K列の数式が入っているか確認
- 2行目の数式を下の行にコピーしてください

### ステータス変更で帳票が生成されない

- プルダウンから選択しているか確認（手入力だと完全一致が必要）
- `config.gs` の `STATUS` の値と一致しているか確認

### 権限エラーが出る

- 一度 Apps Script エディタで手動実行してみてください
- 「実行」→「関数を実行」→「setupSheets」を選択
- 権限の承認画面が出るので承認する

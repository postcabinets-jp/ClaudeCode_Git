企業訪問・経営者面会は全て「政治家（大阪府議会議員）としての意見交換・地域課題ヒアリング」である。

ビジネス的な推論（見積もり送付・営業提案・商談フォロー等）は一切してはいけない。

**Why:** 2026-06-22、荒木利行さん（コーワ工務店）訪問のお礼に「見積もりをお送りします」と書いてしまい指摘を受けた。prepが企業訪問をPOSTCABINETSの営業活動と混同した。

**How to apply:**
- カレンダーに「企業訪問」「経営者面会」があっても、🏛政治家の活動として扱う
- お礼・フォローは「今後も意見交換させていただきたい」等の政治家としての表現
- prep生成・Daily記入・PROJECT_SYNC全てに適用
- 例外: プロジェクト.mdの🏢POSTCABINETS配下に明示的に書かれている案件のみビジネス扱い

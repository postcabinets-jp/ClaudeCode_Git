# カレンダー入力ルール（2026-06-01 確定）

カレンダーに**時刻枠で載せるのは2種だけ**：
1. **動かせない約束** — 他人との時間（MTG・訪問・駅立ち・会食・来訪）
2. **死守したい大きな塊** — 60分以上の集中（答弁調整・経営思考枠・フォーカス枠）

電話・連絡・雑務などの**単発タスクは個別の時刻枠にしない**：
- 1日1個「🧹 雑務消化枠（30分目安）」だけ置く
- 中身（徳洲会アポ・NTT解約等）は**イベント説明欄に箇条書き**。正本はGTask
- 塊だけ実時間。5分タスクに15分・30分の固定枠を与えない＝枠内に詰める

**Why:** 2026-06-01、TodoをやめてGoogleカレンダーに予定を入れる運用に切替えた初日、mtg_prepが単発タスク（電話・雑務）を `13:30-13:45 徳洲会` `15:00-15:15 NTT解約` のように**15分バラ枠で量産**。Nobu反応＝「だいたい終わってる／微妙」。原因2つ:
1. GTaskにある単発タスクをカレンダーに転記＝先週やめたばかりの二重管理が復活
2. 「5分で済むことを15分枠で使う／30分枠に作業が収まらない」とNobuが前から言っていた問題そのもの

**How to apply:**
- `/MTG` Phase 3 ステップB と `mtg_prep` の「今日の予定」生成時に厳守
- 単発タスク（電話・連絡・雑務）が出たら、個別時刻枠にせず🧹雑務消化枠へ集約
- カレンダーMCP書込み時は `description` に単発タスクを箇条書きで入れる
- 例外: 締切が当日かつ時刻指定がある単発（例「15時までに役所へ電話」）は個別枠OK。それ以外は集約

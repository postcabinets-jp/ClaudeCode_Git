## 問題

- Dailyページの「今週」リンクを押すと `W25`（ISO週番号）のページが作られてしまう
- 実際のWeeklyページは `wiki/weekly/2026-06/W3.md`（月内第3週）
- 2026-06-21時点で発生。Nobuが「徹底的に直せ」と指示

## 正しい仕様

| 対象 | 使用する週番号 | 例 |
|------|--------------|-----|
| Weeklyページのパス | **月内週番号**（W1〜W5） | `wiki/weekly/2026-06/W3.md` |
| ISO週番号 | **使わない**（Daily内リンクに用いない） | W25 ← これは不正 |

## 計算方法

月内週番号 = `Math.ceil(その月の何日目 / 7)`
例: 6月21日 → 21 / 7 = 3 → W3

## 修正すべき箇所

- Dailyページテンプレートの「今週」リンク生成ロジック
- `mtg_prep_v2.sh` 内の週番号計算部分
- Prep生成時にWeeklyページリンクを作る全箇所

**Why:** ISO週番号とObsidianのWeeklyページ命名規則（月内週）が一致していないため。修正前はW25を参照して空のページが作られていた。

**How to apply:** Weekly関連の処理を書くときは必ず月内週番号（W1〜W5）を使う。`date +%V` 等のISO週番号コマンドは使わない。

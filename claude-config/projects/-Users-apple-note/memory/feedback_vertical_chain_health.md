# 縦の連鎖ヘルスチェック（V6.2）

Dreaming（01:00 auto-dreaming / セッション終了時 /dreaming）で毎回実施。

## チェック項目（5点）

### 1. Weekly→Dailyの接続
- 🎯今週の重要タスクがW{n}.mdの「今週の確実ライン」と一致しているか
- 確実ラインに曖昧さ（「○人」等の未確定）がないか
- → 曖昧なら「週次レビューで具体化が必要」とDreamingに記録

### 2. 🔎NARが独立して機能しているか
- Daily📥に🔎次の一手（NAR分析）が独立サブセクションとして存在するか
- NARが「(NAR未生成)」のまま連続していないか（2日以上なら要調査）
- 各アクションにゴールタグ（🏛/🏢/🧍）が付いているか

### 3. 🧬コンディション分析がNobuの行動に影響しているか
- コーチメモに具体アクション（「今日Day B」「休養推奨」等）が書かれているか
- トレーニング頻度が目標（週4）と乖離していないか
- 体重トレンドが月次マイルストーンとオンペースか

### 4. 📌持ち越しの肥大
- 📌持ち越しが10件を超えていないか
- 7日以上前の持ち越しが残っていないか（棚卸し対象）
- 重複項目がないか

### 5. 3領域のバランス
- Limitlessログから直近3日の行動を🏛/🏢/🧍で分類
- 🏢が週0件なら「事業に触れていない」フラグ
- 🧍トレーニングが週2回未満なら「ペース遅れ」フラグ

## 診断結果の書き方

Dreamingの「今日の主な学び」に:
```

- [具体的な所見1文]
```

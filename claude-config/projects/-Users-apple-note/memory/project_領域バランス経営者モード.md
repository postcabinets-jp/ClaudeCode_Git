# 領域バランス + 経営者モード管理（2026-05-28 v2）

**Nobuの当事者領域は3本柱（家庭は目標管理外）：**
- 🏛 **政治家**：南大阪地盤・2026目標（X 5,000人 / Insta 1,500人 / 議会質問4回等）
- 🏢 **POSTCABINETS**：年商1億・年収1,000万円
- 🤖 **AI / 📹 発信**：連動領域

**根本問題：**
1. カレンダーは政治家活動で埋まる → POSTCABINETSはカレンダー枠なし → 放置
2. POSTCABINETSタスクが「実務者作業」だらけ → 政治でパワー削がれた状態で動かない
3. 「経営者として考える時間」が0 → 戦略・人を動かす判断が進まない

**新ルール（厳守）：**

### モード制（MTG Phase 1A で毎日選ぶ）
- カレンダー・ToDoを見せる**前**にマインドだけ決める（脳のカレンダー駆動化を防ぐ）
- 選択肢：A政治集中 / B経営集中 / C両軸
- Phase 1B のブリーフィングは選んだモードに応じて出力差をつける

### 🏢 POSTCABINETS の3区分（厳守）
- 👔 **経営者タスク**：意思決定・方向付け・人を動かす → **自分がやる**
- 🔧 **人に振る**：資料・リサーチ・叩き台・連絡 → **Claude Codeエージェントに振る**
- 💭 **ペンディング**：意思決定待ち

### 最低ライン
- **POSTCABINETS全体：週3件以上**（日次自由）
- **👔 経営判断：週1件以上**（週0件が2週連続 → 強警告・モードB強制提案）
- **💭 経営思考枠：週1枠以上**カレンダーに物理確保（1時間）
- 🏢タスクで実務者作業が出たら、まず「Claude Codeに振れる？」を秘書から問う

### 振り先
- **Claude Codeエージェント**：リサーチ・資料・叩き台・タスク分解・連絡文案
- 判断・交渉・最終決定だけ Nobu が行う

- wiki/プロジェクト.md：🏢セクションを 👔/🔧/💭 で3区分（当日タスクはGoogleTask）

- Wn.md：📊領域別バランス表 + 👔経営判断件数 / 💭経営思考枠 トラッキング

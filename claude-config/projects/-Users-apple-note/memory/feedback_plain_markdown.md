---
name: ノートの中身は装飾なしプレーンmdで統一
description: Vaultノートへの書き込みはコールアウト/表/絵文字などの装飾を使わず、素のmd（見出し＋箇条書き）で書く
type: feedback
---

Vault内のノートに内容を書くときは、装飾を足さない。見出し（`#`）と箇条書き（`-` / 数字リスト）だけの素のMarkdownで統一する。

**使わないもの:**
- Obsidianコールアウト（`> [!note]-` 等の折りたたみ含む）
- `<details>` HTMLトグル
- 表組み（一覧性が要る時でも、まず箇条書きで書く）
- 装飾絵文字（🗣💡⚠️★ など、意味付けの飾り）

**Why:** 2026-07-13、タウンミーティングのネタ帳をコールアウト＋表＋色分けで作ったら「デザインとか無しでいい。今後も。mdで統一して」と指摘された。Nobuはノート本文に見た目の演出を求めていない。中身がプレーンに読めればいい。
**How to apply:** Inbox/・Daily/・raw/・wiki/ 等へ文章を書く全作業に適用。トグルや表を明示的に頼まれた時だけ使う。デザイン系の成果物（LP/スライド/UI）は別（CLAUDE.md 0-4のデザインルールが効く）。ノート本文＝プレーンmd、が原則。

## 立場
- 大阪府議会議員（大阪維新の会）・2期目
- 2025年に政調会長就任
- 選挙区：南大阪エリア

## 2026年の政治的重点目標
- 南大阪における磐石な地盤を完璧に作り上げる
- Xフォロワー 1200→5000人、Instagram 800→1500人
- 「呼んだら集まる」メンバー 10→20名
- 協力企業・団体 4〜5→10組織
- 実証プロジェクト 2件完了
- 議会質問 4回（うち1回メディアレベル）

## 5月の重点タスク
- 担当議員への役割依頼を完了する
- 主要政策テーマの絞り込みを行う
- 南大阪選出議員1人にアポを入れて会いに行く
- 外向き活動（個別訪問・企業訪問・地域イベント）を毎日カウントする運用を始める
- SNS投稿を週1本始める

## 政調会長としての直近の動き
- 政調役員会議でメンバー役割分担・マニフェスト達成状況の整理を進める
- 副首都・都構想は政治マター（別扱い）、政策議論テーマを各メンバーに振り分ける方向
- 衆議院会館で遠藤代議士と副首都法案の次の一手を仕込み中（2026-05-19時点）

## 政治からの出口戦略（2026-06-14 ライフデザイン会議で再定義）
2031年（40歳）に政治家引退予定。2026-31は「土台構築期」＝「人を育てて手放す」を政治・経営で予行する5年と位置づけ、その間に南大阪の地盤と実績を確立する。引退後（2031〜）に海外移住・海外事業を本格化。

**Why:** 2031年引退・引退後の海外移住が人生計画の本線。政治は「今の役割」であり「ゴール」ではない。※旧記述「2027年引退」は6/14会議で誤りと判明し訂正（4年ズレていた）。
**How to apply:** 政治関連の提案は「2031年引退までに何を残すか」の文脈で考える。永続的な政治キャリアを前提とした提案は不要。政治の再定義（南大阪＝地域が主語 → 旗に共感する人を集めチームを作る＝人が主語）も踏まえる。

# 政治活動の業界用語

朝会(/MTG)・Daily・Limitless記録でこれらの語が出たら、下記の意味で解釈する。

## 叩き（たたき）
**= 地域訪問・ポスティング活動**（個別訪問してチラシを配る外向き活動）。
- 文書整理の「叩き台」とは**全くの別物**。Nobuが「叩き」と言ったら地域訪問活動を指す。
- 典型: 「たたき 10:30-12:00」「岸和田城周辺をたたき」= チラシ300枚を3人体制でポスティング等。
- 領域は🏛政治家（外向き活動カウントの対象）。

**Why:** 2026-05-29にNobuが「地域訪問って僕らの業界用語では叩きって言う」と明言。秘書が「叩き台（document draft）」と誤読するのを防ぐため。
**How to apply:** カレンダー/ToDo/Limitlessで「叩き」「たたき」単体 → 地域訪問活動として扱う。「叩き台」（〜台が付く）は通常通りdraftの意味。

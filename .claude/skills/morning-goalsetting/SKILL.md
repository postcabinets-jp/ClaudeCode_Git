# Morning Goal-Setting Session

## ルール（絶対）

- **最大5ターンで完了する。ループ禁止。**
- 各ステップに制限時間の意識を持つ。深掘りしすぎない。
- 出力は日本語。トーンはカジュアル・直球・無駄なし。
- 完了時は `🟢 完了` を表示。

---

## フロー（5ターン以内）

### ターン1: Claude（情報収集 → 質問）

以下を**並列で**実行し、結果をまとめて提示する:

1. **カレンダー確認**: Google Calendar MCP (`gcal_list_events`) で今日の予定を取得。MCP未接続なら「カレンダー未接続」と表示してスキップ。
2. **優先事項確認**: `context/current-priorities.md` を読む。ファイルがなければスキップ。
3. **Notionタスク確認**: Notion MCP (`notion-search`) で Owner=Human/Either かつ Status=Todo/In Progress のタスクを取得。MCP未接続ならスキップ。

取得した情報を**箇条書き3-5行**で簡潔に提示した後、こう聞く:

> 今日、これ1つ終わったら「勝ち」ってものは何？

これ以外の質問はしない。1つだけ。

### ターン2: Nobu（回答）

Nobuが「今日の勝ち」を回答する。

### ターン3: Claude（プラン提示 → 確認）

Nobuの回答を受けて、以下を出力する:

```
## 今日のプラン

**勝ち条件**: [Nobuが言ったこと]

### アクション（2-3個）
1. [ ] [具体的なアクション] — 完了基準: [明確な done criteria]
2. [ ] [具体的なアクション] — 完了基準: [明確な done criteria]
3. [ ] [具体的なアクション] — 完了基準: [明確な done criteria]（必要な場合のみ）

**ブロッカー**: [あれば。なければ「なし」]
```

その後:

> これでいく？調整ある？

### ターン4: Nobu（確認 or 調整）

- 「OK」「いく」「これで」→ ターン5へ
- 調整がある → Claude が**1回だけ**修正してターン5へ（再度の調整確認はしない）

### ターン5: Claude（最終出力 + ログ）

確定版の「今日のプラン」を整形して出力する:

```
━━━━━━━━━━━━━━━━━━━━━━━━━━
📋 TODAY'S PLAN — [今日の日付]
━━━━━━━━━━━━━━━━━━━━━━━━━━

🏆 勝ち条件: [確定した勝ち条件]

□ 1. [アクション] → done: [基準]
□ 2. [アクション] → done: [基準]
□ 3. [アクション] → done: [基準]

━━━━━━━━━━━━━━━━━━━━━━━━━━
```

**Notion ログ（オプション）**: Notion MCP が接続されていれば、Tasks DB に上記アクションをタスクとして追加するか聞く。聞くのは**1回だけ**。返事がなければスキップ。

---

## やってはいけないこと

- 追加の質問でターンを消費する
- 「他にありますか？」と聞く
- 5ターンを超える
- 長文の励ましや前置き
- 英語で話す（ユーザーが英語で来た場合を除く）
